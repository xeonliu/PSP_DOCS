PSP® Programmer Tool Runtime Library Release 6.6.0

コンテンツ情報チェック機能

1 コンテンツ情報画面

コンテンツ情報のPSP®システムソフトウェア上での表示です（下図はUMD®VIDEOの場合です）。


コンテンツ情報はいくつかの画像や映像、音声のファイルで構成されています。
PSP®GAME、UMD®VIDEO、UMD®AUDIO、セーブデータで使用されています。

2 コンテンツ情報チェック機能

コンテンツ情報を構成するファイルをチェックする機能です。
チェック結果がエラーになった場合、どのファイルのどこが原因かを表示します。

2. 1 チェック機能の使用手順

以下の操作を行います。

  1. 開発ツールのPSP®システムソフトウェアを起動します。
  2. コンテンツ情報が表示される画面に遷移します。
  3. △ボタンを押して、オプションメニューを開きます。
  4. メニューの中の「Check」を選択します。
  5. チェック機能が動作し、結果が画面に表示されます。

3 チェック結果

チェック結果の見方です。

3. 1 Status: OKと表示されている

正しいコンテンツ情報です。修正する箇所はありません。

3. 2 Status: OKと表示されているが、他に黄色文字で先頭がWarning…となっている行も表示されている

コンテンツ情報としては正しいです。
しかし、設定しておいた方がよい情報やファイルがあります。それについてはWarning…行に表示されますので、なるべく対応していただけますようお願いします。

3. 3 Status: Errorと表示され、他に赤文字で何行か表示されている

コンテンツ情報に間違いがあります。
間違いの内容はその下の行に全て報告されていますので、修正をお願いします。
エラー報告行は、

［エラーを含むファイル名］−［エラーの内容］

の構成になっています。
他にも、Unknown file found: … があります。

3. 4 注意

エラー内容やWarning以外に、設定内容を確認するための行も一緒に表示されることがあります。このような行は文字色が白です。チェック結果とは関係ありません。
PSP®GAMEとセーブデータ、どちらをチェックするかによって、表示されるWarningやエラー行が異なる場合があります。以降の解説では、具体的に表示される行の内容を説明していますが、PSP®GAMEとセーブデータのどちらで報告されるものかは、≪≫内に表記しています。

4 Warning…行について

チェック結果がエラーにはなりませんが、なるべく設定していただきたい項目やファイルについて書かれるのがWarning…行です。
文字色は黄色です。
以下のような場合があります。

4. 1 PARAM.SFOファイルのWarning

Warning PARAM.SFO - DISC_VERSION: …

≪PSP®GAME≫
ディスクバージョンのフォーマットが不正です。弊社品質管理部とのマスターアップ作業を円滑に進めるために、なるべくフォーマットを守って下さい。

Warning PARAM.SFO - SAVEDATA_TITLE is required.

≪セーブデータ≫
セーブデータタイトルがありません。サブタイトルとして、是非ご活用下さい。

Warning PARAM.SFO - SAVEDATA_DETAIL is required.

≪セーブデータ≫
詳細情報がありません。セーブデータの内容がロードせずともわかるような情報表示のために、是非ご活用下さい。

Warning PARAM.SFO - BOOTABLE: 0 (0x0)

≪PSP®追加ゲームデータ≫
追加ゲームデータに対してチェック機能を使用すると、表示されます。BOOTABLE: 0であれば正常ですので、追加の修正や設定は必要ありません。もし、追加ゲームデータでBOOTABLE: 1と表示された場合は、PARAM.SFOの作成手順に誤りがあります。

Warning PARAM.SFO - ATTRIBUTE: 1 (0x1)

≪PSP®ダウンローダブルゲーム≫
追加ゲームデータによるコンテンツ情報のオーバーライドを可能にしたダウンローダブルゲームで、表示されます。それ以外の場合でこの表示がなされた場合は、PARAM.SFOの作成手順に誤りがあります。

5 エラー行について

チェック結果がエラーになった原因を表示しています。
修正しない限り、正しいコンテンツ情報とはなりませんので、対応をお願いします。
文字色は赤です。
以下のような場合があります。

5. 1 PARAM.SFOファイルのエラー

PARAM.SFO - No File

≪PSP®GAME、セーブデータ≫
PARAM.SFOがありません。作成してください。

PARAM.SFO - Corrupted Data

≪PSP®GAME、セーブデータ≫
PARAM.SFOに致命的欠陥があります。正しい方法で再度作成し直して下さい。

PARAM.SFO - DISC_ID Error: …
PARAM.SFO - PSP_SYSTEM_VER Error: …
PARAM.SFO - REGION Error: …
PARAM.SFO - CATEGORY: … is required.
PARAM.SFO - BOOTABLE: 1 is required.
PARAM.SFO - DISC_NUMBER: 1 is required.
PARAM.SFO - DISC_TOTAL: 1 is required.
PARAM.SFO - PARENTAL_LEVEL: 0 to 11 is required.
PARAM.SFO - TITLE is required.
PARAM.SFO - ATTRIBUTE Error: …
PARAM.SFO - ATTRIBUTE: 0 or 1 is required.
PARAM.SFO - USE_USB Error: …
PARAM.SFO - TARGET_APP_VER Error: …
PARAM.SFO - APP_VER Error: …
PARAM.SFO - GAMEDATA_ID Error: …

≪PSP®GAME、セーブデータ≫
PARAM.SFOで設定されている各項目の内容が正しくありません。正しい値に修正して下さい。

PARAM.SFO - SAVEDATA_DIRECTORY is incorrect.

≪セーブデータのみ≫
セーブデータが保存されているディレクトリが適切ではありません。

5. 2 画像ファイルICON0.PNG、PIC0.PNG、PIC1.PNGのエラー

ICON0.PNG - No File

≪PSP®GAME、セーブデータ≫
ICON0.PNGがありません。作成して下さい。

xxx.PNG - Corrupted Data

≪PSP®GAME、セーブデータ≫
画像ファイルが破損しています。作成し直して下さい。

xxx.PNG - png [W] x [H] is required.

≪PSP®GAME、セーブデータ≫
画像ファイルのサイズが正しくありません。正しいサイズは[W] x [H]です。修正して下さい。

xxx.PNG - png [D] bit is required.

≪PSP®GAME、セーブデータ≫
画像ファイルの色数（色深度）が正しくありません。正しい色数（色深度）は[D]ビットです。修正して下さい。

5. 3 動画、音声ファイルICON1.PMF、SND0.AT3のエラー

ICON1.PMF - ICON1.PMF + SND0.AT3 > 500 KB
SND0.AT3 - ICON1.PMF + SND0.AT3 > 500 KB

≪PSP®GAME、セーブデータ≫
動画アイコンファイルICON1.PMFと、BGMファイルSND0.AT3の、ファイルの大きさは合計500キロバイトまでとなっています。収まるように調整して下さい。

5. 4 その他のファイルのエラー

Unknown file found: …

≪PSP®GAME、セーブデータ≫
システム予約名を使ったファイルや、余計なファイルが見つかりました。変名、削除など対応をお願いします。

6 設定内容表示行について

ツールを使用して確認できると便利である設定項目がいくつかありますので、チェック機能では結果と同時にこれらも表示します。
チェック結果には影響しない行です。文字色は白です。
エラーやWarning...行と混同されないようご注意下さい。
以下のような場合があります。

6. 1 PARAM.SFO関連

ms0:/PSP/GAME/…
ef0:/PSP/GAME/...

≪PSP®GAME≫
チェックしているゲームのファイルパスを表示します。

DISC_ID: …

≪PSP®GAME≫
ディスク品番を表示します。

DISC_VERSION: …

≪PSP®GAME≫
ディスクバージョンを表示します。

PSP_SYSTEM_VER: …

≪PSP®GAME≫
起動に必要なPSP®システムソフトウェアのバージョンを表示します。

USE_USB: …

≪PSP®GAME≫
USB機器を利用するかどうかを表示します（0…USB機器を利用しない、1…利用する）。

APP_VER: …

≪PSP®GAME≫
パッチのバージョンを表示します。
詳しくは「PSP®パッチパッケージ作成ガイド」を参考にしてください。

TARGET_APP_VER: …

≪PSP®GAME≫
差分対象となるバージョンを表示します。
詳しくは「PSP®パッチパッケージ作成ガイド」を参考にしてください。

PBOOT_TITLE: …

≪PSP®GAME≫
パッチパッケージ単体用のタイトルを表示します。
詳しくは「PSP®パッチパッケージ作成ガイド」を参考にしてください。

GAMEDATA_ID: …

≪PSP®GAME≫
複数のディスクで共通のパッチを使用するためのパラメータを表示します。
詳しくは「PSP®パッチパッケージ作成ガイド」を参考にしてください。

Version: …

≪セーブデータ≫
セーブデータの形式を表示します（1.00…旧形式、2.00…新形式）。

7 パッチを適用されたゲームのチェック

パッチを適用されたゲームに対してチェック機能を利用する場合、ゲーム本体、パッチそれぞれのパラメータをチェックし、両方の結果を表示します。
「---- EBOOT.PBP ----」以降はゲーム本体のチェック結果を表示します。
「---- PBOOT.PBP ----」以降はパッチのチェック結果を表示します。
それぞれにチェック結果が表示されますのでご注意ください。

また、ゲーム本体とパッチのパラメータの比較も行ないます。
「---- COMPARE ----」以降に、パラメータの比較結果が表示されます。
パラメータが異なる場合、赤い文字で「different」と表示されます。
パラメータの比較表示には、以下のような場合があります。

7. 1 比較表示

DISC_ID: …

ディスク品番の比較結果を表示します。

GAMEDATA_ID: DISC_ID: …

GAMEDATA_IDとディスク品番の比較結果を表示します。
ゲーム本体にGAMEDATA_IDを指定してあり、パッチがGAMEDATA_IDに置かれている場合、ゲーム本体のGAMEDATA_IDとパッチのDISC_IDとの比較が行なわれます。

REGION: …

リージョンの比較結果を表示します。

PARENTAL_LEVEL: …

パレンタルロックレベルの比較結果を表示します。

USE_USB: …

USB機器を利用するかどうかの比較結果を表示します。

8 その他の表示行についてご注意

PSP®GAMEのチェック結果では、

multi-language ***

と表示されますが、この行以降の表示は現状無視していただいて結構です。
