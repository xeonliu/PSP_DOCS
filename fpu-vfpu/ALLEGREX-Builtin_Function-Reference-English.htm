__builtin_allegrex_clz
 [Function]
Count Leading Zeroes

Syntax

int __builtin_allegrex_clz (
	int val
);
Description

This function counts the number of consecutive 0 bits from the MSB of the contents of val and returns the result (0 to 32).

Operation

clz v0, a0

__builtin_allegrex_rotr
 [Function]
Rotate Word Right

Syntax

unsigned int __builtin_allegrex_rotr (
	unsigned int val,
	int n
);
Description

This instruction rotates the contents of val according to an n-bit right shift and returns the result.
Behavior is undefined if n is negative or is greater than or equal to 32.

Operation

When n is an immediate value:
rotr v0, a0, n

When n is a variable:
rotrv v0, a0, a1

Restriction

n is a value which satisfies the following restrictions.

0 <= n < 32

__builtin_allegrex_rotl
 [Function]
Rotate Word Left

Syntax

unsigned int __builtin_allegrex_rotl (
	unsigned int val,
	int n
);
Description

This instruction rotates the contents of val according to an n-bit left shift and returns the result.
Behavior is undefined if n is negative or is greater than or equal to 32.

Operation

When n is an immediate value:
rotr v0, a0, (32 - n)

When n is a variable:
subu v0, zero, a1
rotrv v0, a0, v0

Restriction

n is a value which satisfies the following restrictions.

0 <= n < 32

__builtin_allegrex_ext
 [Function]
Extract Bit Field

Syntax

unsigned int __builtin_allegrex_ext (
	unsigned int val,
	unsigned int pos,
	unsigned int size
);
Description

This function extracts size bits from the pos bit position of val and returns the right-justified result.

Operation

ext v0, a0, pos, size

Restriction

pos and size are immediate values that satisify the following restrictions.

0 < pos <= 32
0 <= size < 32
0 <= pos + size < 32

__builtin_allegrex_ins

);
Description

This function inserts the low-order size bits of val2 at the pos bit position of val1 and returns the result.

Operation

ins a0, a1, pos, size

Restriction

pos and size are immediate values that satisify the following restrictions.

0 < pos <= 32
0 <= size < 32
0 <= pos + size < 32


__builtin_allegrex_cache
 [Function]
Cache Operation

Syntax

void __builtin_allegrex_cache (
	int code,
	void *ptr
);
Description

This function performs the cache operation specified by code for the address ptr.

Operation

cache code, 0(a1)

Restriction

code is a 5-bit immediate value. The following values, which are defined in allegrex.h, can be used.
  • SCE_ALLEGREX_CACHEOP_INDEX_INVALIDATE_I
  • SCE_ALLEGREX_CACHEOP_INDEX_UNLOCK_I
  • SCE_ALLEGREX_CACHEOP_HIT_INVALIDATE_I
  • SCE_ALLEGREX_CACHEOP_FILL_I
  • SCE_ALLEGREX_CACHEOP_FILL_WITH_LOCK_I
  • SCE_ALLEGREX_CACHEOP_INDEX_WRITEBACK_INVALIDATE_D
  • SCE_ALLEGREX_CACHEOP_INDEX_UNLOCK_D
  • SCE_ALLEGREX_CACHEOP_CREATE_DIRTY_EXCLUSIVE_D
  • SCE_ALLEGREX_CACHEOP_HIT_INVALIDATE_D
  • SCE_ALLEGREX_CACHEOP_HIT_WRITEBACK_D
  • SCE_ALLEGREX_CACHEOP_HIT_WRITEBACK_INVALIDATE_D
  • SCE_ALLEGREX_CACHEOP_CREATE_DIRTY_EXCLUSIVE_WITH_LOCK_D
  • SCE_ALLEGREX_CACHEOP_FILL_D
  • SCE_ALLEGREX_CACHEOP_FILL_WITH_LOCK_D

Depending on the particular operation, the interpretation of the address specified by ptr will differ. For details, refer to the ALLEGREX manual.

__builtin_allegrex_round_w_s
 [Function]
Round

Syntax

int __builtin_allegrex_round_w_s (
	float val
);
Description

This function performs neighborhood rounding even if val is an approximate value or intermediate value and returns the result converted to an integer.

If val is the value which is precisely in between two integers, rounding will be carried out toward the even number, as per the IEEE754 standard.

Operation

round.w.s f0, f12
mfc1 v0, f0

__builtin_allegrex_sqrt_s
 [Function]
Square Root

Syntax

float __builtin_allegrex_sqrt_s (
	float val
);
Description

This function calculates the square root of val and returns the result.

Operation

sqrt.s f0, f12

Restriction

If val is a negative number, no error handling is performed.
To perform error handling, use the standard built-in function sqrtf().
