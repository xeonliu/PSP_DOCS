NP Library Common Reference


SceNpCommunicationId
 [Structure]
NP communication ID structure

Structure

#include <np/np_common.h>

typedef struct SceNpCommunicationId {
	char data[9];
	char term;
	SceUChar8 num;
	char dummy;
} SceNpCommunicationId;
Members

  data     NP communication ID string (4 letters + 5 digits)  
  term     Termination string area for terminating data  
  num     Sub ID (always 0)  
  dummy     Unused  

Description

This structure represents the NP communication ID.

SceNpId
 [Structure]
NP ID structure

Structure

#include <np/np_common.h>

typedef struct SceNpId {
	SceNpOnlineId handle;
	SceUChar8 opt[8];
	SceUChar8 reserved[8];
} SceNpId;
Members

  handle     Online ID  
  opt     Optional data  
  reserved     Area for future expansion  

Description

This structure is used by the NP library to identify the user.
It contains the online ID.

SceNpUserInfo
 [Structure]
User information structure

Structure

#include <np/np_common.h>

typedef struct SceNpUserInfo {
	SceNpId userId;
	SceNpOnlineName name;		/* Online Name */
	SceNpAvatarUrl icon;		/* Avatar URL */
} SceNpUserInfo;
Members

  userId     NP ID  
  name     Online name  
  icon     Avatar URL  

Description

This structure is used to represent a user by collecting together the user's NP ID, online name and avatar URL.

SceNpUserInfo2
 [Structure]
User information structure (pointer version)

Structure

#include <np/np_common.h>

typedef struct SceNpUserInfo2 {
	SceNpId npId;
	SceNpOnlineName *onlineName;		/* (pointer) Online Name */
	SceNpAvatarUrl *avatarUrl;		/* (pointer) Avatar URL */
} SceNpUserInfo2;
Members

  npId     NP ID  
  onlineName     Pointer to online name structure  
  avatarUrl     Pointer to avatar URL structure  

Description

This structure collects together user information.
It differs from SceNpUserInfo in that the online name and avatar URL are represented by pointers.

SceNpMyLanguages
 [Structure]
Frequently used language structure

Structure

#include <np/np_common.h>

typedef struct SceNpMyLanguages{
	SceInt32 language1;	/* It will always be set */
	SceInt32 language2;	/* Negative when not set */
	SceInt32 language3;	/* Negative when not set */
	SceUChar8 padding[4];
} SceNpMyLanguages;
Members

  language1     Language setting 1 (always set)  
  language2     Language setting 2 (negative value when not set)  
  language3     Language setting 3 (negative value when not set)  
  padding     Unused  

Description

This structure collects together "frequently used languages" set by the user.
language1, language2, and language3 should each be set with the following values.
If no value is set, a negative value will be set.

SceNpCountryCode
 [Structure]
Country code structure

Structure

#include <np/np_common.h>

typedef struct SceNpCountryCode {
	char data[2];
	char term;
	char padding[1];
} SceNpCountryCode;
Members

  data     Country code (ISO 3166-1)  
  term     Termination string area for terminating data  
  padding     Unused  

Description

This structure represents the country code that is saved in the region code of the NP ticket parameters.
data is a 2-byte ASCII code representing a country as defined by ISO 3166-1.
