libgameupdate


SceGameUpdateResult
 [構造体]
アップデータ情報構造体

構造

#include <libgameupdate.h>

#define SCE_GAMEUPDATE_APP_VER_SIZE	6

typedef struct SceGameUpdateResult {
	unsigned int size;	/* sizeof(SceGameUpdateResult) */
	int status;
	char app_ver[SCE_GAMEUPDATE_APP_VER_SIZE];
	char padding[2];
} SceGameUpdateResult;
メンバ

  size     構造体サイズ  
  status     ステータスコード（SCE_GAMEUPDATE_RESULT_STATUS_XXX）  
  app_ver     最新のアプリケーションバージョン（"XX.XX"）  
  padding     未使用  

解説

アップデータ情報を格納して返すための構造体です。

解説

アップデートチェックを実行します。
アップデートサーバーに接続し、結果取得が完了するまで関数はブロックします。正常終了の場合、結果はresultに返されます。
sceGameUpdateAbort()で処理を中断した場合は、エラーコードにSCE_GAMEUPDATE_ERROR_ABORTEDが返されます。

使用例

int ret;
SceGameUpdateResult result;

memset(&result, 0, sizeof(result));
result.size = sizeof(result);

ret = sceGameUpdateRun(&result);
if (ret < 0) {
	// エラー処理
}
備考

resultで示されるアップデータ情報構造体のsizeメンバに予め構造体のサイズ（sizeof(SceGameUpdateResult)）を設定した上で、呼び出してください。
