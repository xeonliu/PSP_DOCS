sceWlanGetEtherAddr
 [Function]
Get Ethernet address

Syntax

#include <wlan.h>

int sceWlanGetEtherAddr(
	struct SceWlanEtherAddr *addr
	);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Multithread safe

Arguments

  addr     Pointer to Ethernet address structure  

Return Value

When the function completes normally, zero is returned.
If an error occurs, an error code is returned.

Description

This function gets the Ethernet address of the wireless LAN device.
It does not depend on the power state of the wireless LAN device.

Example


struct SceWlanEtherAddr addr;
int ret = 0;

ret = sceWlanGetEtherAddr(&addr);
if(ret < 0){
	/* Error handling */
}

sceWlanGetSwitchState
 [Function]
Get wireless LAN switch state

Syntax

#include <wlan.h>

int sceWlanGetSwitchState(
	void
	);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

None

Return Value

If the wireless LAN switch is OFF, SCE_WLAN_SWITCH_STATE_OFF is returned.
If the wireless LAN switch is ON, SCE_WLAN_SWITCH_STATE_ON is returned.

Description

This function gets the wireless LAN switch state.

Example


int ret = 0;

ret = sceWlanGetSwitchState();
