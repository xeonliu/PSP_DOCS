sceNetSetDropRate
 [パケット棄却確率、期間の設定/取得]
パケット棄却確率、期間の設定

形式

#include <pspnet.h>

int
sceNetSetDropRate(
	SceUInt rate,
	SceUInt duration
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  rate     パケット棄却確率  
  duration     パケット棄却期間  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_INVALID_ARG

解説

PSPNETに、パケット棄却確率と棄却期間を設定します。
rateはパケット棄却確率を表し、パーセンテージで0〜100の値です。
durationはパケット棄却期間を表し、秒単位です。

使用例

int ret = 0;

/* 5秒間の棄却期間を 10% の確率で発生させる */
ret = sceNetSetDropRate(10, 5);
if(ret < 0){
	/* エラー処理 */
}
関連項目

sceNetGetDropRate()

sceNetGetDropRate
 [パケット棄却確率、期間の設定/取得]
パケット棄却確率、期間の取得

形式

#include <pspnet.h>

int
sceNetGetDropRate(
	SceUInt *rate,
	SceUInt *duration
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  rate     パケット棄却確率を格納する領域  
  duration     パケット棄却期間を格納する領域  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_INVALID_ARG

解説

PSPNETに設定されているパケット棄却確率と棄却期間を取得します。
パケット棄却確率はrateに、パケット棄却期間はdurationに格納されます。

使用例

int ret = 0;
SceUInt rate, duration;

ret = sceNetGetDropRate(&rate, &duration);
if(ret < 0){
	/* エラー処理 */
}
printf("rate = %d, duration = %d\n", rate, duration);
