Description

This function executes update checking.
This function blocks until a connection to the update server is established and the result has been obtained. If the function completes normally, the result will be returned in result.
If the function is interrupted by sceGameUpdateAbort(), SCE_GAMEUPDATE_ERROR_ABORTED will be returned as the error code.

Example

int ret;
SceGameUpdateResult result;

memset(&result, 0, sizeof(result));
result.size = sizeof(result);

ret = sceGameUpdateRun(&result);
if (ret < 0) {
	// Error handling
}
Note

Before calling this function, set the size of the structure (sizeof(SceGameUpdateResult)) in the size member of the updater information structure pointed to by result.
