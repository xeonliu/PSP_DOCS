sceSslInit
 [Function]
Initialize libssl

Syntax

int sceSslInit ( 
	SceSize pool_size
);
Calling Conditions

Canbe called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Not multithread safe

Arguments

  pool_size     Size of heap memory to be used during SSL communications  

Return Value

  Value     Macro     Meaning  
  0     SCE_OK     Normal completion  
  0x80435020     SCE_SSL_ALREADY_INITED     sceHttpInit() was called a second time without calling sceHttpEnd()  
  0x804351FE     SCE_SSL_INVALID_VALUE     Zero was specified for the pool_size  


Description

This function initializes libssl. It must be called before libhttp is to be used to perform https communication. pool_size specifies the heap size used for SSL communications.

Example

#define SSL_POOLSIZE 	(150 * 1024U)
ret = sceSslInit(SSL_POOLSIZE);
if(ret < 0){
	/* Error handling */
}
See Also

sceSslGetSerialNumber
 [Function]
Get certificate serial number

Syntax

#include <libssl.h>
int sceSslGetSerialNumber ( 
	const SceSslCert *ssl_cert,
	const SceUChar8 **sbo_data,
	SceSize *sbo_len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  ssl_cert     Pointer to the certificate for which the serial number is to be obtained.  
  The pointer to the certificate is obtained using the callback function set by sceHttpsSetSslCallback(). (IN)  
  sbo_data     Specifies memory location where the starting address of the serial number will be stored.   
  Serial numbers are stored in big-endian format (OUT).  
  sbo_len     Specifies the memory address where the length of the serial number is stored (OUT).  

Return Value

  Value     Macro     Meaning  
  0     SCE_OK     Normal completion  
  0x80435001     SCE_SSL_BEFORE _INITED     Library not initialized  
  0x804351FE     SCE_SSL_INVALID_VALUE     NULL was specified for ssl_cert, sbo_data or sbo_len.  


Description

This function is used for obtaining the serial number of a certificate object. The certificate object is obtained as an argument of the callback function set by sceHttpsSetSslCallback() in the HTTP library. The memory area where the serial number is stored is freed immediately after the callback function ends, so be sure to copy the serial number to a separate memory area within the callback function if you will need to reference it later.

See Also

sceHttpsSetSslCallback()

sceSslGetSubjectName
 [Function]
Get subject name of certificate

Syntax

#include <libssl.h>
const SceSslCertName *sceSslGetSubjectName ( 
	const SceSslCert *ssl_cert
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  ssl_cert     Pointer to the certificate for which the subject name is to be obtained.  
  The pointer to the certificate is obtained using the callback function set by sceHttpsSetSslCallback(). (IN)  

Return Value

If the function completes normally, a pointer to the SceSslCertName object is returned. If the subject name cannot be obtained, NULL is returned.

Description

This function is used for obtaining the name object of the subject of a certificate object. The certificate object is obtained as an argument of the callback function set by sceHttpsSetSslCallback() in the HTTP library. To obtain details on the subject name object, first get the number of subject name entries using sceSslGetNameEntryCount(), and then use sceSslGetNameEntryInfo() to get information for each name entry.

See Also

sceHttpsSetSslCallback()

sceSslGetNameEntryInfo
 [Function]
Get name entry information

Syntax

#include <libssl.h>
int sceSslGetNameEntryInfo ( 
	const SceSslCertName *cert_name,
	int entry_num,
	const SceUChar8 **oidname,
	const SceUChar8 **value,
	SceSize *value_len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  cert_name     Pointer to the SSL name object for which the entry is to be obtained. (IN)  
  entry_num     Number of the entry to be obtained. The number of entries stored in the SSL  
  name object can be obtained using sceSslGetNameEntryCount() (IN).  
  oidname     An ASCIZ string representing the entry's object ID. If set to NULL, it signifies  
  an unknown object ID (OUT).  
  value     Memory location where the starting address of the entry values will be stored. Not in ASCIZ format (OUT).  
  value_len     Memory address where the size of value is stored (OUT).  

Return Value

Positive value Number of name entries contained in cert_name.
  Value     Macro     Meaning  
  0     SCE_OK     Normal completion  
  0x80435001     SCE_SSL_BEFORE _INITED     Library not initialized  
  0x804351FE     SCE_SSL_INVALID_VALUE     NULL was specified foroid_name, value, or value_len.  


Description

This function is used for obtaining the number of name entries contained in an SSL name object. The SSL name object can be obtained by using functions such as sceSslGetSubjectName() or sceSslGetIssuerName(). The memory used to store the value fetched using this function is freed as soon as the callback function set by sceHttpsSetSslCallback() ends, so be sure to copy the value to a separate memory area within the callback function if you wish to save it for later use.

See Also

sceHttpsSetSslCallback()
