libqpcode

sceQPrintableEncoder
 [関数]
quoted-printableエンコード

形式

#inclue <libqpcode.h>
int sceQPrintableEncoder (
	SceChar8 *out,
	const SceUChar8 *in,
	SceSize src_len,
	SceSize *require,
	SceSize prepare
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能(割り込み禁止/許可状態に依存しない)
マルチスレッドセーフである

引数

  in     入力バイト列へのポインタ  
  out     出力バイト列へのポインタ  
  src_len     入力の長さ  
  require     出力バイト数を格納するメモリへのポインタ  
  prepare     出力バイト列に用意したメモリのサイズ  

返り値

  値     マクロ     意味  
  0     SCE_OK     正常終了  
  0x80434022     SCE_QPCODE_OUT_OF_MEMORY     出力に必要なバイト数が  
  prepareで指定された値を超えた  
  0x804341FE     SCE_QPCODE_INVALID_VALUE     outとrequireの両方ともに  
  NULLがセットされた  


解説

inとsrc_lenで指定された入力バイト列をQPrintableエンコードして、outで指定されたメモリ領域へ出力し、出力バイト数をrequireに格納します。出力バイト数がprepareを超えるときはエラーが返ります。出力用のメモリ領域はアプリケーションが用意しなければなりません。outにNULLを指定することで、出力に必要なメモリ領域の大きさがrequireにセットされ取得できます。

sceQPrintableLineDecoder
 [関数]
quoted-printable行デコード

形式

#inclue <libqpcode.h>
int sceQPrintableLineDecoder (
	SceChar8 *out,
	const SceUChar8 *in,
	SceSize src_len,
	SceSize *require,
	SceSize prepare
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能(割り込み禁止/許可状態に依存しない)
マルチスレッドセーフである

引数

  out     出力バイト列へのポインタ  
  in     入力バイト列へのポインタ  
  src_len     入力の長さ  
  require     出力バイト数を格納するメモリへのポインタ  
  prepare     出力バイト列に用意したメモリのサイズ  

返り値

  値     マクロ     意味  
  0     SCE_OK     正常終了  
  0x80434022     SCE_QPCODE_OUT_OF_MEMORY     出力に必要なバイト数が  
  prepareで指定された値を超えた  
  0x804341FE     SCE_QPCODE_INVALID_VALUE     outとrequireの両方ともに  
  NULLがセットされた  
  0x80434108     SCE_QPCODE_INVALID_FORMAT     不正なフォーマットのバイト列が  
  inに指定された  


解説

inとsrc_lenとで指定された入力バイト列（quoted-printableエンコードされたデータの1行分）をデコードして、outで指定されたメモリ領域へ出力し、出力バイト数をrequireに格納します。出力バイト数がprepareを超えるときはエラーが返ります。入力バイト列は、終端のRFC822改行（CRとLFの連続）を含めて長さが78かそれ以下でなければいけません。ilenにこれ以上の値を設定しても無視されます。出力用のメモリ領域はアプリケーションが用意しなければなりません。outにNULLを指定することで、出力に必要なメモリ領域の大きさがrequireにセットされ取得できます。
