int sceNpCommerce2GetPrice(
	const SceNpCommerce2SessionInfo *s,
	char *buf,
	SceSize buflen,
	SceUInt32 price
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  s     セッション情報構造体 [IN]  
  buf     取得した価格文字列を格納するバッファ領域 [IN]  
  buflen     bufの指す領域のサイズ [IN]  
  price     SKU情報構造体から取得した価格 [IN]  

返り値

  値     説明  
  0     正常終了  
  負の値     エラー（「エラーコード一覧」節を参照）  


解説

SKU情報構造体のpriceメンバは価格を意味しますが、一般的には価格を表示する際には、通貨を価格の前後どちらに付けるか、また小数点はカンマかピリオドかなど、国/地域ごとの違いが存在します。
本関数は、この国/地域ごとの違いを加味した価格表現を提供する関数です。ここで取得した価格表現文字列はアプリケーション側でそのまま表示に使うことができます。
本関数は、アカウントの国/地域ごとの情報をセッション作成時にサーバーから取得していることが前提のため、事前にセッションを作成している必要があります。

備考

価格（数値）を価格フォーマット（文字列）に変換する際は、本関数の使用をお勧めします。
ユーザに対して価格を表示する際に御利用下さい。
