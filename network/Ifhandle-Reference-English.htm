sceNetSetDropRate
 [Setting/Getting Packet Drop Rate and Duration]
Set packet drop rate and duration

Syntax

#include <pspnet.h>

int
sceNetSetDropRate(
	SceUInt rate,
	SceUInt duration
	);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Multithread safe

Arguments

  rate     Packet drop rate  
  duration     Packet drop duration  

Return Value

If the function completes normally, 0 is returned.
If an error occurs, the following error code is returned.
SCE_ERROR_NET_INVALID_ARG

Description

This function sets the packet drop rate and drop duration for PSPNET.
rate, which represents the packet drop rate, is a percentage value from 0 to 100.
duration, which represents the packet drop duration, is in units of seconds.

Example

int ret = 0;

/* Drop packets for 5 seconds with a 10% drop rate */
ret = sceNetSetDropRate(10, 5);
if(ret < 0){
	/* Error handling */
}

sceNetGetDropRate
 [Setting/Getting Packet Drop Rate and Duration]
Get packet drop rate and duration

Syntax

#include <pspnet.h>

int
sceNetGetDropRate(
	SceUInt *rate,
	SceUInt *duration
	);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Multithread safe

Arguments

  rate     Area for storing the packet drop rate  
  duration     Area for storing the packet drop duration  

Return Value

If the function completes normally, 0 is returned.
If an error occurs, the following error code is returned.
SCE_ERROR_NET_INVALID_ARG

Description

This function gets the packet drop rate and drop duration that have been set for PSPNET.
The packet drop rate is stored in rate and the packet drop duration is stored in duration.

Example

int ret = 0;
SceUInt rate, duration;

ret = sceNetGetDropRate(&rate, &duration);
if(ret < 0){
	/* Error handling */
}
printf("rate = %d, duration = %d\n", rate, duration);
