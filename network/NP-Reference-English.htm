Get this user's region

Syntax

int sceNpGetAccountRegion(
	SceNpCountryCode *countryCode,
	int *languageCode
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Multithread safe

Arguments

  countryCode     Country code storage destination  
  languageCode     Language code storage destination  

Return Value

When the function completes normally, 0 is returned.
If an error occurs, an error code is returned.

Description

This function gets this user's region information, which is obtained when signing in to the NP.
The function stores the country code in countryCode and the language code in languageCode and then returns.
The values stored in languageCode are the SCE_UTILITY_LANG_xxx values defined in utility/utility_common.h.

sceNpGetContentRatingFlag
 [Getting User's Own Information]
Get parental control information for this user's age restriction

Syntax

#include <np/np.h>

int sceNpGetContentRatingFlag(
	int *isRestricted,
	int *age
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Multithread safe

Arguments

  isRestricted     Storage destination of flag indicating whether content is restricted (true if it is restricted)  
  age     Storage destination of this user's age that is set for the account  

Return Value

When the function completes normally, 0 is returned.
If an error occurs, an error code is returned.

Description

This function gets this user's parental control information, which is obtained when signing in to the NP.
If the value that is obtained in isRestricted is true, it indicates that an age-based content usage restriction is effective for this account.

Example


int isRestricted, age;
int ret = 0;

ret = sceNpGetContentRatingFlag(&isRestricted, &age);
if(ret < 0){
	/* Error handling*/
}
#define CONTENT_RATING_AGE 18 //Age restriction
if (isRestricted && age < CONTENT_RATING_AGE) {
	//Content viewing is restricted
}
