 [定数定義]

定義

#include <wlan.h>

#define SCE_WLAN_SWITCH_STATE_OFF 0

関連項目

sceWlanGetEtherAddr()

sceWlanGetEtherAddr
 [関数]
イーサネットアドレスを取得する

形式

#include <wlan.h>

int sceWlanGetEtherAddr(
	struct SceWlanEtherAddr *addr
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能 (割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  addr     イーサネットアドレス構造体へのポインタ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、エラーコードを返します。

解説

無線LANデバイスのイーサネットアドレスを取得します。
無線LANデバイスの電源状態に依りません。

使用例


struct SceWlanEtherAddr addr;
int ret = 0;

ret = sceWlanGetEtherAddr(&addr);
if(ret < 0){
	/* エラー処理 */
}

sceWlanGetSwitchState
 [関数]
ワイヤレスLANスイッチの状態を取得する

形式

#include <wlan.h>

int sceWlanGetSwitchState(
	void
);
呼出条件

割り込みハンドラから呼び出し可
スレッドから呼び出し可（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

なし

返り値

ワイヤレスLANスイッチがOFFの場合は、SCE_WLAN_SWITCH_STATE_OFFを返します。
ワイヤレスLANスイッチがONの場合は、SCE_WLAN_SWITCH_STATE_ONを返します。

解説

ワイヤレスLANスイッチの状態を取得します。

使用例


int ret = 0;

ret = sceWlanGetSwitchState();
