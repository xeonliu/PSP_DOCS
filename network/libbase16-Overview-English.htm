libbase16 Overview

1 Library Overview

1. 1 Overview

libbase16 is a library for performing BASE16-format encoding/decoding, as defined in RFC4648.

2 Using the Library

2. 1 Basic Usage Procedure

No specific initialization is required to use libbase16.

Encoding to BASE16 format
The following procedure can be used to encode a byte sequence into BASE16 format. Note that since libbase16 does not allocate memory internally, the application must allocate memory for the output beforehand. The required memory size is ((number of input bytes) * 2 + 1) bytes.

u_char	plain[SOURCE_LEN];
u_char	*base16;
SceSize malloc_size;

malloc_size = 	SOURCE_LEN * 2 + 1;		// Calculate output size 
base16 = (u_char*)malloc(malloc_size);		// Allocate memory for output
sceBase16Encoder(base16, plain, SOURCE_LEN);	// Encode
BASE16 encoding is performed by calling the sceBase16Encoder() function as shown above.

Decoding from BASE16 format
The following procedure can be used to decode a byte sequence that was previously encoded into BASE16 format. Note that since libbase16 does not allocate memory internally, the application must allocate memory for the output in advance. The required memory size is ((number of input bytes) / 2 + 1) bytes.

u_char	base16[SOURCE_LEN];
u_char	*plain;
SceSize malloc_size;

malloc_size = SOURCE_LEN / 2 + 1;		// Calculate output size
plain = (u_char*) malloc(malloc_size);		// Allocate memory for output
sceBase16Decoder (plain, base16, SOURCE_LEN);	// Decode

BASE16 decoding is performed by calling the sceBase16Decoder() function as shown above.
