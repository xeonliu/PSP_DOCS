pspnet_adhoc_discover

SceNetAdhocDiscoverParam
 [構造体]
パラメータ構造体

構造

#include <pspnet_adhoc_discover.h>

struct SceNetAdhocDiscoverParam {
	int mode;
	struct SceNetAdhocctlGroupName group_name;
	unsigned int timeout;
	int result;
};
メンバ

  mode     探索モード  
  group_name     グループ名  
  timeout     接続フェーズのタイムアウト（秒）  
  result     返り値  

解説

アドホックディスカバライブラリの設定パラメータです。
resultは、ステータスがSCE_NET_ADHOC_DISCOVER_STATUS_FINISHEDになったときに参照します。resultの値は、近傍端末探索の成否を示す値、またはアドホックディスカバライブラリ動作中に発生したpspnet_adhocctlのERRORイベントのエラーコードです。

sceNetAdhocDiscoverInitStart()

sceNetAdhocDiscoverInitStart
 [関数]
初期化・起動

形式

#include <pspnet_adhoc_discover.h>

int sceNetAdhocDiscoverInitStart(
	struct SceNetAdhocDiscoverParam *param
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  param     設定パラメータへのポインタ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_ADHOC_DISCOVER_BUSY
SCE_ERROR_NET_ADHOC_DISCOVER_INVALID_ARG

解説

アドホックディスカバライブラリを初期化・起動します。
正常終了の場合、ステータスはSCE_NET_ADHOC_DISCOVER_STATUS_RUNNINGに移行します。

使用例

int ret = 0;
struct SceNetAdhocDiscoverParam param;
struct SceNetAdhocctlGroupName group_name;

// group_nameには適当なグループ名が設定されているものとします

memset(&param, 0, sizeof(struct SceNetAdhocDiscoverParam));
param.mode = SCE_NET_ADHOC_DISCOVER_MODE_WOL;
memcpy(&param.group_name, &group_name, sizeof(group_name));
param.timeout = DISCOVER_TIMEOUT;
param.result = 0;

ret = sceNetAdhocDiscoverInitStart(&param);
if (ret < 0) {
    /* エラー処理 */
}

sceNetAdhocDiscoverRequestSuspend
 [関数]
サスペンド要求を発行

形式

#include <pspnet_adhoc_discover.h>

int sceNetAdhocDiscoverRequestSuspend (
	void
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

なし

返り値

正常終了の場合、０を返します。
サスペンド要求条件を満たさない場合、以下のエラーコードを返します。
SCE_ERROR_NET_ADHOC_DISCOVER_NOT_ALLOWED_SUSPEND

解説

アプリケーションからの要求によってPSP®をサスペンド状態にする関数です。
この関数の呼び出しによりサスペンドさせるのは、事前にWoLモード探索によって端末を発見し、sceNetAdhocDiscoverTerm()によってライブラリの終了処理が完了している場合に限ります。

使用例

int ret;

ret = sceNetAdhocDiscoverRequestSuspend();
if (ret < 0) {
    // エラー処理
}
