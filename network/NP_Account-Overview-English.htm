PlayStation®Network Account Service Overview

1 Account

Each user on the PlayStation®Network ("NP") is allocated with an account. Services of the PlayStation®Network can be used while signed onto the PlayStation®Network with this allocated account.

1. 1 Signup

Signup refers to the creation of a new PlayStation®Network account.
Signup can be carried out from system software using the "Sign Up for PlayStation®Network" icon under "PlayStation®Network". If signup is successful, the account that was created is bound.

Note
An account can be created from a Web browser on a PC. In that case, a previously created account is bound from the "Sign Up for PlayStation®Network" icon.
To bind a different account after an account has been bound, use the system software "Account Management" icon under "PlayStation®Network" and select "Delete account" from the options menu.

1. 2 Master Account and Sub Account

The PlayStation®Network account makes a distinction between the master account and the sub account.
A master account refers to an account that is created by users above a certain age (standards for this age level differ by country or region).
A sub account refers to an account that is issued to users who are considered underage. A sub account is issued with the understanding that the user of the master account will supervise the PlayStation®Network usage of the sub account. Thus, the master account user is able to set usage limitations on several of the functions of the sub account.

1. 3 Sign-in

Sign-in refers to the process of connecting to the PlayStation®Network. To use the PlayStation®Network from a title, first connect to the network and then call the sign-in utility to perform sign-in processing. The sign-in status is valid until the valid period of the sign-in ticket has elapsed, or until you disconnect from the network.

1. 4 Account Information

Basic information regarding an account includes the following.

Account ID
The Account ID is the primary key used in identifying accounts. It is used for management purposes by the system and is not visible to the user. One Account ID will be used for the entire time that an account is valid.

Online ID (SceNpOnlineId)
The Online ID is used to identify accounts, and is chosen by the user upon signup. It is a 3- to 16-character (A - Z, a-z, 0-9) string composed of alphanumeric characters, hyphens, and/or underscores. The Online ID is guaranteed to be unique.
In principle, one Online ID will be used for the entire time that an account is valid, but it can be changed by a request from the system or the user.

NP ID (SceNpId)
The NP ID consists of the Online ID, with option fields and version information that are necessary for server access. Each NP system utility uses this NP ID to identify a user.

Region (SceNpCountryCode)
Region refers to the country and region of residence selected by the user upon signup. The region cannot be changed after the signup. For a list of countries and regions that can be selected, see the Description of sceNpGetAccountRegion() in the np Library Reference.

Age
The user's age is calculated from the birth date input by the user upon signup. The birth date cannot be changed after signup.

Content Rating Flag
This flag represents the user's parental control setting for age-based content restriction. The user with the master account (supervisor) can set or make changes to this flag setting for its sub account.

Chat Restriction Flag
This flag represents the user's parental control setting for chat restriction. The user with the master account (supervisor) can set or make changes to this flag setting for its sub account.

1. 5 Player Profile

Online Name (SceNpOnlineName)
The Online Name can be set by the user. It consists of a 3- to 16-character string (UTF-encoding), and characters (excluding numeric characters) of multiple languages can be used. Uniqueness is not guaranteed. When the system software is to display a user name, this Online Name will be displayed. The default is for the Online Name to be the same as the Online ID. Depending on the territory targeted by the application, it may not be possible to modify the Online Name from the default setting.
The Online Name supports characters of multiple languages. When displaying it using the application, check whether the characters to be displayed are supported by the application. If an unsupported character exists, use a replacement or display the Online ID instead of the Online Name.

Avatar URL (SceNpAvatarUrl)
The avatar URL is the URL for the user's image. The user image is selected beforehand by the user from a set of provided images (the user is not allowed to provide an image).

Self Introductory Note (SceNpAboutMe)
This is a self introductory note describing the user. Multiple languages can be used. The text can consist of up to 21 characters (UTF-8 encoding). When displaying it using the application, make sure to check for unsupported characters in the same manner as the Online Name.

Languages Often Used (SceNpMyLanguages)
These are languages used by the user. Up to 3 languages can be selected. A selection can be made from the following (including those that are planned to be included).
Japanese, English, French, Spanish, German, Italian, Dutch, Portuguese, Russian, Korean, Chinese (traditional), Chinese (simplified), Finnish, Swedish, Danish, Norwegian

2 Client Libraries

The client libraries for using the PlayStation®Network account service from the application are provided as system utilities. The following libraries are currently provided.
For details on functionalities, please refer to the overview and reference documents for each library.

2. 1 NP Sign-in Utility

This system utility displays a sign-in dialog and performs sign-in processing.
  • Get sign-in ticket
  • Get account information of signed-in user
  • Get player profile of signed-in user

2. 2 NP Library

This library provides functions related to the account information and player profile of the user who is currently signed in.

2. 3 NP Service Library

This library provides functions related to the updating of sign-in tickets.

2. 4 NP Auth Library

This library provides functions for acquiring a ticket for which authentication is required using the Ticket Checker Module (TCM) on a title-specific server.

3 Precautions

3. 1 Display of the Online Name

The Online Name and self-introductory note of the player profile information support multiple languages. Note that these may contain characters that cannot be displayed by the application.
When displaying player profile information using the application, check whether the characters to be displayed are supported by the application. If an unsupported character exists, replace it with a symbol such as "_" to display the information, or display the Online ID instead of the Online Name.

3. 2 Signing In Under the Same Account Multiple Times

A user is free to sign in to the same account multiple times using different PSP™ systems, however, unexpected errors may occur for some services.

Errors occur in the following services.

Matching 2
Refer to the section "Simultaneous Sign-In Support" in the "Matching 2 Overview" document.
