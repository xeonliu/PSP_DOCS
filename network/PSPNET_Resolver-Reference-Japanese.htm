sceNetResolverStartNtoA
 [関数]
正引き名前解決の開始

形式

#include <pspnet_resolver.h>
#include <pspnet/netinet/in.h>

int sceNetResolverStartNtoA(
	int rid,
	const char *hostname,
	struct SceNetInetInAddr *addr,
	SceUInt32 timeout,
	int retry
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  rid     リゾルバID  
  hostname     名前解決対象のホスト名  
  addr     ホスト名に対応するIPアドレスを格納する領域へのポインタ  
  timeout     問い合わせ再送間隔  
  retry     問い合わせ再送回数  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_RESOLVER_INVALID_ID
SCE_ERROR_NET_RESOLVER_INVALID_PTR
SCE_ERROR_NET_RESOLVER_ID_NOT_FOUND
SCE_ERROR_NET_RESOLVER_CTX_BUSY
SCE_ERROR_NET_RESOLVER_NO_DNS_SERVER
SCE_ERROR_NET_RESOLVER_BUF_NO_SPACE
SCE_ERROR_NET_RESOLVER_INTERNAL
SCE_ERROR_NET_RESOLVER_STOPPED
SCE_ERROR_NET_RESOLVER_TIMEOUT
SCE_ERROR_NET_RESOLVER_SOCKET
SCE_ERROR_NET_RESOLVER_INVALID_PACKET
SCE_ERROR_NET_RESOLVER_NOT_SUPPORTED
SCE_ERROR_NET_RESOLVER_RES_PACKET_FORMAT
SCE_ERROR_NET_RESOLVER_RES_SERVER_FAILURE
SCE_ERROR_NET_RESOLVER_NO_HOST
SCE_ERROR_NET_RESOLVER_RES_NOT_IMPLEMENTED
SCE_ERROR_NET_RESOLVER_RES_SERVER_REFUSED
SCE_ERROR_NET_RESOLVER_NO_RECORD

解説

ridで指定されたリゾルバコンテキストを使用して、正引き名前解決を開始します。
hostnameで指定されたホスト名に対応するIPアドレスを問い合わせ、addrで指定された領域に格納します。
名前解決は呼び出しスレッドが行ない、本関数は名前解決が正常終了またはエラーになるまでブロックします。

使用例

#define SERVER_NAME "www.scei.co.jp"
#define RESOLVER_TIMEOUT (5 * 1000 * 1000)
#define RESOLVER_RETRY 5

struct SceNetInetInAddr addr;

ret = sceNetResolverStartNtoA(rid, SERVER_NAME, &addr, RESOLVER_TIMEOUT, RESOLVER_RETRY);
if(ret < 0){
	/* エラー処理 */
}

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  rid     リゾルバID  
  hostname     名前解決対象のホスト名  
  addr     ホスト名に対応するIPアドレスを格納する領域へのポインタ  
  timeout     問い合わせ再送間隔

	/* エラー処理 */
}

sceNetResolverStartNtoAAsync()、sceNetResolverStop()、sceNetResolverPollAsync()、sceNetResolverWaitAsync()
sceNetResolverPollAsync
 [関数]
ノンブロッキング名前解決の状況取得

形式

#include <pspnet_resolver.h>
#include <pspnet/netinet/in.h>

int sceNetResolverPollAsync(
	int rid,
	void *result,
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  rid     リゾルバID  
  result     未使用  

返り値

名前解決が完了していない場合1を返し、完了していれば0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_RESOLVER_ID_NOT_FOUND
SCE_ERROR_NET_RESOLVER_INVALID_MODE
SCE_ERROR_NET_RESOLVER_NOT_STARTED
SCE_ERROR_NET_RESOLVER_NO_DNS_SERVER
SCE_ERROR_NET_RESOLVER_BUF_NO_SPACE
SCE_ERROR_NET_RESOLVER_INTERNAL
SCE_ERROR_NET_RESOLVER_STOPPED
SCE_ERROR_NET_RESOLVER_TIMEOUT
SCE_ERROR_NET_RESOLVER_SOCKET
SCE_ERROR_NET_RESOLVER_INVALID_PACKET
SCE_ERROR_NET_RESOLVER_NOT_SUPPORTED
SCE_ERROR_NET_RESOLVER_RES_PACKET_FORMAT
SCE_ERROR_NET_RESOLVER_RES_SERVER_FAILURE
SCE_ERROR_NET_RESOLVER_NO_HOST
SCE_ERROR_NET_RESOLVER_RES_NOT_IMPLEMENTED
SCE_ERROR_NET_RESOLVER_RES_SERVER_REFUSED
SCE_ERROR_NET_RESOLVER_NO_RECORD

解説

ridで指定されたリゾルバコンテキストで開始したノンブロッキング名前解決の状況を取得します。事前にsceNetResolverStartNtoAAsync()もしくはsceNetResolverStartAtoNAsync()が呼ばれていないとSCE_ERROR_NET_RESOLVER_INVALID_MODEエラーになります。
本関数はノンブロッキング関数です。

使用例

#define SERVER_NAME "www.scei.co.jp"
#define RESOLVER_TIMEOUT (5 * 1000 * 1000)
#define RESOLVER_RETRY 5
#define DELAY_TIME (1000 * 1000)

struct SceNetInetInAddr addr;

/* ノンブロッキング名前解決開始 */
ret = sceNetResolverStartNtoAAsync(rid, SERVER_NAME, &addr, RESOLVER_TIMEOUT, RESOLVER_RETRY);
if(ret < 0){
	/* エラー処理 */
}

/* 名前解決の完了待ちループ */
while(1){
	ret = sceNetResolverPollAsync(rid, NULL);
	if(ret < 0){
		/* エラー処理 */
	}
	else if(ret == 0){
		/* 名前解決完了 */
		break;
	}
	sceKernelDelayThread(DELAY_TIME);
}

/* この時点では名前解決が完了している */
