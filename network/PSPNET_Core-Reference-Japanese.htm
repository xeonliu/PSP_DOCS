sceNetInit
 [関数]
PSP®ネットワークライブラリ初期化

形式

#include <pspnet.h>

int sceNetInit(
	SceSize poolsize,
	int callout_tpl,
	SceSize callout_stack,
	int netintr_tpl,
	SceSize netintr_stack
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフでない

引数

  poolsize     pspnet全体で使用するメモリプールサイズ  
  callout_tpl     内部スレッドの1つであるcalloutスレッドの優先度  
  callout_stack     未使用（0を指定）  
  netintr_tpl     内部スレッドの1つであるnetintrスレッドの優先度  
  netintr_stack     未使用（0を指定）  

返り値

正常終了の場合は0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_CORE_NOT_TERMINATED

解説

pspnetライブラリの初期化を行います。
本関数内部で、2つのライブラリ内部スレッドを作成します。

使用例

#define PSPNET_POOLSIZE (512 * 1024)
#define CALLOUT_SPL 30
#define NETINTR_SPL 30

ret = sceNetInit(PSPNET_POOLSIZE, CALLOUT_SPL, 0, NETINTR_SPL, 0);
if(ret < 0){
	/* エラー処理 */
}
備考

calloutスレッドおよびnetintrスレッドのスタックサイズは、それぞれ4096バイトとなります。

sceNetTerm
 [関数]
PSP®ネットワークライブラリ終了

形式

#include <pspnet.h>

int sceNetTerm(
	void
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフでない

引数

なし

返り値

正常終了の場合は0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_CORE_INTERFACE_BUSY

解説

pspnetライブラリの終了処理を行ないます。
ネットワーク使用中の場合は、エラーとなります。
本関数を呼び出す前にネットワークから切断しておく必要があります。

使用例

ret = sceNetTerm();
if(ret < 0){
	/* エラー処理 */
}

sceNetThreadAbort
 [関数]
スレッドのブロッキングを解除

形式

#include <pspnet.h>

int sceNetThreadAbort(
	SceUID tid
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  tid     アボート対象スレッドのスレッドID  

返り値

正常終了の場合は0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_NO_SPACE
SCE_ERROR_NET_CORE_THREAD_NOT_FOUND

解説

アドホックソケットAPI、およびBSDソケットAPI呼び出し中にブロックしているスレッドのブロッキングを解除します。
ブロックを解除された関数はエラーとなり、アドホックソケットAPIの場合は、SCE_ERROR_NET_ADHOC_THREAD_ABORTEDが返ります。
BSDソケットAPIの場合は、sceNetInetGetErrno()でEINTRが返ります。

使用例

int tid = get_target_tid(); /* アボート対象のスレッドIDを取得 */

sceNetThreadAbort(tid);

sceNetEtherNtostr
 [関数]
48ビットイーサネットアドレスから文字列表現アドレスに変換

形式

	const struct SceNetEtherAddr *n,
	char *str
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)

引数

  n     イーサネットアドレス構造体

なし

解説

48ビットイーサネットアドレスを文字列表現(xx:xx:xx:xx:xx:xx)に変換します。

使用例

	/* エラー処理 */
}
sceNetEtherNtostr(&n, str);

printf("Local MAC Addr: %s\n", str);

sceNetGetLocalEtherAddr
 [インタフェース情報取得]
MACアドレスの取得

形式

#include <pspnet.h>

int sceNetGetLocalEtherAddr(
	struct SceNetEtherAddr *ether
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  ether     MACアドレス  

返り値

正常終了の場合は0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_CORE_INVALID_ARG

解説

ネットワークインタフェースのMACアドレスを取得し、etherに格納します。

注意事項
本関数は、ネットワークに接続済の状態でないとMACアドレスが取得できない仕様になっています。
代わりに、接続状態によらずMACアドレスを取得する関数sceWlanGetEtherAddr()が用意されていますので、そちらをお使いください。

使用例

struct SceNetEtherAddr n;
char str[sizeof("XX:XX:XX:XX:XX:XX")];

ret = sceNetGetLocalEtherAddr(&n);
if(ret < 0){
	/* エラー処理 */
}
sceNetEtherNtostr(&n, str);

printf("Local MAC Addr: %s\n", str);
