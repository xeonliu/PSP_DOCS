SceNetAdhocDiscoverParam
 [Structure]
Parameter structure

Structure

#include <pspnet_adhoc_discover.h>

struct SceNetAdhocDiscoverParam {
	int mode;
	struct SceNetAdhocctlGroupName group_name;
	unsigned int timeout;
	int result;
};
Members

  mode     Search mode  
  group_name     Group name  
  timeout     Connection phase timeout (seconds)  
  result     Return value  

Description

This structure contains the Ad hoc Discover library configuration parameters.
result is referenced when the status is SCE_NET_ADHOC_DISCOVER_STATUS_FINISHED. The value of result is a value that indicates whether the search for a neighboring peer succeeded or failed or is the error code of a pspnet_adhocctl ERROR event that occurred during Ad hoc Discover library operation.

sceNetAdhocDiscoverInitStart
 [Function]
Initialization and startup

Syntax

#include <pspnet_adhoc_discover.h>

int sceNetAdhocDiscoverInitStart(
	struct SceNetAdhocDiscoverParam *param
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  param     Pointer to configuration parameters  

Return Value

When the function completes normally, 0 is returned.
If an error occurs, one of the following error codes is returned.
SCE_ERROR_NET_ADHOC_DISCOVER_BUSY
SCE_ERROR_NET_ADHOC_DISCOVER_INVALID_ARG

Description

This function initializes and starts up the Ad hoc Discover library.
When the function completes normally, the status transitions to SCE_NET_ADHOC_DISCOVER_STATUS_RUNNING.

Example

int ret = 0;
struct SceNetAdhocDiscoverParam param;
struct SceNetAdhocctlGroupName group_name;

// Assume that an appropriate group name has been set for group_name

memset(&param, 0, sizeof(struct SceNetAdhocDiscoverParam));
param.mode = SCE_NET_ADHOC_DISCOVER_MODE_WOL;
memcpy(&param.group_name, &group_name, sizeof(group_name));
param.timeout = DISCOVER_TIMEOUT;
param.result = 0;

ret = sceNetAdhocDiscoverInitStart(&param);
if (ret < 0) {
    /* Error handling */
}

sceNetAdhocDiscoverTerm
 [Function]
Termination processing

Syntax

#include <pspnet_adhoc_discover.h>

int sceNetAdhocDiscoverTerm(
	void
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

None

Return Value

When the function completes normally, 0 is returned.
If an error occurs, the following error code is returned.
SCE_ERROR_NET_ADHOC_DISCOVER_BUSY

Description

This function terminates the Ad hoc Discover library.
If the status is SCE_NET_ADHOC_DISCOVER_STATUS_RUNNING, call sceNetAdhocDiscoverStop() first, wait for the status to transition to SCE_NET_ADHOC_DISCOVER_STATUS_FINISHED, and then call this function.

Example

int ret = 0;

ret = sceNetAdhocDiscoverTerm();
if (ret < 0) {
    /* Error handling */
}

sceNetAdhocDiscoverRequestSuspend
 [Function]
Issue a suspend request

Syntax

#include <pspnet_adhoc_discover.h>

int sceNetAdhocDiscoverRequestSuspend (
	void
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

None

Return Value

When the function completes normally, 0 is returned.
If the suspend request conditions are not satisfied, the following error code is returned.
SCE_ERROR_NET_ADHOC_DISCOVER_NOT_ALLOWED_SUSPEND

Description

This function places the PSP™ in suspend state due to a request from the application.
This function should only be used to suspend the PSP™ after a peer is discovered by a WoL mode search and library termination processing has been completed by calling sceNetAdhocDiscoverTerm().

Example

int ret;

ret = sceNetAdhocDiscoverRequestSuspend();
if (ret < 0) {
    // Error handling
}
