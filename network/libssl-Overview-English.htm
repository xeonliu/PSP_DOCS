libssl Overview

1 Library Overview

1. 1 Overview

libssl is a module that is required when you wish to perform HTTPS communication using libhttp. Note that libssl alone cannot be used to perform SSL communication.

2 Usage Procedure

2. 1 Basic Usage Procedure

Load the SSL module set in the module utility API. Call sceSslInit() to initialize libssl after the network library has been initialized but before initializing libhttp.
Call sceSslEnd() to terminate libssl after libhttp has been terminated but before network library termination.
Initialization and termination processing are shown in the example below.


#define SSL_POOL_SIZE	(1024 * 150U)
#define HTTP_POOL_SIZE (1024 * 30U)

/* Network initialization processing */
...
/* Initialize libssl */
ret = sceSslInit(SSL_POOL_SIZE);
/* Initialize libhttp */
sceHttpInit(HTTP_POOL_SIZE);
sceHttpsInit(0, NULL, NULL, NULL)
sceHttpsLoadDefaultCert(SCE_HTTPS_ISSUER_ALL, 0);
...
/* Terminate libhttp */
sceHttpEnd();
/* Terminate libssl */
sceSslEnd();
/* Network termination processing */

3 Precautions

3. 1 Required stack size for API calls

The required stack size for libssl API calls is 1024bytes (SCE_SSL_LEAST_STACK_SIZE). If the stack size is insufficient, the error SCE_ERROR_NET_INSUFFICIENT_STACKSIZE will be returned.
