When the function completes normally, it returns the context ID.

Example

#define MATCHING_MAXNUM	16
#define MATCHING_PORT	1
#define MATCHING_RXBUFLEN	2048
#define HELLO_INTERVAL	5*1000*1000	// 5 sec
#define KEEPALIVE_INTERVAL	10*1000*1000	// 10 sec
#define INIT_COUNT	        3
#define REXMT_INTERVAL	500*1000	// 500 msec

void handler(int id, int event, struct SceNetEtherAddr *peer, int optlen, void *opt);

int ret;
int id;	// Context ID

ret = sceNetAdhocMatchingCreate(
		SCE_NET_ADHOC_MATCHING_MODE_PARENT,
		MATCHING_MAXNUM, MATCHING_PORT,
		MATCHING_RXBUFLEN, HELLO_INTERVAL,
		KEEPALIVE_INTERVAL, INIT_COUNT, REXMT_INTERVAL,
		handler);
if (ret < 0) {
	// Error handling
}
id = ret;
Notes

maxnum is valid only when mode is SCE_NET_ADHOC_MATCHING_MODE_PARENT.
hello_int and keepalive_int are valid only when mode is SCE_NET_ADHOC_MATCHING_MODE_P2P or SCE_NET_ADHOC_MATCHING_MODE_PARENT.

Example

#define MATCHING_EVENT_TPL		16
#define MATCHING_EVENT_ HANDLER_STACK_SIZE	(1*1024)		/* 1KB */

char *helloopt;	// Optional data for Hello message

		MATCHING_EVENT_TPL,
		SCE_NET_PSPNET_ADHOC_MATCHING_LEAST_STACK_SIZE
			+ MATCHING_EVENT_STACK_SIZE,
		MATCHING_INPUT_TPL, 0,
		hellooptlen, helloopt);
if (ret < 0) {
	// Error handling
}
Notes

hello_optlen and hello_opt are valid only when mode is SCE_NET_ADHOC_MATCHING_MODE_P2P or SCE_NET_ADHOC_MATCHING_MODE_PARENT.

The stack size for the message receive thread is 3072 bytes.
This function will return SCE_ERROR_NET_ADHOC_INVALID_ADDR if no connection has been established to the ad hoc network. Be sure to call this function only after a connection is established.

	int id,

Calling Conditions

Can be called from a thread (must be called in interrupt-enabled state)
Multithread safe

Arguments

  id     Context ID  
  buflen     Size of memory area indicated by buf

Return Value

0 is returned on normal completion.
If an error occurs, one of the following error codes is returned.
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ARG
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ID
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_MODE
SCE_ERROR_NET_ADHOC_MATCHING_NOT_RUNNING

Description

This function gets the previously set optional data for a Hello message. The data is stored in the area pointed to by buf and the length is returned in buflen.
When buflen is less than the length of the optional data, only the first buflen bytes of optional data are copied to buf.
When buf is NULL, only the length of the optional data is returned in buflen.

Example

int ret;
int id;	// Context ID
int buflen;		// Buffer size
void *buf;		// Buffer

ret = sceNetAdhocMatchingGetHelloOpt(id, &buflen, NULL);
if (ret < 0) {
	// Error handling
	return;
}
else if (buflen == 0) {
	// No optional data has been set
	return;
}

buf = malloc(buflen);
if (buf == NULL) {
	// Could not allocate memory
	return;
}

ret = sceNetAdhocMatchingGetHelloOpt(id, &buflen, buf);
if (ret < 0) {
	// Error handling
}

free(buf);

Return Value

If an error occurs, one of the following error codes is returned.
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ARG
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ID
SCE_ERROR_NET_ADHOC_MATCHING_NOT_RUNNING

Description

This function gets a list of members who currently have a participation agreement.
When this function is called, buflen should specify the size of buf.
The member list is returned as a list of member list structures (struct SceNetAdhocMatchingMember) with the head of the list pointed to by buf.
When NULL is specified for buf, the buffer size needed to get the list is returned in buflen.

Example

int ret;
int id;	// Context ID
int buflen;		// Buffer size
struct SceNetAdhocMatchingMember *buf, *ptr;

ret = sceNetAdhocMatchingGetMembers(id, &buflen, NULL);
if (ret < 0) {
	// Error handling
	return;
}

buf = malloc(buflen);
if (buf == NULL) {
	// Could not allocate memory
	return;
}

ret = sceNetAdhocMatchingGetMembers(id, &buflen, buf);
if (ret < 0) {
	// Error handling
}
else {
	for (ptr = buf; ptr != NULL; ptr = ptr->next) {
		// Process the acquired member list
		...
	}
}

free(buf);
Notes

The member list includes the player who invoked the function.
Therefore, if no other players have established a participation agreement, the only member in the list will be the player who invoked the function.

  The optional data opt is valid when optlen >0  
  SCE_NET_ADHOC_MATCHING_EVENT_REQUEST     A join request was received  
  The optional data opt is valid when optlen >0  
  SCE_NET_ADHOC_MATCHING_EVENT_LEAVE     The participation agreement was canceled  
  by the target player  
  SCE_NET_ADHOC_MATCHING_EVENT_DENY     The join request was denied  
  SCE_NET_ADHOC_MATCHING_EVENT_CANCEL     The join request was canceled  
  SCE_NET_ADHOC_MATCHING_EVENT_ACCEPT     The join request was accepted  
  The optional data opt is valid when optlen >0  
  SCE_NET_ADHOC_MATCHING_EVENT_ESTABLISHED     A participation agreement was established  
  SCE_NET_ADHOC_MATCHING_EVENT_TIMEOUT     The participation agreement was canceled  
  because of a Keep Alive timeout  
  SCE_NET_ADHOC_MATCHING_EVENT_ERROR     A protocol error occurred

Example

/* Handler that simply displays accepted events */
void
adhoc_matching_handler(int id, int event, int optlen, void *opt)
{
	printf("context id: %d, event: %d, optlen = %d", id, event, optlen);
	if (optlen > 0) {
		printf(", opt: ");
		for (i = 0; i < optlen; i++) {
			printf("%02x", *((char *)opt + i));
		}
	}        
	printf("\n");
}
Notes

With threads that call the Matching library API and threads in event handler functions, you should be careful to avoid waiting for the same resource (such as using sceKernelWaitSema() to wait for the same semaphore). This is because threads calling the Matching library API and event-handler threads can get deadlocked.

The optional data indicated by opt is valid until the handler function returns.
To continue to use optional data after the handler returns, be sure to copy it to a local memory area.

The stack size available for use within event handler functions is equal to the stack size specified for event handler threads minus SCE_NET_PSPNET_ADHOC_MATCHING_LEAST_STACK_SIZE (3072 bytes).
