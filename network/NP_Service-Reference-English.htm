SceNpManagerSigninUpdateParam
 [Structure]
Sign-in ticket update parameters

Structure

#include <np_service.h>

typedef struct SceNpManagerSigninUpdateParam {
	SceSize size;
	SceLong64 ttl;
	int result;
	int reserved;
} SceNpManagerSigninUpdateParam;
Members

  size     Size of this structure  
  ttl     Valid period of updated ticket  
  result     Update result  
  reserved     Reserved area (specify 0)  

Description

This structure represents the sign-in ticket update parameters.
For size, specify the size of the structure itself. Sign-in ticket update results are returned in ttl and result. For reserved, always specify 0.

sceNpServiceInit
 [Function]
Initialize NP service library

Syntax

#include <np_service.h>

#define SCE_NP_MANAGER_INTERNAL_THREAD_LEAST_STACK_SIZE (14 * 1024)
int sceNpServiceInit(
	SceSize poolsize,
	SceSize np_manager_stacksize,
	int np_manager_priority
};
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  poolsize     Memory pool size  
  np_manager_stacksize     Stack size of internal thread   
  np_manager_priority     Priority of internal thread  

Return Value

When the function completes normally, 0 is returned.
If an error occurs, an error code is returned.

Description

This function initializes the NP service library. When it completes normally, it creates an internal thread for updating the ticket.
For np_manager_stacksize, specify the size of the stack consumed by the internal callback (1024) plus the value SCE_NP_MANAGER_INTERNAL_THREAD_LEAST_STACK_SIZE.

Example

int ret;

ret = sceNpServiceInit(
	4*1024,
	SCE_NP_MANAGER_INTERNAL_THREAD_LEAST_STACK_SIZE+1024,
	42);
if (ret < 0) {
	// Error Handling
}

sceNpManagerSigninUpdateInitStart
 [Function]
Start sign-in ticket update processing

Syntax

#include <np_service.h>

int sceNpManagerSigninUpdateInitStart(
	SceNpManagerSigninUpdateParam *param
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  param     Update parameters  

Return Value

When the function completes normally, 0 is returned.
If an error occurs, an error code is returned.

Description

This function updates the sign-in ticket.
Before calling this API, be sure to use the NP sign-in utility to perform sign-in processing. Otherwise, an SCE_NP_MANAGER_ERROR_NO_CREDENTIAL error will occur.

If the TTL period since the sign-in ticket was issued has elapsed (this period is currently set to 24 hours), the ticket will considered to be expired and will no longer be valid. APIs which use that ticket will no longer function. To avoid this, either sign in by calling the NP sign-in utility again or use this API to update the sign-in ticket.

For the size member of the param structure, specify the size of the param structure itself. Specify 0 for the other members. If this API call is successful, the internal status will change from SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_NONE to SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_RUNNING. To get the internal status, call the sceNpManagerSigninUpdateGetStatus() function. When ticket updating is completed, the internal status will become SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_FINISHED regardless of whether updating was successful or if it failed.
After the internal status becomes SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_FINISHED, the values of the sign-in ticket update processing results will be set in the ttl and result members of the param structure. If an error occurs, 0 is set for ttl and an error code is set for result. If update processing completes normally, the TTL of the updated ticket will be set for ttl and 0 will be set for result.
After the internal status becomes SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_FINISHED, call the sceNpManagerSigninUpdateShutdownStart() function to release internal resources.
Note: this differs from a true utility API in that termination processing will have been completed when control returns from the sceNpManagerSigninUpdateShutdownStart() call.

Example

int ret;
SceNpManagerSigninUpdateParam param;

memset(&param, 0x00, sizeof(param));

param.size = sizeof(param);
ret = sceNpManagerSigninUpdateInitStart(&param);
if (ret < 0){
	// Error handling
}

// Wait for completion of ticket update processing
while(sceNpManagerSigninUpdateGetStatus() !=
	SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_FINISHED)
{
	sceKernelDelayThread(100 * 1000);
}

// If the status is SCE_NP_MANAGER_SIGNIN_UPDATE_STATUS_FINISHED,
// sceNpManagerSigninUpdateShutdownStart() is always successful.
// Perform shutdown processing even if ticket update processing terminates with an error.
if (param.result < 0) {
	// Error handling
	sceNpManagerSigninUpdateShutdownStart();
}

sceNpManagerSigninUpdateShutdownStart();
