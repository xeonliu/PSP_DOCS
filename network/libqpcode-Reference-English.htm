sceQPrintableEncoder
 [Function]
Encode quoted-printable

Syntax

#include <libqpcode.h>
int sceQPrintableEncoder (
	SceChar8 *out,
	const SceUChar8 *in,
	SceSize src_len,
	SceSize *require,
	SceSize prepare
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-enabled or -disabled state)
Multithread safe

Arguments

  in     Pointer to input byte sequence  
  out     Pointer to output byte sequence  
  src_len     Input length  
  require     Pointer to memory for storing number of output bytes  
  prepare     Size of memory prepared for output byte sequence  

Return Value

  Value     Macro     Meaning  
  0     SCE_OK     Normal completion  
  0x80434022     SCE_QPCODE_OUT_OF_MEMORY     Number of bytes required for  
  output exceeded value  
  specified by prepare  
  0x804341FE     SCE_QPCODE_INVALID_VALUE     NULL was set for both out and  
  require  


Description

This function performs quoted-printable encoding of the input byte sequence specified by in and src_len, outputs the result to the memory area specified by out, and stores the number of output bytes in require. If the number of output bytes exceeds prepare, an error is returned. The application must provide the memory area for output. If out is set to NULL, the size of the memory area required for output will be stored in require and can be retrieved.

sceQPrintableLineDecoder
 [Function]
Line decode quoted-printable

Syntax

#include <libqpcode.h>
int sceQPrintableLineDecoder (
	SceChar8*out,
	const SceUChar8 *in,
	SceSize src_len,
	SceSize *require,
	SceSize prepare
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-enabled or -disabled state)
Multithread safe

Arguments

  out     Pointer to output byte sequence  
  in     Pointer to input byte sequence  
  src_len     Input length  
  require     Pointer to memory for storing number of output bytes  
  prepare     Size of memory prepared for output byte sequence  

Return Value

  Value     Macro     Meaning  
  0     SCE_OK     Normal completion  
  0x80434022     SCE_QPCODE_OUT_OF_MEMORY     Number of bytes required for  
  output exceeded value specified  
  by prepare  
  0x804341FE     SCE_QPCODE_INVALID_VALUE     NULL was set for both out and  
  require  
  0x80434108     SCE_QPCODE_INVALID_FORMAT     A byte sequence having an  
  invalid format was specified for in  


Description

This function decodes the input byte sequence (one line of data that was encoded using quoted-printable encoding) specified by in and src_len, outputs the result to the memory area specified by out, and stores the number of output bytes in require. If the number of output bytes exceeds prepare, an error is returned. The length of the input byte sequence including the terminating RFC822 newline code (consecutive CR and LF) must not exceed 78. If a larger value is set for ilen, it is ignored. The application must provide the memory area for output. If out is set to NULL, the size of the memory area required for output will be stored in require and can be retrieved.
