SceNpCommunicationId
 [構造体]
NPコミュニケーションID構造体

構造

#include <np/np_common.h>

typedef struct SceNpCommunicationId {
	char data[9];
	char term;
	SceUChar8 num;
	char dummy;
} SceNpCommunicationId;
メンバ

  data     NPコミュニケーションID文字列(アルファベット4文字＋数字5文字)  
  term     dataを終端するための終端文字列用領域  
  num     サブID(通常は0)  
  dummy     未使用  

解説

NPコミュニケーションIDを表す構造体です。

SceNpOnlineId
 [構造体]
オンラインID構造体

構造

#include <np/np_common.h>

#define SCE_NET_NP_ONLINEID_MIN_LENGTH		3
#define SCE_NET_NP_ONLINEID_MAX_LENGTH		16

typedef struct SceNpOnlineId {
	char data[SCE_NET_NP_ONLINEID_MAX_LENGTH];
	char term;
	char dummy[3];
} SceNpOnlineId;
メンバ

  data     オンラインID文字列  
  term     dataを終端するための終端文字列用領域  
  dummy     未使用  

解説

ユーザのオンラインIDを表す構造体です。

SceNpId
 [構造体]
NP ID構造体

構造

#include <np/np_common.h>

typedef struct SceNpId {
	SceNpOnlineId handle;
	SceUChar8 opt[8];
	SceUChar8 reserved[8];
} SceNpId;
メンバ

  handle     オンラインID  
  opt     オプションデータ  
  reserved     将来の拡張用領域  

解説

NPライブラリにおいて、ユーザを識別するための構造体です。
オンラインIDを内包しています。

SceNpUserInfo
 [構造体]
ユーザ情報構造体

構造

#include <np/np_common.h>

typedef struct SceNpUserInfo {
	SceNpId userId;
	SceNpOnlineName name;		/* Online Name */
	SceNpAvatarUrl icon;		/* Avatar URL */
} SceNpUserInfo;
メンバ

  userId     NP ID  
  name     オンラインネーム  
  icon     アバターURL  

解説

ユーザを表す情報として、識別子であるNP IDと、付加情報であるオンラインネームおよびアバターURLをまとめた構造体です。

SceNpUserInfo2
 [構造体]
ユーザ情報構造体(ポインタ版)

構造

#include <np/np_common.h>

typedef struct SceNpUserInfo2 {
	SceNpId npId;
	SceNpOnlineName *onlineName;   	/* (pointer) Online Name */
	SceNpAvatarUrl *avatarUrl;		/* (pointer) Avatar URL */
} SceNpUserInfo2;
メンバ

  npId     NP ID  
  onlineName     オンラインネーム構造体へのポインタ  
  avatarUrl     アバターURL構造体へのポインタ  

解説

ユーザ情報をまとめた構造体です。
SceNpUserInfoと異なり、オンラインネームとアバターURLがポインタになっています。

SceNpMyLanguages
 [構造体]
よく使う言語構造体

構造

#include <np/np_common.h>

typedef struct SceNpMyLanguages{
	SceInt32 language1;	/* It will always be set */
	SceInt32 language2;	/* Negative when not set */
	SceInt32 language3;	/* Negative when not set */
	SceUChar8 padding[4];
} SceNpMyLanguages;
メンバ

  language1     言語設定 1 (必ず設定される)  
  language2     言語設定 2 (設定されていない場合、負の値となる)  
  language3     言語設定 3 (設定されていない場合、負の値となる)  
  padding     未使用  

解説

ユーザが設定した"よく使う言語"をまとめた構造体です。
language1、language2、language3には、それぞれ以下の値が入ります。
値が設定されていない場合、負の値が設定されます。

  SCE_NP_LANG_GERMAN     4     ドイツ語  
  SCE_NP_LANG_ITALIAN     5     イタリア語  
  SCE_NP_LANG_DUTCH     6     オランダ語

  SCE_NP_LANG_RUSSIAN     8     ロシア語  
  SCE_NP_LANG_KOREAN     9     韓国語  
  SCE_NP_LANG_CHINESE_T     10     中国語（繁体字）  
  SCE_NP_LANG_CHINESE_S     11     中国語（簡体字）  
  SCE_NP_LANG_FINNISH     12     フィンランド語

SceNpCountryCode
 [構造体]
国コード構造体

構造

#include <np/np_common.h>

typedef struct SceNpCountryCode {
	char data[2];
	char term;
	char padding[1];
} SceNpCountryCode;
メンバ

  data     国コード(ISO 3166-1)  
  term     dataを終端するための終端文字列用領域  
  padding     未使用  

解説

NPチケットパラメータのリージョンコードに格納されている国コードを表す構造体です。
dataはISO 3166-1で定義されている国を表す2バイトのアスキーコードです。
