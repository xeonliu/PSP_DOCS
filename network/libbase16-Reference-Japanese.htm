libbase16

sceBase16Encoder
 [関数]
BASE16エンコード

形式

#include <libbase16.h>
int sceBase16Encoder (
	unsigned char *pDat,
	const unsigned char *pSrc,
	unsigned int uiSize
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能(割り込み許可/禁止状態に依存しない)
マルチスレッドセーフである

引数

  pDat     出力バッファへのポインタ  
  pSrc     入力バイト列へのポインタ  
  uiSize     入力バイト数  

返り値

出力バイト数

解説

pSrcとuiSizeで指定された入力バイト列をBASE16エンコードして、pDatで指定されたメモリ領域へ出力し、出力バイト数を返します。出力用のメモリ領域は、アプリケーションが用意する必要があり、そのサイズは(uiSize * 2 + 1)バイトとなります。

備考

出力したデータは、安全のためNULLターミネートしておりますが、NULL分の1バイトは返り値の出力バイト数には含まれません。

sceBase16Decoder
 [関数]
BASE16デコード

形式

#include <libbase16.h>
int sceBase16Decoder (
	unsigned char *pDat,
	const unsigned char *pSrc,
	unsigned int uiSize
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能(割り込み許可/禁止状態に依存しない)
マルチスレッドセーフである

引数

  pDat     出力バッファへのポインタ  
  pSrc     入力バイト列へのポインタ  
  uiSize     入力バイト数  

返り値

  値     結果  
  正の値     出力バイト数  
  -1     エラー  


解説

pSrcとuiSizeとで指定された入力バイト列(BASE16エンコードされたデータ)をデコードして、pDatで指定されたメモリ領域へ出力します。返り値として正常終了した場合は、出力バイト数がエラーの時は、-1が返ります。出力用のメモリ領域は、アプリケーションが用意する必要があり、そのサイズは(uiSize / 2 + 1)バイトとなります。

備考

出力するデータは、安全のためNULLターミネートしておりますが、NULL分の1バイトは返り値の出力バイト数には含まれません。
