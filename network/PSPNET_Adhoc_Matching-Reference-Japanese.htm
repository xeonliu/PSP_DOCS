sceNetAdhocMatchingInit
 [関数]
アドホックマッチングライブラリ初期化

形式

#include <pspnet_adhoc_matching.h>

int sceNetAdhocMatchingInit(
	SceSize poolsize
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフでない

引数

  poolsize     メモリプールサイズ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のエラーコードを返します。
SCE_ERROR_NET_ADHOC_MATCHING_ALREADY_INITIALIZED

解説

アドホックマッチングライブラリを初期化します。

ライブラリ内部で使用するメモリはすべてこの領域から使用します。

使用例

ret = sceNetAdhocMatchingInit(MATCHING_POOLSIZE);
if (ret < 0) {
	// エラー処理
}

sceNetAdhocMatchingCreate
 [関数]
マッチングコンテキストの作成

形式

#include <pspnet_adhoc_matching.h>

int sceNetAdhocMatchingCreate(
	int mode,
	int maxnum,
	SceUShort16 port,
	int rxbuflen,
	unsigned int hello_int,
	unsigned int keepalive_int,
	int init_count,
	unsigned int rexmt_int,
	void (*handler)(int, int, struct SceNetEtherAddr *, int, void *)
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  mode     動作モード  
  maxnum     マッチンググループ最大メンバ数  
  port     ポート番号  
  rxbuflen     受信バッファサイズ  
  hello_int     Helloメッセージインターバル(マイクロ秒)  
  keepalive_int     KeepAliveRequestメッセージインターバル(マイクロ秒)  
  init_count     再送カウンタおよびKeepAliveカウンタ初期値  
  rexmt_int     メッセージ再送間隔(マイクロ秒)  
  handler     イベントハンドラ関数  

返り値

正常終了の場合、0以上のコンテキストIDを返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_MAXNUM
SCE_ERROR_NET_ADHOC_MATCHING_RXBUF_TOO_SHORT
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ARG
SCE_ERROR_NET_ADHOC_MATCHING_NO_SPACE
SCE_ERROR_NET_ADHOC_MATCHING_ID_NOT_AVAIL
SCE_ERROR_NET_ADHOC_MATCHING_PORT_IN_USE

解説

マッチングコンテキストを作成します。
正常終了した場合は、返り値としてコンテキストIDが渡されます。

使用例

#define MATCHING_MAXNUM	16
#define MATCHING_PORT	1
#define MATCHING_RXBUFLEN	2048
#define HELLO_INTERVAL	5*1000*1000	// 5sec
#define KEEPALIVE_INTERVAL	10*1000*1000	// 10sec
#define INIT_COUNT		3
#define REXMT_INTERVAL	500*1000	// 500msec

void handler(int id, int event, struct SceNetEtherAddr *peer, int optlen, void *opt);

int ret;
int id;	// コンテキストID

ret = sceNetAdhocMatchingCreate(
		SCE_NET_ADHOC_MATCHING_MODE_PARENT,
		MATCHING_MAXNUM, MATCHING_PORT,
		MATCHING_RXBUFLEN, HELLO_INTERVAL,
		KEEPALIVE_INTERVAL, INIT_COUNT, REXMT_INTERVAL,
		handler);
if (ret < 0) {
	// エラー処理
}
id = ret;
備考

maxnumはmodeがSCE_NET_ADHOC_MATCHING_MODE_PARENTのときのみ有効です。
hello_int, keepalive_intはmodeがSCE_NET_ADHOC_MATCHING_MODE_P2P,SCE_NET_ADHOC_MATCHING_MODE_PARENTのときのみ有効です。

sceNetAdhocMatchingStart
 [関数]
マッチングの開始

形式

#include <pspnet_adhoc_matching.h>

int sceNetAdhocMatchingStart(
	int id,
	int event_th_prio,
	int event_th_stack,
	int input_th_prio,
	int input_th_stack,
	int hello_optlen,
	const void *hello_opt
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  id     コンテキストID  
  event_th_prio     イベントハンドラスレッド優先度  
  event_th_stack     イベントハンドラスレッドスタックサイズ  
  input_th_prio     メッセージ受信スレッド優先度  
  input_th_stack     未使用（0を指定）  
  hello_optlen     HELLOメッセージのオプションデータサイズ  
  hello_opt     HELLOメッセージのオプションデータ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_STACKSIZE_TOO_SHORT
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ID
SCE_ERROR_NET_ADHOC_MATCHING_IS_RUNNING
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_OPTLEN
SCE_ERROR_NET_ADHOC_MATCHING_NO_SPACE

解説

指定したコンテキストIDのマッチングを開始します。

使用例

#define MATCHING_EVENT_TPL		16
#define MATCHING_EVENT_HANDLER_STACK_SIZE	(1*1024)		/* 1KB */
#define MATCHING_INPUT_TPL		16

int id;	// コンテキストID
int hellooptlen;	// Helloオプションのサイズ
char *helloopt;	// Helloオプション

ret = sceNetAdhocMatchingStart(id,
		MATCHING_EVENT_TPL,
		SCE_NET_PSPNET_ADHOC_MATCHING_LEAST_STACK_SIZE
			+ MATCHING_EVENT_STACK_SIZE,
		MATCHING_INPUT_TPL, 0,
		hellooptlen, helloopt);
if (ret < 0) {
	// エラー処理
}
備考

hello_optlen, hello_optは、modeがSCE_NET_ADHOC_MATCHING_MODE_P2P,SCE_NET_ADHOC_MATCHING_MODE_PARENTのときのみ有効です。

イベントハンドラスレッドのスタックサイズには、SCE_NET_PSPNET_ADHOC_MATCHING_LEAST_STACK_SIZE（=3072バイト）とsceNetAdhocMatchingCreate()で指定したイベントハンドラ関数で使用するスタックサイズの合計を指定してください。
メッセージ受信スレッドのスタックサイズは、3072バイトとなります。
アドホックネットワークに接続していない状態で呼び出すとSCE_ERROR_NET_ADHOC_INVALID_ADDRが返ります。アドホックネットワークに接続した後に呼び出してください。

int sceNetAdhocMatchingSendData(
	int id,
	const struct SceNetEtherAddr *target,
	int datalen,

呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  id     コンテキストID  
  target     送信相手のアドレス  
  datalen     データサイズ  
  data     データ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ARG
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ID
SCE_ERROR_NET_ADHOC_MATCHING_NOT_RUNNING
SCE_ERROR_NET_ADHOC_MATCHING_UNKNOWN_TARGET
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_DATALEN
SCE_ERROR_NET_ADHOC_MATCHING_NOT_ESTABLISHED
SCE_ERROR_NET_ADHOC_MATCHING_NO_SPACE
SCE_ERROR_NET_ADHOC_MATCHING_DATA_BUSY

解説

targetに対してデータを送信します。
targetはESTABLISHED状態である必要があります。
送信結果はハンドラ関数経由にて通知されます。
送信結果の通知、またはsceNetAdhocMatchingAbortSendData()を呼び出すまで、新たなデータを送信することはできません。
データの再送間隔および再送回数はsceNetAdhocMatchingCreate()にて指定した値になります。

使用例

int id;	// コンテキストID
struct SceNetEtherAddr addr;	// 送信相手のアドレス
int datalen;		// データサイズ
char *data;		// データ

ret = sceNetAdhocMatchingSendData(id, &addr, datalen, data);
if (ret < 0) {
	// エラー処理
}

  buf     オプションを格納するメモリ領域へのポインタ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ARG
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ID
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_MODE
SCE_ERROR_NET_ADHOC_MATCHING_NOT_RUNNING

解説

設定されているHelloメッセージのオプションデータを取得します。
bufで示された領域に設定されているオプションデータをコピーし、buflenにその長さを返します。
オプションデータの長さがbuflenより大きい場合は、先頭からbuflen分だけコピーされます。
bufがNULLの場合、単にbuflenにオプションデータの長さを返します。

使用例

int ret;
int id;	// コンテキストID
int buflen;		// バッファサイズ
void *buf;		// バッファ

ret = sceNetAdhocMatchingGetHelloOpt(id, &buflen, NULL);
if (ret < 0) {
	// エラー処理
	return;
}
else if (buflen == 0) {
	// オプションデータは設定されていない
	return;
}

buf = malloc(buflen);
if (buf == NULL) {
	// メモリが確保出来なかった
	return;
}

ret = sceNetAdhocMatchingGetHelloOpt(id, &buflen, buf);
if (ret < 0) {
	// エラー処理
}

free(buf);

sceNetAdhocMatchingGetMembers
 [関数]
メンバの取得

形式

#include <pspnet_adhoc_matching.h>

int sceNetAdhocMatchingGetMembers(
	int id,
	int *buflen,
	void *buf
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  id     コンテキストID  
  buflen     バッファサイズ（IN）、リストサイズ（OUT）  
  buf     メンバリストを格納するバッファ領域へのポインタ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、以下のいずれかのエラーコードを返します。
SCE_ERROR_NET_ADHOC_MATCHING_NOT_INITIALIZED
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ARG
SCE_ERROR_NET_ADHOC_MATCHING_INVALID_ID
SCE_ERROR_NET_ADHOC_MATCHING_NOT_RUNNING

解説

現在対戦合意しているメンバのリストを取得します。
buflenにbufで指定されるバッファ領域のサイズを指定して呼び出します。
メンバリストはbufで指定されるポインタを先頭としたメンバリスト構造体(struct SceNetAdhocMatchingMember)のリストとして返されます。
bufにNULLを指定した場合、リストを取得するのに必要なバッファサイズがbuflenに返されます。

使用例

int ret;
int id;	// コンテキストID
int buflen;		// バッファサイズ
struct SceNetAdhocMatchingMember *buf, *ptr;

ret = sceNetAdhocMatchingGetMembers(id, &buflen, NULL);
if (ret < 0) {
	// エラー処理
	return;
}

buf = malloc(buflen);
if (buf == NULL) {
	// メモリが確保できなかった
	return;
}

ret = sceNetAdhocMatchingGetMembers(id, &buflen, buf);
if (ret < 0) {
	// エラー処理
}
else {
	for (ptr = buf; ptr != NULL; ptr = ptr->next) {
		// 取得したメンバリストに対する処理
		...
	}
}

free(buf);
備考

メンバリストには自端末を含みます。
したがって対戦合意したメンバが存在しない場合は、自端末のみがメンバとして返されます。

adhocmatchinghandler
 [関数]
イベントハンドラのプロトタイプ

形式

#include <pspnet_adhoc_matching.h>

void adhocmatchinghandler(
	int id,
	int event,
	struct SceNetEtherAddr *peer,
	int optlen,
	void *opt
);
引数

  id     コンテキストID  
  event     イベント  
  peer     イベント対象のアドレス  
  optlen     オプションデータサイズ  
  opt     オプションデータ  

解説

イベントハンドラはマッチングライブラリからイベントを通知するために、イベントハンドラスレッドから呼び出されます。gpの値は、設定されたイベントハンドラが存在するモジュールのgpの値にセットされます。

イベントの種類と意味は以下の通りです。
  イベント     意味  
  SCE_NET_ADHOC_MATCHING_EVENT_HELLO     Helloメッセージを受信した  
  optlen > 0でオプションデータoptが有効  
  SCE_NET_ADHOC_MATCHING_EVENT_REQUEST     対戦要求を受けた  
  optlen > 0でオプションデータoptが有効  
  SCE_NET_ADHOC_MATCHING_EVENT_LEAVE     対戦合意が相手によって解消された  
  SCE_NET_ADHOC_MATCHING_EVENT_DENY     対戦要求が拒否された  
  SCE_NET_ADHOC_MATCHING_EVENT_CANCEL     対戦要求がキャンセルされた  
  SCE_NET_ADHOC_MATCHING_EVENT_ACCEPT     対戦要求が受け入れられた  
  optlen > 0でオプションデータoptが有効  
  SCE_NET_ADHOC_MATCHING_EVENT_ESTABLISHED     対戦合意した  
  SCE_NET_ADHOC_MATCHING_EVENT_TIMEOUT     対戦合意がキープアライブタイムアウトに  
  より解消された  
  SCE_NET_ADHOC_MATCHING_EVENT_ERROR     プロトコルエラー  


またモードによって発生しうるイベント種類は以下の通りです。
  イベント     P2P     PARENT     CHILD  
  SCE_NET_ADHOC_MATCHING_EVENT_HELLO     ○     −     ○  
  SCE_NET_ADHOC_MATCHING_EVENT_REQUEST     ○     ○     −  
  SCE_NET_ADHOC_MATCHING_EVENT_LEAVE     ○     ○     ○  
  SCE_NET_ADHOC_MATCHING_EVENT_DENY     ○     −     ○  
  SCE_NET_ADHOC_MATCHING_EVENT_CANCEL     ○     ○     −  
  SCE_NET_ADHOC_MATCHING_EVENT_ACCEPT     ○     −     ○  
  SCE_NET_ADHOC_MATCHING_EVENT_ESTABLISHED     ○     ○     ○  
  SCE_NET_ADHOC_MATCHING_EVENT_TIMEOUT     ○     ○     ○  
  SCE_NET_ADHOC_MATCHING_EVENT_ERROR     ○     ○     ○  

○：発生する
−：発生しない

使用例

/* 受け取ったイベントを表示するだけのハンドラ */
void
adhoc_matching_handler(int id, int event, int optlen, void *opt)
{
	printf("context id: %d, event: %d, optlen = %d", id, event, optlen);
	if (optlen > 0) {
		printf(", opt: ");
		for (i = 0; i < optlen; i++) {
			printf("%02x", *((char *)opt + i));
		}
	}
	printf("\n");
}
備考

イベントハンドラ関数内とマッチングライブラリAPIを呼び出すスレッドで、同一資源に対する待ち合わせ（同一セマフォに対するsceKernelWaitSema()など）を行なわないようにしてください。マッチングライブラリAPIを呼び出したスレッドとイベントハンドラスレッドがデッドロック状態に陥る可能性があります。

optで示されるオプションデータの有効範囲は、ハンドラ関数のリターン時までです。
後にオプションデータを利用する場合は、必ずローカルのメモリ領域にコピーしてください。

イベントハンドラ関数内で使用できるスタックサイズは、イベントハンドラスレッドに指定したスタックサイズからSCE_NET_PSPNET_ADHOC_MATCHING_LEAST_STACK_SIZE（=3072バイト）を引いたサイズとなります。
