libbase32 Overview

1 Library Overview

1. 1 Overview

libbase32 is a library for performing BASE32-format or BASE32HEX encoding/decoding, as defined in RFC4648.

1. 2 Related Files

The following files are required to use libbase32.

2 Using the Library

2. 1 Basic Usage Procedure

No specific initialization is required to use libbase32.

Encoding to BASE32 format
The following procedure can be used to encode a byte sequence into BASE32 format. Note that since libbase32 does not allocate memory internally, the application must allocate memory for the output in advance. The required memory size is ((number of input bytes + 4) / 5 * 8 + 1) bytes.

u_char	plain[SOURCE_LEN];
u_char	*base32;
SceSize malloc_size;

malloc_size = 	(SOURCE_LEN + 4) / 5 * 8 + 1;	// Calculate output size
base32 = (u_char*)malloc(malloc_size);		// Allocate memory for output
sceBase32Encoder(base32, plain, SOURCE_LEN);	// Encode
BASE32 encoding is performed by calling the sceBase32Encoder() function as shown above. To encode into BASE32HEX format, use the sceBase32HexEncoder() function.

Decoding from BASE32 format
The following procedure can be used to decode a byte sequence that was previously encoded into BASE32 format. Note that since libbase32 does not allocate memory internally, the application must allocate memory for the output in advance. The required memory size is ((number of input bytes) / 8 * 5 + 1) bytes.

u_char	base32[SOURCE_LEN];
u_char	*plain;
SceSize malloc_size;

malloc_size = SOURCE_LEN / 8 * 5 + 1;		// Calculate output size
plain = (u_char*) malloc(malloc_size);		// Allocate memory for output
sceBase32Decoder (plain, base32, SOURCE_LEN);	// Decode

BASE32 decoding is performed by calling the sceBase32Decoder() function as shown above. To decode BASE32HEX format, use the sceBase32HexDecoder() function.
