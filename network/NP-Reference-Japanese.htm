sceNpInit
 [関数]
ライブラリを初期化

形式

#include <np/np.h>
int sceNpInit(
	void
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能 (割り込み許可状態で呼ぶこと)
マルチスレッドセーフでない

引数

なし

返り値

正常終了の場合、0を返します。
エラーがあった場合、エラーコードを返します。

解説

NPライブラリを初期化します。

使用例

ret = sceNpInit();
if(ret < 0){
	/* エラー処理 */
}

sceNpGetContentRatingFlag
 [自ユーザの情報取得]
自ユーザの年齢制限に関するパレンタルコントロール情報を取得

形式

#include <np/np.h>
int sceNpGetContentRatingFlag(
	int *isRestricted,
	int *age
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能 (割り込み許可状態で呼ぶこと)
マルチスレッドセーフである

引数

  isRestricted     制限されているかどうかを示すフラグ(制限されていたら真値)の格納先  
  age     アカウントに設定されている自ユーザの年齢の格納先  

返り値

正常終了の場合、0を返します。
エラーがあった場合、エラーコードを返します。

解説

NPへのサインイン時に取得した、自ユーザのパレンタルコントロール情報を取得します。
isRestrictedに取得される値が真値の場合、このアカウントにおいて、年齢によるコンテンツ利用制限が有効であることを示します。

使用例

int isRestricted, age;
int ret = 0;

ret = sceNpGetContentRatingFlag(&isRestricted, &age);
if(ret < 0){
	/* エラー処理 */
}
#define CONTENT_RATING_AGE 18 //年齢制限
if (isRestricted && age < CONTENT_RATING_AGE) {
	//コンテンツ視聴は制限されています
}
