  SCE_MPEG_ERROR_MP4_INVALID_VALUE     Invalid input parameter  
  SCE_MPEG_ERROR_MP4_ILLEGAL_FORMAT     Invalid file format

  SCE_MPEG_ERROR_MP4_FILE_ACCESS     Problem occurred during file

  insufficient  


Description

This function allocates and initializes memory for the SceMp4 structure used by the MP4 library. In the pSceMp4 argument, pass a pointer to the SceMp4 structure provided by the application. In the pReadCacheBuf and iReadCacheSize arguments, specify the read cache buffer for file access and MP4 demultiplexing and its size, respectively. Set the size to match the file read destination device and the MP4 file that is used. The size must be set to a value of at least 1024 bytes. After the sceMp4Init() function has completed, call the sceMp4Create() function to allocate and initialize the SceMp4 structure.

sceMp4InitAu
 [Function]
Initialize SceMpegAu structure for managing extractions to elementary streams

Syntax

#include<libmp4.h>
SceInt32 *sceMp4InitAu (
	SceMp4 *pSceMp4,
	SceMpegHandle hEsBuf,
	SceMpegAu *pAu
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread
Multithread safe

Arguments

  pSceMp4     MP4 library demultiplexer management structure  
  hEsBuf     Handle returned by sceMpegAvcResourceGetAvcEsBuf() function or memory address where an elementary stream size worth of memory was allocated  
  pAu     Structure for managing extractions to elementary streams.  
  Specify the pointer address where memory of the relevant size was allocated.  

Return Value

  Macro     Description  
  SCE_OK     Normal completion  
  SCE_MPEG_ERROR_MP4_NOT_INITIALIZED     Not initialized  
  SCE_MPEG_ERROR_MP4_INVALID_VALUE     Invalid input parameter  
  SCE_MPEG_ERROR_MP4_INSUFFICIENT_STACKSIZE     Remaining stack size is   
  insufficient  


Description

This function initializes the SceMpegAu structure that is required by functions such as sceMp4GetAvcAu() and sceMp4GetAacAu(), which extract consecutive elementary streams in one access unit increments.
For hEsBuf, specify a pointer to the beginning of the buffer for placing the one access unit elementary streams that were extracted by sceMp4GetAvcAu() and sceMp4GetAacAu(). For AVC, specify the handle that was obtained from the sceMpegAvcResourceGetAvcEsBuf() function in hEsBuf. For AAC, allocate 8192 bytes of memory at a 64-byte boundary and set its starting address in hEsBuf.
Note that the operation of this function is the same as that of the MPEG library function sceMpegInitAu(), and the SceMpegAu structure is initialized in the same way by both functions.
