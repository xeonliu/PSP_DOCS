scePsmfPlayerSetTempBuf
 [Function]
Set temporary read buffer

Syntax

#include<libpsmfplayer.h>
SceInt32 scePsmfPlayerSetTempBuf (
	ScePsmfPlayer *pCore,
	SceUChar8* tempBuf,
	SceSize tempBufSize
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pCore     ScePsmfPlayer structure  
  tempBuf     Temporary buffer address (64-byte alignment)  
  tempBufSize     Temporary buffer size  

Return Value

A negative value indicates that an error occurred.
  Macro     Description  
  SCE_OK     Normal completion  
  SCE_MPEG_ERROR_PSMFPLAYER_NOT_INITIALIZED     PsmfPlayer has not been properly initialized  
  SCE_MPEG_ERROR_PSMFPLAYER_BUSY     Other processing is being executed  


Description

This function sets a temporary buffer for reading a file. Use it when using volatile memory as the player's memory, or in similar situations.
tempBuf specifies the address of memory that was allocated with a size of SCE_PSMFPLAYER_TEMP_BUF_SIZE. The allocated memory must have 64-byte alignment.
For tempBufSize, specify SCE_PSMFPLAYER_TEMP_BUF_SIZE.
To use this function, the player status must be SCE_PSMFPLAYER_STATUS_INIT.

When the player's memory specified by ScePsmfPlayerParam is volatile memory, use this function to set temporary memory so that data read from the UMD™ will not be read directly into volatile memory.
scePsmfPlayerChangeReadBufSize
 [Function]
Change read buffer size

Syntax

#include<libpsmfplayer.h>
SceInt32 scePsmfPlayerChangeReadBufSize(
	SceUInt32 bufSize
)
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  bufSize     New buffer size  

Return Value

A negative value indicates that an error occurred.
  Macro     Description  
  SCE_OK     Normal completion  
  SCE_MPEG_ERROR_PSMFPLAYER_NOT_INITIALIZED     PsmfPlayer has not been properly initialized  


Description

This function changes the size of the read buffer. Call this function before calling scePsmfPlayerCreate(). If the buffer size is set after calling scePsmfPlayerCreate(), the new size will not be reflected properly.
The time it takes for a movie to begin can be reduced by specifying an appropriate read buffer size. The default read buffer size is 611 packs (1251328 bytes). Note that if a larger size is specified, that amount will need to be specified in the memory size that is set in ScePsmfPlayerParam. If less than 611 packs is specified, memory can be reduced by the corresponding amount.
For information about an appropriate read buffer size to use, refer to the "libmpeg Overview" document.
