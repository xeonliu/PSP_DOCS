SceMpegAu
 [構造体]
アクセスユニット管理構造体

構造

#include<libmpeg.h>
typedef struct {
	SceMpegTimeStamp pts;
	SceMpegTimeStamp dts;
	SceUChar8 *pEsBuf;
	SceInt32 iEsSize;
	SceUChar8 reserved[40];
} SceMpegAu;
メンバ

  pts     PTS(Presentation Time Stamp)を示します  
  dts     DTS(Decoding Time Stamp)を示します  
  pEsBuf     1DecodeSize(1アクセスユニット)サイズ分のバッファの  
  先頭アドレスを格納します  
  iEsSize     上記pEsBufに切り出された1アクセスユニットのサイズ  
  reserved     予約領域です  

解説

SceMpegStream構造体で切り出された各ストリームのPES packetを1アクセスユニットサイズのバッファにまとめます。sceMpegAvcDecode()及びsceMpegAtracDecode()では、必ず1アクセスユニットごとに切り出されたSceMpegAu構造体を指定してください。
pts,dtsが必要な場合にはsceMpegGetAvcAu()及びsceMpegGetAtracAu()関数で1アクセスユニットごとのエレメンタリストリームに切り出された後にSceMpegAu構造体のメンバを参照します。
pts, dtsが無効な場合には0xffffffff, 0xffffffffの値がそれぞれ返ります。

sceMpegQueryStreamOffset
 [関数]
ストリームの先頭までのオフセットを返す

形式

#include<libmpeg.h>
SceInt32 sceMpegQueryStreamOffset(
	SceMpeg *mp,
	Const void *addr,
	SceUInt32 *offset
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能
マルチスレッドセーフではない（割り込み禁止、許可状態に依存しない）

引数

  mp     MPEGライブラリ管理構造体  
  addr     PSMFファイルの先頭アドレス  
  offset  
    		   PSMFファイルの先頭からストリームの先頭までの  
  オフセットを格納するためのポインタ  

返り値

  マクロ     説明  
  SCE_MPEG_OK     正常終了  
  SCE_MPEG_ERROR_UNMATCHED_VERSION     対応しないバージョン  
  SCE_MPEG_ERROR_INVALID_VALUE     入力したデータが正しくない  
  SCE_MPEG_ERROR_INSUFFICIENT_STACKSIZE     残りスタックサイズが不足している  
  SCE_MPEG_ERROR_NOT_INITIALIZE     sceMpegInit(), sceMpegCreate()で  
  初期化されていない  


解説

ファイルの先頭からストリームの先頭までのオフセットを返します。この関数を呼び出す際には、あらかじめ2048byte以上PSMFを読んでおく必要があります。sceMpegCreate()関数の後に呼び出してください。
PSMFファイルは、先頭のPSMFヘッダ部とその後に続くPSMFストリーム部から構成されておりますが、PSMFファイルのうちPSMFヘッダ部を読み飛ばしsceMpegRingbufferPut()関数に渡すPSMFストリーム部の開始位置を求めるために使用します。
また、PSMFヘッダ部にはトリックプレイ等を行う際のメタデータも含まれております。これらメタデータを使用してトリックプレイ等を行うためにPSMFヘッダ部のサイズを知るときにも使用することができます。
