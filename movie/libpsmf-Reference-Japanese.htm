呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフでない

引数

  psmf     PSMFライブラリ管理構造体  
  type     ストリームタイプ  
  streamTypeNumber     ストリームタイプと組み合わせて  
  ストリームを特定するためのid  

返り値

  マクロ     説明  
  SCE_OK     正常終了  
  SCE_MPEG_ERROR_PSMF_INVALID_PSMF     無効なPSMF  
  SCE_MPEG_ERROR_PSMF_INVALID_ID     存在しないストリームタイプId  
  SCE_MPEG_ERROR_PSMF_UNMATCHED_VERSION     サポートしていないバージョンの  
  PSMF  
  SCE_MPEG_ERROR_PSMF_NOT_INITIALIZED     ScePsmf構造体が初期化されていない  


解説

ストリームタイプとstreamTypeNumberに対応したストリームを対象として指定します。ストリームを指定することで、そのストリームに関数情報が取得できるようになります。streamTypeNumberは0以上scePsmfGetNumberOfSpecificStreams() - 1以下のIDで、ストリームタイプ別にPSMFヘッダに書かれた順でidを0から割り振ったものです。

チャンネルはlibmpegのsceMpegRegistStream()のものと同様で以下のようになります。

 要素 
 ストリーム 		  ストリームタイプ 
 (iStreamType) 		  ストリームID   プライベートストリームID   streamType 
 Number 
 MPEG4 AVC   SCE_MPEG_STREAM_M4AVC   '1110 xxxxb'   なし   0-15 
 ATRAC3plus™   SCE_MPEG_STREAM_ATRAC   '1011 1101b'   '0000 xxxxb'   0-15 
 PCM   SCE_MPEG_STREAM_PCM   '1011 1101b'   '0001 xxxxb'   0-15 
 AUDIO   SCE_MPEG_STREAM_AUDIO   ATRAC3plus™もしくはPCM 


SCE_MPEG_STREAM_AUDIOを指定した場合はATRAC3plus™とPCMどちらかだった場合カウントしていきます。したがってSCE_MPEG_STREAM_AUDIOの総数はSCE_MPEG_STREAM_ATRACの総数とSCE_MPEG_STREAM_PCM総数の和と等しくなります。
