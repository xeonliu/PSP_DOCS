This function finds the offset from the beginning of a PSMF file to the start of the stream. You must have already read at least the first 2048 bytes of the PSMF file before calling this function. Be sure to call sceMpegCreate() before calling this function.

Arguments

  iStreamType     SceMpegStrType stream type to be registered

Return Value

Returns a normal handler when the return value is other than SCE_NULL. SCE_NULL indicates an error.

Description

This function registers a stream type and stream channel in the MPEG library, corresponding to iStreamType and ch, which will be used for demultiplexing. When processing completes normally, an SceMpegStream structure for the registered handler is returned. The SceMpegStream handler is used for extracting individual streams during demultiplexing. Note that if you do not register the iStreamType and ch using this function in advance, an elementary stream will not be output even if the sceMpegRingbufferPut() function is called to read stream data into the MPEG library. The following shows valid settings for the stream type and channel.
