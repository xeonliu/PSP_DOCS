Description

Specifies a stream corresponding to a stream type and streamTypeNumber as the target. By specifying a stream, the information relating to that stream can be obtained. streamTypeNumber is an ID between 0 and scePsmfGetNumberOfSpecificStreams()-1 which is assigned in the order written in the PSMF header, starting with 0.

The channels, which are the same as those for sceMpegRegistStream() in libmpeg, are as follows.

 Elementary 
 Stream 		  Stream Type 
 (iStreamType) 		  Stream  
 ID 		  Private 
 Stream ID 		  streamTypeNumber 
 MPEG4 AVC   SCE_MPEG_STREAM_M4AVC   '1110 xxxxb'   None   0-15 
 ATRAC3plus™   SCE_MPEG_STREAM_ATRAC   '1011 1101b'   '0000 xxxxb'   0-15 
 PCM   SCE_MPEG_STREAM_PCM   '1011 1101b'   '0001 xxxxb'   0-15 
 AUDIO   SCE_MPEG_STREAM_AUDIO   ATRAC3plus™ or PCM 


If SCE_MPEG_STREAM_AUDIO is specified, a count is maintained for when the stream is either ATRAC3plus™ or PCM. As a result, the total for SCE_MPEG_STREAM_AUDIO is equal to the sum of the total for SCE_MPEG_STREAM_ATRAC and the total for SCE_MPEG_STREAM_PCM.
