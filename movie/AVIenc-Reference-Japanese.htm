sceAviInitInfo
 [関数]
SceAviInfo構造体の初期化

形式

#include <avienc.h>
void sceAviInitInfo(
	SceAviInfo *ai,
	void *ioref,
	SceAviRiffIOread readfunc,
	SceAviRiffIOwrite writefunc,
	SceAviRiffIOlseek lseekfunc
);

呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  ai     SceAviInfo構造体へのポインタ  
  ioref     I/O関数に渡すパラメータ  
  readfunc     アプリケーションで用意した読み込み関数のエントリポインタ  
  writefunc     アプリケーションで用意した書き込み関数のエントリポインタ  
  lseekfunc     アプリケーションで用意したシーク関数のエントリポインタ  

返り値

なし

解説

SceAviInfo構造体を初期化し、内部でioref、readfunc、writefunc、lseekfuncを引数としてsceAviRiffSetIOfunc()関数を呼び出します。readfuncは使用しませんのでNULLを指定してください。

 [関数]

#include <avienc.h>
void sceAviSimpleAudioHeader(

	int format,
	int channels,
	SceAviRiffUint32 samplesPerSec,
	int bitsPerSample
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  ai     SceAviInfo構造体へのポインタ  
  format     オーディオフォーマット  
  channels     オーディオチャネル数

  bitsPerSample     サンプルのビット数

なし

formatには以下のいずれかを指定します。

SCE_AVI_WAVE_FORMAT_ULAW

formatがSCE_AVI_WAVE_FORMAT_ULAWの場合は、bitsPerSampleは常に8になります。

sceAviRiffSetIOfunc
 [関数]
I/O関数をSceAviRiffInfo構造体に設定

形式

#include <avienc.h>
void sceAviRiffSetIOfunc(
	SceAviRiffInfo *riffInfo,
	void *ioref,
	SceAviRiffIOread readfunc,
	SceAviRiffIOwrite writefunc,
	SceAviRiffIOlseek lseekfunc
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  riffInfo     SceAviRiffInfo構造体へのポインタ  
  ioref     ユーザ定義I/O関数へ渡す任意のパラメータへのポインタ  
  readfunc     アプリケーションで用意した読み込み関数のエントリポインタ  
  writefunc     アプリケーションで用意した書き込み関数のエントリポインタ  
  lseekfunc     アプリケーションで用意したシーク関数のエントリポインタ  

返り値

なし

解説

アプリケーションで用意したI/O関数をSceAviRiffInfo構造体に設定します。
SceAviRiffInfo構造体を利用する全ての関数に先立って設定する必要があります。
ioref引数は、各I/O関数にそのまま渡されます。
各I/O関数は以下の形式でなければなりません。readfuncは使用しませんのでNULLを指定してください。

SceSSize readfunc(void *ioref, void *buf, SceSize count);
SceSSize writefunc(void *ioref, const void *buf, SceSize count);
SceAviRiffOff_t lseekfunc(void *ioref, SceAviRiffOff_t offset, int whence );

sceAviRiffOpen
 [関数]
RIFFアクセスモードの指定

形式

#include <avienc.h>
int sceAviRiffOpen(
	SceAviRiffInfo *riffInfo,
	int mode
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  riffInfo     SceAviRiffInfo構造体へのポインタ  
  mode     アクセスモード  

返り値

正常終了の場合、SCE_AVIRIFF_NOERR
I/Oエラーが起きた場合、SCE_AVIRIFF_ERR_IO
RIFFではない場合、SCE_AVIRIFF_ERR_NOTRIFF

解説

RIFFアクセスモードを指定します。
mode引数には、SCE_AVIRIFF_MODE_WRITEを指定します。
