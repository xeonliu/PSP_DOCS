sceAviRiffOpen
 [Function]
Set RIFF access mode

Syntax

#include <avienc.h>
int sceAviRiffOpen(
	SceAviRiffInfo *riffInfo,
	int mode
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  riffInfo     Pointer to SceAviRiffInfo structure  
  mode     Access mode  

Return Value

If processing successfully completes, SCE_AVIRIFF_NOERR is returned.
If an I/O error occurs, SCE_AVIRIFF_ERR_IO is returned.
If the data format is not RIFF, SCE_AVIRIFF_ERR_NOTRIFF is returned.

Description

This function sets the RIFF access mode.
For the mode argument, specify SCE_AVIRIFF_MODE_WRITE.
The sceAviRiffSetIOfunc() function must be used in advance to set the functions that will actually perform reading and writing.
After the sceAviRiffOpen() function is called, data can be read from and written to a RIFF file through the SceAviRiffInfo structure.
No file is actually opened.

 [Function]

	SceAviInfo *ai,
	SceAviOldIndexEntry *entry,
	int chunktype,
	SceAviRiffUint32 flags,
	SceAviRiffUint32 size,
	void *data
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  ai     Pointer to SceAviInfo structure

  chunktype     Stream chunk type  
  flags     Stream flags  
  size     Stream size  
  data     Stream data  

Return Value

If processing successfully completes, SCE_AVIRIFF_NOERR is returned.
If an I/O error occurs, SCE_AVIRIFF_ERR_IO is returned.

Description

This function sets the SceAviOldIndexEntry structure and writes stream data.
When nothing needs to be set in the SceAviOldIndexEntry structure, NULL can be passed in the entry argument.

  Value     Meaning

  SCE_AVI_STRCHUNK_db     Uncompressed video stream  
  SCE_AVI_STRCHUNK_dc     Compressed video stream  


The following values can be set for the flags argument.
  Value     Meaning  
  0     None  
  SCE_AVI_AVIIF_KEYFRAME     Key frame  

For a motion JPEG video stream, set SCE_AVI_AVIIF_KEYFRAME.
