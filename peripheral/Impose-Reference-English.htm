sceImposeSetUMDPopup
 [Function]
Set popup state when UMD™ cover is opened

Syntax

#include <impose.h>
int sceImposeSetUMDPopup (
	int mode
);
Calling Conditions

Can be called from an interrupt handler (does not depend on interrupt-disabled or -enabled state)
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

 mode   Popup state 
 SCE_IMPOSE_POPUP_DISABLE   Suppress popup when the UMD™ cover is opened 
 SCE_IMPOSE_POPUP_ENABLE   Enable popup when the UMD™ cover is opened 

Return Value

When an error occurs, a negative value is returned.
  Value     Result  
  0     Success  
  < 0     Error  


Description

This function sets the popup state when the UMD™ cover is opened.

Notes

This function should only be used when discs need to be swapped such as with a two-disc set.

See Also

sceImposeGetUMDPopup(), SCE_IMPOSE_POPUP_DISABLE, SCE_IMPOSE_POPUP_ENABLE

sceImposeSetLanguageMode
 [Function]
Set Impose screen display language

Syntax

#include <impose.h>
#include < utility_common.h>
int sceImposeSetLanguageMode (
	int language,
	int button_assign
);
Calling Conditions

Can be called from an interrupt handler (does not depend on interrupt-disabled or -enabled state)
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  language     Language code       
  button_assign     Button assigned to Enter key       
       SCE_UTILITY_CTRL_ASSIGN_CIRCLE_IS_ENTER     Assign O button for Enter key  
       SCE_UTILITY_CTRL_ASSIGN_CROSS_IS_ENTER     Assign X button for Enter key  

Return Value

When an error occurs, a negative value is returned.
  Value     Result  
  0     Success  
  < 0     Error  


Description

This function sets the Impose screen display language and Enter key assignment state.
The valid language codes to which language can be set are the SCE_UTILITY_LANG_xxx constants of the utility service.
The values to which button_assign can be set are the same as those for the controller button assignments of the utility service (SCE_UTILITY_CTRL_ASSIGN_CIRCLE_IS_ENTER or SCE_UTILITY_CTRL_ASSIGN_CROSS_IS_ENTER).
If language is set to a language code that is not available in the system, the language code used will be the one that was set for the "Display Language" in the "PSP™ settings," and no error will be returned.
If this function is called when the Impose screen is being displayed, the language will change beginning with the next time the Impose screen is displayed.

See Also

sceImposeGetLanguageMode(), SCE_UTILITY_LANG_xxx, SCE_UTILITY_CTRL_ASSIGN_CIRCLE_IS_ENTER, SCE_UTILITY_CTRL_ASSIGN_CROSS_IS_ENTER

sceImposeSetBacklightOffTime
 [Function]
Set time until backlight turns off

Syntax

#include <impose.h>
int sceImposeSetBacklightOffTime (
	int param
);
Calling Conditions

Can be called from an interrupt handler (does not depend on interrupt-disabled or -enabled state)
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  param     Time until backlight turns off (seconds)  
  30 - 600 [seconds]  

Return Value

If an error occurs, a negative value is returned.
  Value     Result  
  0     Processing was successful  
  < 0     An error occurred  


Description

This function sets the time until the backlight turns off. Any value from 30 to 600 [seconds] can be specified for the time.
If no operation is performed within 60 seconds, the PSP™' built-in software makes the backlight one level darker, and then turns it off. If the time until the backlight goes off is set to a value less than 60 seconds, the operation for making the backlight one level darker is not performed.

See Also

sceImposeGetBacklightOffTime()
