DMACドライバ

sceDmacMemcpy
 [関数]
DMACによるメモリ転送

形式

#include <dmacdrv.h>
int sceDmacMemcpy (
	void *dst,
	const void *src,
	unsigned int uSize
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  dst     デスティネーションアドレス  
  src     ソースアドレス  
  uSize     転送カウント(転送バイト数)  

返り値

成功した場合は、0を返します。
エラーの場合は、負の数を返します。

解説

DMACによるメモリ転送をおこなう関数です。

ALLEGREX CPUのDキャッシュはライトバック方式ですが、ハードウェアではキャッシュスヌーピングなどのコヒーレンシ保証は行われていません。sceDmacMemcpy()関数によってメモリ転送を行う場合は、ソース領域のDキャッシュライトバックおよびデスティネーション領域のインバリデートを手動で行わなくてはならないことに気をつけてください。
sceDmacMemcpy()による転送は、最大128ビット幅で行われます。引数dstおよびsrcで示されたポインタが16バイトアラインメント境界に設定されているとき、効率が最大になります。

引数dstおよびsrcで示されるポインタと、引数uSizeで示される転送バイト数の関係から転送開始位置と終端付近では、CPUによるアンキャッシュメモリアクセスが行われる事があります。
引数dstおよびuSizeで示される転送先エリアと、引数srcおよびuSizeで示される転送元エリアは重複しないようにしてください。

DMACはシステム内で共有されるリソースであるため、別のスレッドがsceDmacMemcpy()関数等でこのDMACを使用していた場合には、処理がブロックされます。
