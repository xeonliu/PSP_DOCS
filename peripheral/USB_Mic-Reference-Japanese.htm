SceUsbMicParam
 [構造体]
コーデックの設定値

定義

#include <usbmic.h>
typedef struct SceUsbMicParam {
	int alc;
	int gain;
	int noiz;
	int hold;
	int decay;
	int attack;
} SceUsbMicParam;
メンバ

  alc     ALC ON/OFFおよび目標レベル  
  0　　　　:ALC=OFF  
  -6 〜 -29 : ALC=ON -6 〜 -29[dB]  
  gain     入力ゲイン設定  
  ALC=ON時では最大ゲイン  
  　　-12, -6, 0, +6, +12, +18, +24, +30[dB]のいずれか  
  ALC=OFF時は通常の入力ゲイン  
  　　-17.25〜0〜+30[dB]  
  noiz     ノイズゲートON/OFFノイズゲートのスレッショルドレベル  
  (ALC=OFF時は無効)  
  0　　　　:ノイズゲート=OFF  
  -76〜-30 : ノイズゲート=ON -76〜-30[dB]  
  hold     ALCによるゲインアップが始まるまでの時間(ALC=OFF時は無効)  
  0　　:0[ms]  
  1〜15 : 2.667*(2^(hold-1))[ms] → 2.667〜43691[ms]  
  ※効果は設定値に対してべき乗に変化します  
  decay     ALCによるゲインアップの変化時間(ALC=OFF時は無効)  
  0〜10 : 24.0*(2^(decay))[ms] → 24.0〜24580[ms]  
  ※効果は上式のように設定値に対してべき乗に変化します  
  attack     ALCによるゲインダウンの変化時間(ALC=OFF時は無効)  
  0〜10 : 6.0*(2^(attack))[ms] → 6.0〜6144[ms]  
  ※効果は上式のように設定値に対してべき乗に変化します  

解説

コーデックの設定パラメータを表す構造体です。
sceUsbMicInputInitEx()の引数として使用します。
alc, gain, noizの値はsceUsbMicInputInit()に渡す引数と同じ値です。

sceUsbStart
 [関数]
USBドライバの始動

形式

#include <usb.h>
int sceUsbStart(
	const char *name,
	unsigned int args,
	const void *argp
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  name     始動するUSBドライバの名前  
  args     0  
  argp     NULL  

返り値

正常終了の場合は0を返します。
エラーの場合は以下のusb.hに定義されたエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USB_ERROR_ALREADY, SCE_USB_ERROR_NOTAVAIL
SCE_USB_ERROR_NOENT, SCE_USB_ERROR_INPROGRESS
SCE_USB_ERROR_INVAL

解説

USBバスドライバが提供するAPIです。nameで指定した名前のUSBドライバを始動します。
このAPIを使用する場合にはusb_stub.aをリンクしてください。

USBマイクを使用する場合には、以下の順番でUSBドライバを始動させてください。
  1. USBバスドライバ (SCE_USB_BUS_DRIVER_NAME)
  2. USBアクセサリベースドライバ (SCE_USBACC_NAME)
  3. USBマイクドライバ (SCE_USBMIC_NAME)

使用例

// USBバスドライバの始動
if (sceUsbStart(SCE_USB_BUS_DRIVER_NAME, 0, NULL) < 0) {
	// エラー処理
}

// USBアクセサリベースドライバの始動
if (sceUsbStart(SCE_USBACC_NAME, 0, NULL) < 0) {
	// エラー処理
}

// USBマイクドライバの始動
if (sceUsbStart(SCE_USBMIC_NAME, 0, NULL) < 0) {
	// エラー処理
}

sceUsbAccGetAuthStat
 [関数]
USBアクセサリ認証状態の取得

形式

#include <usbacc.h>
int sceUsbAccGetAuthStat(
	void
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフである

引数

なし

返り値

USBアクセサリに接続されている場合は0が返ります。
USBアクセサリ以外のUSBホストに接続されている場合にはSCE_USBACC_ERROR_NOT_ACCESSORYのエラー、USBホストに認識されていない(USB仕様上の状態がSCE_USB_CONFIGUREDになっていない)場合にはSCE_USB_ERROR_NOTAVAILのエラーが返ります。

解説

USBアクセサリ認証状態を取得します。
例えば、PCなどUSBアクセサリ以外のUSBホストに接続された場合には、USBアクセサリの接続を促すダイアログを出すなど、USBアクセサリに接続されているかどうかを判断したい場合に使用してください。

USBマイクライブラリに対応していないUSBアクセサリが接続された場合には、USB仕様上の状態がSCE_USB_CONFIGUREDにならず、SCE_USBACC_ERROR_NOT_ACCESSORYのエラーが返ります。

使用例

int stat;
int ret;

ret = sceUsbWaitState(SCE_USB_CONFIGURED, SCE_KERNEL_EW_AND, NULL);
if (ret < 0) {
	// エラー処理
}

stat = sceUsbAccGetAuthStat();
if (stat == SCE_OK) {
	// USBアクセサリに接続された場合
} else if (stat == SCE_USBACC_ERROR_NOT_ACCESSORY) {
	// USBアクセサリ以外のUSBホストに接続された場合
} else {
	// SCE_USB_CONFIGURED状態ではない場合
}

sceUsbMicInputInit
 [関数]
マイク入力の初期化とコーデックのパラメータ設定

形式

#include <usbmic.h>
int sceUsbMicInputInit(
	int alc,
	int gain,
	int noiz
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  alc     ALC ON/OFFおよび目標レベル  
  gain     最大ゲイン  
  noiz     ノイズゲートON/OFFおよびノイズゲートのスレッショルドレベル  

返り値

正常終了の場合は0を返します。
エラーの場合は以下のエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USBMIC_ERROR_NOT_STARTED

解説

マイク入力の初期化とコーデックのパラメータ設定を行います。
マイク入力を実行中に呼び出すと、マイク入力を停止してパラメータを設定します。

引数alcを負の値に設定することでALC(Auto Level Control)機能を有効にします。
ALCとは音量が小さ過ぎたり大きすぎる場合に入力レベルを一定に保つよう自動調整するための機能です。
alcの値は0[dB]を最大とした場合の目標レベルを整数値[dB]で設定します。ハードウェア仕様より-6〜-29[dB] の範囲で、値は約1[dB]ステップでの設定が可能です。
alcを0以上の値にするとALC機能はOFFになります。

引数gainはALC ONの場合はALCによる最大ゲインの設定となります。ハードウェア仕様より-12〜0〜+30[dB]の範囲で、値は6[dB]ステップになります。よって引数としては1ステップで設定できますが、実際の効果は-12,-6,0,+6,+12,+18,+24,+30 のいずれかになります。
ALC OFF の場合はゲインの絶対値の設定になります。ハードウェア仕様より-18〜+30[dB]の範囲が有効です。約1[dB]ステップでの変更が可能です。

引数noizはノイズゲート機能の設定です。ある一定レベル以下の入力に対してはALCによるゲイン調整を行わないようにします。これにより無音中のヒスノイズがALCによりゲインアップされないようにします。ノイズゲート機能はALC ONの場合のみ有効です。
noizに0を設定した場合はノイズゲート機能はOFFされます。負の値では入力レベルが設定した値をしきい値としてそれ以下の入力レベルではALCによるゲイン調整が効かなくなります。ハードウェア仕様より-76〜-30[dB]の範囲が有効です。約1[dB]ステップで変更可能です。

例1：
設定値 alc= -6 gain = +30 noiz=-60 の場合
入力レベルが-60[dB]以下ではゲイン調整は行われません。
入力レベルが -60〜-6[dB]ではレベルが -6[dB]になるようゲインアップされます。
ゲインアップは最大30[dB]までです。

例2：
設定値 alc=0 gain = +12 noiz=0 の場合
ALC=OFFのため入力信号レベルによらず +12[dB] されます。
ALC=OFFのためノイズゲート機能は働きません。
noizの値は無視されます。

sceUsbMicInputInitEx()との違い
sceUsbMicInputInitEx()ではさらに3つの引数を設定できます。
それ以外は同じ処理を行います。
この関数で設定できない3つの引数は推奨値として以下の値に設定されます。
hold = 0 → 0[ms]
decay = 3 → 192[ms]
attack = 2 → 24[ms]

sceUsbMicInputBlocking
 [関数]
マイク入力（ブロッキング）

形式

#include <usbmic.h>
int sceUsbMicInputBlocking(
	int len,
	int frequency,
	void *datapointer
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  len     入力サンプル数（64の倍数）  
  frequency     取得データのサンプリング周波数 (44100, 22050, 11025のいずれか)  
  datapointer     取得データの格納先へのポインタ  

返り値

正常終了の場合は取得したサンプル数を返します。
エラーの場合は以下のエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USBMIC_ERROR_NOT_STARTED, SCE_USBMIC_ERROR_NOT_AVAILABLE,
SCE_USBMIC_ERROR_INVALID_SIZE, SCE_USBMIC_ERROR_PRIV_REQUIRED,
SCE_USBMIC_ERROR_INPUT_BUSY, SCE_USBMIC_ERROR_INVALID_FREQUENCY

解説

マイク入力を行います。入力が完了するかエラーになるまではスレッドは待ち状態になります。
この関数を実行する前にsceUsbMicInputInit()などで、マイク入力の初期化を行っておく必要があります。初期化をしていない場合には、SCE_ERROR_NOT_INITIALIZEDのエラーが返ります。
datapointerにNULLを設定することで、マイク入力を明示的に終了させることができます。

USBマイクが接続されていない場合や、マイク入力中にサスペンドが行われた場合には、SCE_USBMIC_ERROR_NOT_AVAILABLEのエラーが返ります。ユーザにUSBマイクの接続を促したり、再度入力を要求するなどの対処を行ってください。

使用例

#define SAMPLES  4096

short InputBuf[SAMPLES];  /* 入力バッファ */
int ret;

/* 初期化 ALC_Target=-12[dB] MaxGain=+30[dB] NoiseGateThreshold=-60[dB] */
if ((ret = sceUsbMicInputInit(-12, 30, -60)) < 0) {
	// エラー処理
}

/* マイク入力を行う */
if ((ret = sceUsbMicInputBlocking(SAMPLES, 44100, InputBuf)) < 0) {
	// エラー処理
}

/* マイク入力終了 */
if ((ret = sceUsbMicInputBlocking(SAMPLES, 44100, NULL)) < 0) {
	// エラー処理
}

sceUsbMicInput
 [関数]
マイク入力（ノンブロッキング）

形式

#include <usbmic.h>
int sceUsbMicInput(
	int len,
	int frequency,
	void *datapointer
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフである

引数

  len     入力サンプル数（64の倍数）  
  frequency     取得データのサンプリング周波数 (44100, 22050, 11025のいずれか)  
  datapointer     取得データの格納先へのポインタ  

返り値

正常終了の場合は0を返します。
エラーの場合は以下のエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USBMIC_ERROR_NOT_STARTED, SCE_USBMIC_ERROR_NOT_AVAILABLE,
SCE_USBMIC_ERROR_INVALID_SIZE, SCE_USBMIC_ERROR_PRIV_REQUIRED,
SCE_USBMIC_ERROR_INPUT_BUSY, SCE_USBMIC_ERROR_INVALID_FREQUENCY

解説

マイク入力を行います。マイク入力を開始した時点で関数から戻り、待ち状態になることはありません。マイク入力の完了はsceUsbMicWaitInputEnd()やsceUsbMicPollInputEnd()で確認することができます。
この関数を実行する前にsceUsbMicInputInit()などで、マイク入力の初期化を行っておく必要があります。初期化をしていない場合には、SCE_ERROR_NOT_INITIALIZEDのエラーが返ります。
datapointerにNULLを設定することで、マイク入力を明示的に終了させることができます。この場合にも、sceUsbMicWaitInputEnd()やsceUsbMicPollInputEnd()で終了を確認してください。

USBマイクが接続されていない場合や、サスペンド処理を実行中であった場合には、SCE_USBMIC_ERROR_NOT_AVAILABLEのエラーが返ります。ユーザにUSBマイクの接続を促したり、再度入力を要求するなどの対処を行ってください。
使用例

#define SAMPLES  4096

short InputBuf[SAMPLES];  /* 入力バッファ */
int ret;

/* 初期化 ALC_Target=-12[dB] MaxGain=+30[dB] NoiseGateThreshold=-60[dB] */
if ((ret = sceUsbMicInputInit(-12, 30, -60)) < 0) {
	// エラー処理
}

/* マイク入力を行う */
if ((ret = sceUsbMicInput(SAMPLES, 44100, InputBuf)) < 0) {
	// エラー処理
}

/* マイク入力完了待ち */
if ((ret = sceUsbMicWaitInputEnd()) < 0) {
	// エラー処理
}
