SCE_USBGPS_NUM_SATELLITE
 [定数定義]
USB GPSドライバが対応するGPS衛星の最大数

定義

#include <usbgps.h>
#define SCE_USBGPS_NUM_SATELLITE		(32)
解説

SCE_USBGPS_STATE_WORKING
 [定数定義]
GPSレシーバーの状態を表す定数

定義

#include <usbgps.h>
#define SCE_USBGPS_STATE_WORKING		3
解説

sceUsbGpsGetState()で取得可能な、GPSレシーバーの状態を表す定数です。GPSレシーバーが起動し、sceUsbGpsGetData()で位置情報・GPS衛星情報を取得可能な状態をあらわします。ただし、sceUsbGpsGetData()で取得した位置情報が実際に有効なものかどうかは、SceUsbGpsPositionData構造体のmodeもしくは、qualityで確認します。modeがSCE_USBGPS_POSITION_MODE_NOT_AVAILABLE、もしくは、qualityがSCE_USBGPS_POSITION_QUALITY_NOT_AVAILABLEである場合、SceUsbGpsPositionData構造体のその他のメンバの値は無効です。

SCE_USBGPS_POSITION_QUALITY_DGPS
 [定数定義]
測位クォリティを表す定数

定義

#include <usbgps.h>
#define SCE_USBGPS_POSITION_QUALITY_DGPS	2
解説

GPSレシーバーの測位クォリティを表す定数です。DGPSを併用して測位していることを表します。

備考

DGPSとは、位置の分かっている基準局が発信する電波を利用して、GPS計測結果の誤差を修正して精度を高める技術のことをいいます。GPSレシーバーでは、SBAS方式（静止衛星型衛星航法補強システム）によるDGPS測位に対応しています。

SceUsbGpsPositionAngle
 [構造体]
緯度経度を度・分・秒・ミリ秒で表す構造体

定義

#include <usbgpsh>
typedef struct SceUsbGpsPositionAngle {
	SceShort16 degree;
	SceUShort16 minute;
	SceUShort16 second;
	SceUShort16 milisecond;
} SceUsbGpsPositionAngle;
メンバ

  degree     緯度・経度の度成分。0より大きい場合は、北緯または、東経を表し、  
  0より小さい場合は、南緯または、西経を表します。  
  minute     緯度・経度の分成分  
  second     緯度・経度の秒成分  
  milisecond     緯度・経度のミリ秒成分  

解説

緯度・経度を度・分・秒・ミリ秒で表す構造体です。degreeの符号によって、北緯・南緯、東経・西経を判別できます。また、sceUsbGpsFloatToPositionAngle(), sceUsbGpsPositionAngleToFloat()を使ってSceFloat32形式の緯度経度と相互に変換可能です。

関連項目

sceUsbGpsFloatToPositionAngle(), sceUsbGpsPositionAngleToFloat()

SceUsbGpsPositionData
 [構造体]
位置情報を表す構造体

定義

#include <usbgps.h>
typedef struct SceUsbGpsPositionData {
	ScePspDateTime datetime;
	SceFloat32 hdop;
	SceUChar8 quality;
	SceUChar8 mode;
	SceUChar8 reserve0[2];
	SceFloat32 latitude;
	SceFloat32 longitude;
	SceFloat32 altitude;
	SceFloat32 reserve1;
	SceFloat32 speed;
	SceFloat32 course;
} SceUsbGpsPositionData;
メンバ

  datetime     GPSレシーバーから衛星情報を取得した日時をUTC形式で返します。  
  hdop     水平精度低下率  
  quality     測位クォリティ  
  mode     測位モード  
  reserve0     予備（内容を評価しないでください）  
  latitude     緯度（度）  
  longitude     経度（度）  
  altitude     平均海水面からの高度（メートル）  
  reserve1     予備（内容を評価しないでください）  
  speed     地表速度（キロメートル毎時）  
  course     進行方向（度）  

解説

位置情報を表す構造体です。
sceUsbGpsGetData()の引数として使用します。

備考

  • GPSによる測位では、天空におけるGPS衛星の配置によって単独測位、相対測位ともに測位精度が影響をうけます。その度合いを示す数値の一つがhdop（水平精度低下率）で、値が大きくなるほど精度が低くなります。
  • altitude（平均海水面からの高度）は、mode（測位モード）が、SCE_USBGPS_POSITION_MODE_3Dの場合のみ有効です。
  • course（進行方向）は、移動時のみ有効です。

関連項目

sceUsbGpsGetData(), sceUsbGpsFloatToPositionAngle()

sceUsbStart
 [関数]
USBドライバの始動

形式

#include <usb.h>
int sceUsbStart(
	const char *name,
	unsigned int args,
	const void *argp
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  name     始動するUSBドライバの名前  
  args     0  
  argp     NULL  

返り値

正常終了の場合は0を返します。
エラーの場合は以下のusb.hに定義されたエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USB_ERROR_ALREADY, SCE_USB_ERROR_NOTAVAIL
SCE_USB_ERROR_NOENT, SCE_USB_ERROR_INPROGRESS
SCE_USB_ERROR_INVAL

解説

USBバスドライバが提供するAPIです。nameで指定した名前のUSBドライバを始動します。
このAPIを使用する場合にはusb_stub.aをリンクしてください。

USBGPSレシーバーを使用する場合には、以下の順番でUSBドライバを始動させてください。
  1. USBバスドライバ(SCE_USB_BUS_DRIVER_NAME)
  2. USBアクセサリベースドライバ(SCE_USBACC_NAME)
  3. USB GPSドライバ(SCE_USBGPS_NAME)

使用例

// USBバスドライバの始動
if (sceUsbStart(SCE_USB_BUS_DRIVER_NAME, 0, NULL) < 0) {
	// エラー処理
}

// USBアクセサリベースドライバの始動
if (sceUsbStart(SCE_USBACC_NAME, 0, NULL) < 0) {
	// エラー処理
}

// USB GPSドライバの始動
if (sceUsbStart(SCE_USBGPS_NAME, 0, NULL) < 0) {
	// エラー処理
}

引数

  name     停止するUSBドライバの名前  
  args     0

返り値

正常終了の場合は0を返します。
エラーの場合は以下のusb.hに定義されたエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USB_ERROR_ALREADY, SCE_USB_ERROR_NOTAVAIL

	// エラー処理
}
// USBバスドライバの停止
if (sceUsbStop(SCE_USB_BUS_DRIVER_NAME, 0, NULL) < 0) {
	// エラー処理

sceUsbActivate
 [関数]
USBデバイスの始動

形式

#include <usb.h>
int sceUsbActivate(
	unsigned int idProduct
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  idProduct     プロダクトID  

返り値

正常終了の場合は0を返します。
エラーの場合は以下のusb.hに定義されたエラーコードかkerror.hやpsperror.hに定義されたエラーコードを返します。

SCE_USB_ERROR_ALREADY, SCE_USB_ERROR_NOENT, SCE_USB_ERROR_NOSPC
SCE_USB_ERROR_NOMEM, SCE_USB_ERROR_INPROGRESS
SCE_USB_ERROR_INVAL, SCE_USB_ERROR_NOTAVAIL

解説

USBバスドライバが提供するAPIです。プラグインされたUSBドライバを取りまとめて、１つのUSBデバイスとして始動します。USBデバイスとして始動するまではUSBケーブルを接続してもホストには認識されません。
このAPIを使用する場合には、usb_stub.aをリンクしてください。
USBデバイスとして始動するためには、予め必要なUSBドライバを始動しておく必要があります。

GPSレシーバーを使用する場合には、引数のプロダクトIDにusbgps.hで定義されているSCE_USB_PID_GPSを指定してください。

使用例

if (sceUsbActivate(SCE_USB_PID_GPS) < 0) {
	// エラー処理
} else {
	printf("success to activate\n");
}
