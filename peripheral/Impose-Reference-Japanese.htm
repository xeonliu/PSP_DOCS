sceImposeSetUMDPopup
 [関数]
UMD®蓋開けによるポップアップ状態を設定

形式

#include <impose.h>
int sceImposeSetUMDPopup (
	int mode
);
呼出条件

割り込みハンドラから呼び出し可能（割り込み禁止、許可状態に依存しない）
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

 mode   ポップアップ状態 
 SCE_IMPOSE_POPUP_DISABLE   UMD®蓋開けによるポップアップを抑制 
 SCE_IMPOSE_POPUP_ENABLE   UMD®蓋開けによるポップアップを許可 

返り値

エラーのとき負の値を返します。
  値     結果  
  0     成功  
  <0     エラー  


解説

UMD®蓋開けによるポップアップ状態を設定します。

備考

当該関数はディスク2枚組等のディスク入れ替えが必要な場合においてのみご使用いただけます。

関連項目

sceImposeGetUMDPopup(), SCE_IMPOSE_POPUP_DISABLE, SCE_IMPOSE_POPUP_ENABLE

sceImposeSetLanguageMode
 [関数]
インポーズ画面の表示言語を設定

形式

#include <impose.h>
#include < utility_common.h>
int sceImposeSetLanguageMode (
	int language,
	int button_assign
);
呼出条件

割り込みハンドラから呼び出し可能（割り込み禁止、許可状態に依存しない）
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  language     言語コード       
  button_assign     決定キーの割り当て状態       
       SCE_UTILITY_CTRL_ASSIGN_CIRCLE_IS_ENTER     ○キーを決定キーにする  
       SCE_UTILITY_CTRL_ASSIGN_CROSS_IS_ENTER     ×キーを決定キーにする  

返り値

エラーのとき負の値を返します。
  値     結果  
  0     成功  
  <0     エラー  


解説

インポーズ画面の表示言語および決定キーの割り当て状態を設定します。
languageに設定する言語コードはユーティリティサービスのSCE_UTILITY_LANG_xxxと共通です。
button_assignに設定する値はユーティリティサービスのコントローラのボタンアサイン（SCE_UTILITY_CTRL_ASSIGN_CIRCLE_IS_ENTER, SCE_UTILITY_CTRL_ASSIGN_CROSS_IS_ENTER）と共通です。
languageに指定した言語コードがシステムに存在しない言語コードだった場合、システムの「本体設定」の「表示言語」で設定した言語コードとなり、エラーは返しません。
インポーズ画面が表示されているときに呼び出した場合、次のインポーズ画面の表示から設定した言語に変わります。

関連項目

sceImposeGetLanguageMode(), SCE_UTILITY_LANG_xxx, SCE_UTILITY_CTRL_ASSIGN_CIRCLE_IS_ENTER, SCE_UTILITY_CTRL_ASSIGN_CROSS_IS_ENTER

sceImposeSetBacklightOffTime
 [関数]
バックライト消灯までの時間を設定

形式

#include <impose.h>
int sceImposeSetBacklightOffTime (
	int param
);
呼出条件

割り込みハンドラから呼び出し可能（割り込み禁止、許可状態に依存しない）
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  param     バックライト消灯までの時間(秒単位)  
  30〜600[秒]  

返り値

エラーのとき負の値を返します。
  値     結果  
  0     成功  
  <0     エラー  


解説

バックライト消灯までの時間を設定します。
消灯までの時間には30〜600[秒]を与えることができます。
PSP®内蔵ソフトウェアでは、60秒間オペレーションがないとバックライトを1段階暗くし、その後消灯します。消灯までの時間を60秒より小さくした場合、バックライトを1段階暗くする動作は行われません。

関連項目

sceImposeGetBacklightOffTime()
