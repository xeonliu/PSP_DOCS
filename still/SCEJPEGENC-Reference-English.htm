JPEG Encoder Reference

sceJpegEncoderGetContextSize
 [Function]
Get work memory size

Syntax

#include <scejpegenc.h>
int sceJpegEncoderGetContextSize(
	void
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

None

Return Value

The work memory size to be used for encoding is returned.

Description

This function must be used to obtain the work memory size.

sceJpegEncoderInit
 [Function]
Initialize encoder

Syntax

#include <scejpegenc.h>
int sceJpegEncoderInit(
	void *pContext,
	int iFrameWidth,
	int iFrameHeight,
	void *pJpeg,
	SceSize oJpegbufSize
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  iFrameWidth     Width of frame buffer where image to be encoded is stored  
  iFrameHeight     Height of frame buffer where image to be encoded is stored  
  pJpeg     Pointer to buffer for storing encoded result  
  oJpegbufSize     Size of buffer for storing encoded result  

Return Value


  Value     Description  
  SCE_JPEGENC_OK     Processing completed successfully  
  SCE_JPEGENC_ERROR_IMAGE_SIZE     Size of frame buffer where image to be  
  encoded is stored is invalid  
  SCE_JPEGENC_ERROR_BAD_ALIGNMENT     Encoder work area starting address is   
  invalid  


Description

This function initializes the encoder.
For pContext, specify the encoder work area. Specify a pointer to an area with a byte size greater than the value returned by the sceJpegEncoderGetContextSize() function. Make sure that the starting address of the area is a multiple of 16. Do not allow this area to be accessed directly until JPEG encoding is finished.

For iFrameWidth and iFrameHeight, specify the size of the frame buffer where the image to be encoded is stored. iFrameWidth and iFrameHeight are the x- and y-direction sizes (number of pixels), respectively. Make sure they are both multiples of 16.
For pJpeg, specify a pointer to the buffer for storing the encoded result, and for oJpegbufSize, specify the size of that buffer.

sceJpegEncoderEncode
 [Function]
Perform encoding processing

Syntax

#include <scejpegenc.h>
int sceJpegEncoderEncode(
	void *pContext,
	int *pRGB
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  pRGB     Pointer to frame buffer where image to be encoded is stored  

Return Value


  Value     Description  
  Positive number     Processing completed successfully.  
  Value represents JPEG size of   
  encoded result.  
  SCE_JPEGENC_ERROR_INSUFFICIENT_BUFFER     Size of buffer for storing encoded   
  result is insufficient  
  SCE_JPEGENC_ERROR_BAD_ALIGNMENT     Starting address of frame buffer   
  where image to be encoded is stored   
  is invalid  


Description

This function performs encoding processing. The encoded result is stored in the buffer pointed to by pJpeg previously specified with sceJpegEncoderInit(). The return value is the JPEG size.
For pContext, specify the work area of the encoder that was initialized by sceJpegEncoderInit().

For pRGB, specify a pointer to a memory location where the top left pixel of the image to be encoded is stored. Ensure that the starting address of the area is a multiple of 16.

sceJpegEncoderEncodeVFPU
 [Function]
Perform encoding processing (using the VFPU and saving VFPU registers)

Syntax

#include <scejpegenc.h>
int sceJpegEncoderEncodeVFPU(
	void *pContext,
	int *pRGB
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  pRGB     Pointer to frame buffer where image to be encoded is stored  

Return Value


  Value     Description  
  Positive number     Processing completed successfully.  
  Value represents JPEG size of   
  encoded result.  
  SCE_JPEGENC_ERROR_INSUFFICIENT_BUFFER     Size of buffer for storing encoded   
  result is insufficient  
  SCE_JPEGENC_ERROR_BAD_ALIGNMENT     Starting address of frame buffer   
  where image to be encoded is   
  stored is invalid  


Description

This function performs encoding quickly by using the VFPU. At this time, it saves the VFPU registers. To use this API, link the static link library file libscejpegenc_vfpu.a. Some precautions must be noted when using the VFPU. For details, refer to the VFPU Overview.
The encoded result is saved in the buffer pointed to by pJpeg previously specified by sceJpegEncoderInit(). The JPEG size is returned as the return value.
For pContext, specify the work area of the encoder that was initialized by sceJpegEncoderInit().

For pRGB, specify a pointer to a memory location where the top left pixel of the image to be encoded is stored. Make sure that the starting address of the area is a multiple of 16.

sceJpegEncoderSetValidRegion
 [Function]
Specify encoding region

Syntax

#include <scejpegenc.h>
int sceJpegEncoderSetValidRegion(
	void *pContext,
	int iFrameWidth,
	int iFrameHeight
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  iFrameWidth     Width of frame buffer for encoding image to be encoded  
  iFrameHeight     Height of frame buffer for encoding image to be encoded  

Return Value


  Value     Description  
  SCE_JPEGENC_OK     Processing completed successfully  
  SCE_JPEGENC_ERROR_IMAGE_SIZE     Size of frame buffer where image to be encoded is stored is invalid  


Description

This function specifies the region to be encoded.
If you want to set only part of the frame buffer as the valid region, you can use this function to specify the size of the valid region so that the unnecessary part will not be encoded.

For pContext, specify the work area of the encoder that was initialized by sceJpegEncoderInit(). For iFrameWidth and iFrameHeight ,specify the size of the region to be encoded. iFrameWidth and iFrameHeight are the x- and y-direction sizes (number of pixels), respectively.

Note

Be sure to call sceJpegEncoderInit() before calling this function.

sceJpegEncoderSetCompressionRatio
 [Function]
Set compression ratio

Syntax

#include <scejpegenc.h>
int sceJpegEncoderSetCompressionRatio(
	void *pContext,
	int compratio
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  compratio     Compression ratio  

Return Value


  Value     Description  
  SCE_JPEGENC_OK     Processing completed successfully  
  SCE_JPEGENC_ERROR_INVALID_COMPRATIO     Compression ratio is invalid  


Description

This function sets the JPEG compression ratio.
For pContext, specify the work area of the encoder that was initialized by sceJpegEncoderInit().

For compratio, specify the compression ratio. Values from 0 (SCE_JPEGENC_COMPRATIO_NONE) to 255 (SCE_JPEGENC_COMPRATIO_HIGH) can be specified. When 0 is set, the compression ratio will be at its minimum (highest quality). When 255 is set, the compression ratio will be at its maximum (lowest quality).
After the encoder is initialized, 64 (SCE_JPEGENC_COMPRATIO_DEFAULT) is specified by default.

Note

Be sure to call sceJpegEncoderInit() before calling this function.

sceJpegEncoderSetPixelFormat
 [Function]
Set input frame buffer pixel format

Syntax

#include <scejpegenc.h>
int sceJpegEncoderSetPixelFormat(
	void *pContext,
	int pixelformat
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  pixelformat     Pixel format of frame buffer for image to be encoded  

Return Value


  Value     Description  
  SCE_JPEGENC_OK     Processing completed successfully  
  SCE_JPEGENC_ERROR_INVALID_PIXELFORMAT     Compression ratio is invalid  


Description

This function sets the pixel format of the input frame buffer.
For pContext, specify the work area of the encoder that was initialized by sceJpegEncoderInit().

For pixelformat, specify the value that indicates the pixel format.
Specify either SCE_JPEGENC_PIXEL_RGBA5551 (R:G:B:A=5:5:5:1 16 bits) or SCE_JPEGENC_PIXEL_RGBA8888 (R:G:B:A=8:8:8:8 32 bits).
After the encoder is initialized, SCE_JPEGENC_PIXEL_RGBA8888 is specified by default.

Note

Be sure to call sceJpegEncoderInit() before calling this function.

sceJpegEncoderSetHeaderMode
 [Function]
Specify header format

Syntax

#include <scejpegenc.h>
int sceJpegEncoderSetHeaderMode(
	void *pContext,
	int headermode
)
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Not multithread safe

Arguments

  pContext     Pointer to encoder work area  
  headermode     Format of header part of output data  

Return Value


  Value     Description  
  SCE_JPEGENC_OK     Processing completed successfully  
  SCE_JPEGENC_ERROR_INVALID_HEADER_MODE     Header format is invalid  


Description

This function specifies the format of the header part of the output data.
For pContext, specify the work area of the encoder that was initialized by sceJpegEncoderInit().

For headermode, specify the value that indicates the header format.
To output data as Motion JPEG, specify SCE_JPEGENC_MODE_MJPEG, and to output it as a normal JPEG file, specify SCE_JPEGENC_MODE_JPEG.

After the encoder is initialized, SCE_JPEGENC_MODE_JPEG is specified by default.

Note

Be sure to call sceJpegEncoderInit() before calling this function.
