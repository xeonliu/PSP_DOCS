SCE_PNG_FORMAT_BPP_MASK
 [定数定義]
formatを示す戻り値からビット数を取得するのに使う値

定義

#include <scepng.h>
#define SCE_PNG_FORMAT_BPP_MASK	(0xff)
解説

scePngDec()から返されるformatの値に対して、この値と論理積をとると、出力された画像データの1ピクセルあたりのビット数を得ることができます。

scePngDec
 [関数]
PNGデコード関数

形式

#include <scepng.h>
int scePngDec(
	void *dst,
	unsigned int outsize,
	const unsigned char *src,
	unsigned int insize,
	int *width,
	int *height,
	int *format
)
呼出条件

割り込みハンドラから呼び出し可
スレッドから呼び出し可能（割り込み禁止・許可状態に依存しない）
マルチスレッドセーフである

引数

  dst     デコード結果格納先バッファへのポインタ  
  outsize     デコード結果格納先バッファのサイズ  
  src     PNGデータが格納されたバッファへのポインタ  
  insize     PNGデータのサイズ  
  width     デコード結果データの幅(ピクセル数)を格納する先へのポインタ  
  height     デコード結果データの高さ(ピクセル数)を格納する先へのポインタ  
  format     デコード結果データの形式を格納する先へのポインタ  

返り値

正常動作時、デコード結果データのバイト数を返します。
負数の時、以下のいずれかのエラー
  値     説明  
  SCE_ERROR_INVALID_FORMAT     不正なデータなのでデコードできない  
  SCE_ERROR_INVALID_SIZE     出力バッファが小さいためデコードできない  
  SCE_ERROR_NOT_SUPPORTED     このライブラリで対応していない形式の  
  PNGデータなのでデコードできない  


解説

与えられたPNGデータを展開し、ラスタースキャン形式（またはADAM7インターレーススキャン形式）のデータに変換します。また入力されたPNGデータがCLUT(PALETTE)を使うものであった場合、CLUTの内容もあわせて出力されます。
出力データは（もしあれば）CLUTデータで始まり、続けて画像データとなります。詳しくは「PNGライブラリ」概要ドキュメントをご覧ください。

*formatにはデコードされた結果のデータ形式が格納されます。これは、以下の値の論理和となります。

1ピクセルあたりのビット数(1〜64)
SCE_PNG_FORMAT_FLAG_ALPHA
SCE_PNG_FORMAT_FLAG_CLUT
SCE_PNG_FORMAT_FLAG_GRAYSCALE
SCE_PNG_FORMAT_FLAG_INTERLACE

scePngGetEncSize
 [関数]
PNGエンコード結果サイズの見積もり

形式

#include <scepng.h>
int scePngGetEncSize(
	int width,
	int height
)
呼出条件

割り込みハンドラから呼び出し可
スレッドから呼び出し可能（割り込み禁止・許可状態に依存しない）
マルチスレッドセーフである

引数

  width     エンコードしたいデータの幅(ピクセル数)  
  height     エンコードしたいデータの高さ(ピクセル数)  

返り値

正常動作時、与えられたサイズの画像をエンコードした結果のデータサイズ(バイト数)の最大値を返します。
負数の時、以下のエラー
  値     説明  
  SCE_ERROR_INVALID_SIZE     画像サイズが巨大すぎる  


解説

エンコード結果の見積もりを返します。通常はscePngEnc()で生成されるデータのサイズは、この関数の戻り値を超えることはありません（ただし、数バイト程度小さくなることはあります）。
エンコードする前にこの関数を呼び出すことで、出力バッファのサイズが足りているかチェックすることができます。

  width     エンコードしたいデータの幅(ピクセル数)  
  height     エンコードしたいデータの高さ(ピクセル数)  
  framewidth     フレームバッファの幅(ピクセル数)  
  format     エンコードしたいデータのピクセルフォーマット  

返り値

正常動作時、エンコード結果データのバイト数を返します。
負数の時、以下のエラー
  値     説明  
  SCE_ERROR_NOT_SUPPORTED     このライブラリで対応していない形式なので  
  エンコードできない  


解説
