sceJpegEncoderGetContextSize
 [関数]
ワークメモリのサイズ取得

形式

#include <scejpegenc.h>
int sceJpegEncoderGetContextSize(
	void
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

なし

返り値

エンコードに使うワークメモリのサイズを返します。

解説

ワークメモリのサイズ取得は必ずこの関数を使用してください。

sceJpegEncoderInit
 [関数]
エンコーダの初期化

形式

#include <scejpegenc.h>
int sceJpegEncoderInit(
	void *pContext,
	int iFrameWidth,
	int iFrameHeight,
	void *pJpeg,
	SceSize oJpegbufSize
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  pContext     エンコーダのワーク領域へのポインタ  
  iFrameWidth     エンコード対象画像の格納されたフレームバッファの幅  
  iFrameHeight     エンコード対象画像の格納されたフレームバッファの高さ  
  pJpeg     エンコード結果を格納するバッファへのポインタ  
  oJpegbufSize     エンコード結果を格納するバッファのサイズ  

返り値


  値     説明  
  SCE_JPEGENC_OK     正常終了  
  SCE_JPEGENC_ERROR_IMAGE_SIZE     エンコード対象画像の格納された  
  フレームバッファのサイズ不正  
  SCE_JPEGENC_ERROR_BAD_ALIGNMENT     エンコーダのワーク領域先頭アドレス不正  


解説

エンコーダの初期化を行います。
pContextにはエンコーダのワーク領域を指定します。sceJpegEncoderGetContextSize()関数が返した値よりも大きなバイト数の領域へのポインタを指定してください。領域の先頭アドレスは16の倍数になるようにしてください。JPEGエンコードを終了するまで領域内に直接アクセスせずに保持するようにしてください。

iFrameWidth、iFrameHeightはエンコードしたい画像の格納されたフレームバッファのサイズを指定してください。それぞれx, y方向のサイズ（ピクセル数）となります。どちらも16の倍数にしてください。
pJpegにはエンコード結果を格納するバッファへのポインタ、oJpegbufSizeにはそのバッファのサイズを指定します。

sceJpegEncoderEncode
 [関数]
エンコード処理の実行

形式

#include <scejpegenc.h>
int sceJpegEncoderEncode(
	void *pContext,
	int *pRGB
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  pContext     エンコーダのワーク領域へのポインタ  
  pRGB     エンコード対象画像の格納されたフレームバッファへのポインタ  

返り値


  値     説明  
  正数     正常終了  
  エンコード結果のJPEGサイズ  
  SCE_JPEGENC_ERROR_INSUFFICIENT_BUFFER     エンコード結果を格納するバッファの  
  サイズ不足  
  SCE_JPEGENC_ERROR_BAD_ALIGNMENT     エンコード対象画像の格納された  
  フレームバッファ先頭アドレス不正  


解説

エンコード処理を行います。エンコード結果はsceJpegEncoderInit()で指定したpJpegに格納されます。JPEGのサイズは戻り値として返します。
pContextにはsceJpegEncoderInit()で初期化したエンコーダのワーク領域を指定します。

pRGBにはエンコードしたい画像の左上ピクセルが格納された先へのポインタを指定してください。領域の先頭アドレスは16の倍数になるようにしてください。

sceJpegEncoderSetValidRegion
 [関数]
エンコード領域の指定

形式

#include <scejpegenc.h>
int sceJpegEncoderSetValidRegion(
	void *pContext,
	int iFrameWidth,
	int iFrameHeight
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  pContext     エンコーダのワーク領域へのポインタ  
  iFrameWidth     エンコード対象画像のエンコードしたいフレームバッファの幅  
  iFrameHeight     エンコード対象画像のエンコードしたいフレームバッファの高さ  

返り値


  値     説明  
  SCE_JPEGENC_OK     正常終了  
  SCE_JPEGENC_ERROR_IMAGE_SIZE     エンコード対象画像のエンコードしたい  
  フレームバッファのサイズ不正  


解説

エンコードしたい領域を指定します。
フレームバッファ中で一部分のみ有効な領域としたい場合、有効な領域のサイズを指定することで、不要な部分をエンコードしないようにすることができます。

pContextにはsceJpegEncoderInit()で初期化したエンコーダのワーク領域を指定します。iFrameWidth、iFrameHeightにエンコードしたい領域サイズを指定します。それぞれx, y方向のサイズ（ピクセル数）となります。

注意

sceJpegEncoderInit()を呼び出した後に呼び出すようにしてください。

sceJpegEncoderSetCompressionRatio
 [関数]
圧縮係数の設定

形式

#include <scejpegenc.h>
int sceJpegEncoderSetCompressionRatio(
	void *pContext,
	int compratio
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  pContext     エンコーダのワーク領域へのポインタ  
  compratio     圧縮係数  

返り値


  値     説明  
  SCE_JPEGENC_OK     正常終了  
  SCE_JPEGENC_ERROR_INVALID_COMPRATIO     圧縮係数不正  


解説

JPEGの圧縮係数を設定します。
pContextにはsceJpegEncoderInit()で初期化したエンコーダのワーク領域を指定します。

compratioに圧縮係数を指定します。0（SCE_JPEGENC_COMPRATIO_NONE）から255（SCE_JPEGENC_COMPRATIO_HIGH）までの値が指定できます。0にした時に最も圧縮率が下がり（品質は良くなり）、255にしたときに最も圧縮率が上がります（品質は下がります）。
エンコーダ初期化後、デフォルトでは64（SCE_JPEGENC_COMPRATIO_DEFAULT）を指定しています。

注意

sceJpegEncoderInit()を呼び出した後に呼び出すようにしてください。

sceJpegEncoderSetPixelFormat
 [関数]
入力フレームバッファのピクセルフォーマットの設定

形式

#include <scejpegenc.h>
int sceJpegEncoderSetPixelFormat(
	void *pContext,
	int pixelformat
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  pContext     エンコーダのワーク領域へのポインタ  
  pixelformat     エンコード対象画像のフレームバッファのピクセルフォーマット  

返り値


  値     説明  
  SCE_JPEGENC_OK     正常終了  
  SCE_JPEGENC_ERROR_INVALID_PIXELFORMAT     圧縮係数不正  


解説

入力フレームバッファのピクセルフォーマットを設定します。
pContextにはsceJpegEncoderInit()で初期化したエンコーダのワーク領域を指定します。

pixelformatにピクセルフォーマットを示す値を指定します。
SCE_JPEGENC_PIXEL_RGBA5551 (R:G:B:A=5:5:5:1 16bit)または、SCE_JPEGENC_PIXEL_RGBA8888 (R:G:B:A=8:8:8:8 32bit)のどちらかを指定してください。
エンコーダ初期化後、デフォルトではSCE_JPEGENC_PIXEL_RGBA8888を指定しています。

注意

sceJpegEncoderInit()を呼び出した後に呼び出すようにしてください。

sceJpegEncoderSetHeaderMode
 [関数]
ヘッダのフォーマットを指定

形式

#include <scejpegenc.h>
int sceJpegEncoderSetHeaderMode(
	void *pContext,
	int headermode
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  pContext     エンコーダのワーク領域へのポインタ  
  headermode     出力データのヘッダ部分のフォーマット  

返り値


  値     説明  
  SCE_JPEGENC_OK     正常終了  
  SCE_JPEGENC_ERROR_INVALID_HEADER_MODE     ヘッダのフォーマット不正  


解説

出力データのヘッダ部分のフォーマットを指定します。
pContextにはsceJpegEncoderInit()で初期化したエンコーダのワーク領域を指定します。

headermodeにはヘッダ形式を示す値を指定します。
Motion JPEGとして出力したい時にはSCE_JPEGENC_MODE_MJPEGを、通常のJPEGファイルとして出力したい時にはSCE_JPEGENC_MODE_JPEGを指定してください。

エンコーダ初期化後、デフォルトではSCE_JPEGENC_MODE_JPEGを指定しています。

注意

sceJpegEncoderInit()を呼び出した後に呼び出すようにしてください。
