SceBool
 [Primitive Types]
Boolean value

Syntax

#include <psptypes.h>
typedef int SceBool;
Description

This type represents a Boolean value. The following macros are defined in psptypes.h.
#define SCE_FALSE 0
#define SCE_TRUE 1
A variable of type SceBool must have one of these two values.
Also, the value must not depend on sizeof(SceBool).

ScePspFColor
 [Color Types]
Floating-point color

Syntax

typedef struct ScePspFColor {
	float r, g, b, a;
} ScePspFColor __attribute__((aligned(16)));
Description

This is a color vector containing 4 elements of type float.
The color vector represents black level when each of the r, g, and b elements has a normalized level of 0.0, or white level when the normalized level of those elements is 1.0.
The color is transparent when the normalized level of the a element is 0.0, and opaque when it is 1.0.

ScePspRGBA8888 / ScePspRGBA32
 [Color Types]
32-bit color

Syntax

#include <psptypes.h>
#define ScePspRGBA32	ScePspRGBA8888;
typedef unsigned int    ScePspRGBA8888;
Description

This type represents a 32-bit color value. The breakdown of the 32 bits is as follows:
bits 31 to 24: A (alpha channel)
bits 23 to 16: B (blue)
bits 15 to 8: G (green)
bits 7 to 0: R (red)

See Also

ScePspRGBA4444, ScePspRGBA5551, ScePspRGB565

ScePspFQuaternion
 [Structure]
4-dimensional quaternion (floating-point)

Structure

#include <psptypes.h>
typedef struct ScePspFQuaternion {
	float x, y, z, w;
} ScePspFQuaternion;
Description

This structure is a 4-element, 4-dimensional quaternion with elements of type float.

x, y, and z represent axes, and w represents the rotation amount. The relationships between the unit quaternion q, the normalized rotation axis vector axis, and the rotation amount θ are as follows.
q.x = sin(θ/2) * axis.x
q.y = sin(θ/2) * axis.y
q.z = sin(θ/2) * axis.z
q.w = cos(θ/2)

See Also

sceVfpuQuaternionFromRotate()
