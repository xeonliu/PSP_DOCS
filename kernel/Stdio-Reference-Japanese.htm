STDIOマネージャ


sceKernelRegisterStdoutPipe
 [リダイレクト操作]
標準出力をリダイレクトするメッセージパイプの登録

形式

#include <kernel.h>
int sceKernelRegisterStdoutPipe (
	SceUID uid
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能(割り込み許可, ディスパッチ許可で呼ぶこと)
マルチスレッドセーフである

引数

  uid     メッセージパイプのSceUID  

返り値

エラーのとき負の値を返します。
  値     結果  
  0     成功  
  <0     エラー  


解説

標準出力に対して出力されたデータを、引数uidで指定されたメッセージパイプに対してリダイレクトされるように設定します。標準出力からリダイレクトされたデータはsceKernelReceiveMsgPipe()関数によって取得できます。
メッセージパイプが満杯の状態になっていると、標準出力に対して出力を行おうとしたスレッドがブロックします。

関連項目

sceKernelRegisterStderrPipe(), sceKernelReceiveMsgPipe()
