SceBool
 [プリミティブ型]
ブール値

#include <psptypes.h>

解説

ブール値を表現します。psptypes.hでは

#define SCE_TRUE 1
というマクロが定義されています。SceBoolはこの真偽値のいずれかでなくてはなりません。
また、sizeof(SceBool)の値に依存してはいけません。

ScePspFColor
 [カラー型]
浮動小数点数カラー

形式

typedef struct ScePspFColor {
	float r, g, b, a;
} ScePspFColor __attribute__((aligned(16)));
解説

float 4要素のカラーベクトルです。
r, g, bの各要素の正規化レベルは、0.0のとき黒レベル、1.0のとき白レベルです。
a要素の正規化レベルは、0.0のとき透明、1.0のとき不透明です。

ScePspFQuaternion
 [構造体]
4元数・クォータニオン(浮動小数点数)

構造

#include <psptypes.h>
typedef struct ScePspFQuaternion {
	float x, y, z, w;
} ScePspFQuaternion;
解説

float 4要素の4元数、クォータニオンです。

x, y, zが軸を表わし、wで回転量を表わされます。単位クォータニオンqと、正規化された回転軸ベクトルaxisと回転量θの関係は、
q.x = sin(θ/2) * axis.x
q.y = sin(θ/2) * axis.y
q.z = sin(θ/2) * axis.z
q.w = cos(θ/2)
となります。

関連項目

sceVfpuQuaternionFromRotate()

#include <psptypes.h>
typedef struct ScePspDateTime {
	unsigned short year;
	unsigned short month;
	unsigned short day;
	unsigned short hour;
	unsigned short minute;
	unsigned short second;
	unsigned int microsecond;
} ScePspDateTime;
メンバ

  year     年を示します(1〜9999)  
  month     月を示します(1〜12)  
  day     月を示します(1〜31)  
  hour     時を示します(0〜23)  
  minute     分を示します(0〜59)  
  second     秒を示します(0〜59)  
  microsecond     マイクロ秒を示します(0〜999999)  

解説

時刻情報を統一的に扱うための構造体です。各種ライブラリによる時刻情報と変換を行う時などに使います。
