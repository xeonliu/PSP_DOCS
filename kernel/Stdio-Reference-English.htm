STDIO Manager Reference


sceKernelRegisterStdoutPipe
 [Redirection Operations]
Register message pipe to which standard output is to be redirected

Syntax

#include <kernel.h>
int sceKernelRegisterStdoutPipe (
	SceUID uid
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled, dispatch-enabled state)
Multithread safe

Arguments

  uid     SceUID of message pipe  

Return Value

If an error occurs, a negative value is returned.
  Value     Result  
  0     Success  
  <0     Error  


Description

This function causes data that was output to standard output to be redirected to the message pipe specified by the argument uid. Data that was redirected from standard output can be obtained by using the sceKernelReceiveMsgPipe() function.
If the message pipe becomes full, a thread that attempts to output to standard output will block.

See Also

sceKernelRegisterStderrPipe(), sceKernelReceiveMsgPipe()
