intrman(PSP®割り込みマネージャ)

sceKernelCpuSuspendIntr
 [割り込み制御]
割り込み可否状態保存と禁止

形式

#include <kernel.h>
int sceKernelCpuSuspendIntr(
	void
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフである

引数

なし

返り値

直前の割り込み可否状態

解説

すべての割り込みを禁止します。同時に禁止操作をする直前の可否状態を返します(状態の復帰にはsceKernelCpuResumeIntr()を使います)。
割り込み禁止は、CPU本体の割り込み禁止を行い、各割り込み要因個別の割り込み許可／禁止状態とは独立に行われます（すなわち、割り込みコントローラの要因別割り込みマスクレジスタは変更しません）。
sceKernelCpuResumeIntr()またはsceKernelCpuResumeIntrWithSync()と対にして使います。

sceKernelCpuResumeIntr
 [割り込み制御]
割り込み可否状態の復帰

形式

#include <kernel.h>
void sceKernelCpuResumeIntr(
	int oldstat
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフである

引数

  oldstat     sceKernelCpuSuspendIntrで取得した前の可否状態  

返り値

なし

解説

割り込み可否状態をoldstatに待避してある以前の状態に復帰します。
sceKernelCpuSuspendIntr()と対にして使います。

使用例

int oldstat;
oldstat = sceKernelCpuSuspendIntr();
/* 割り込み禁止期間 */
sceKernelCpuResumeIntr( oldstat );
sceKernelCpuResumeIntrWithSync
 [割り込み制御]
割り込み可否状態の復帰

形式

#include <kernel.h>
void sceKernelCpuResumeIntrWithSync(
	int oldstat
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフである

引数

  oldstat     sceKernelCpuSuspendIntrで取得した前の可否状態  

返り値

なし

解説

割り込み可否状態をoldstatに待避してある以前の状態に復帰します。
sceKernelCpuResumeIntr()との違いは、CPUの割り込み許可状況を変更する前にsync命令を実行して、確実にバス(デバイス等)への書き込みを終了させます。
sceKernelCpuSuspendIntr()と対にして使います。

使用例

int oldstat;
oldstat = sceKernelCpuSuspendIntr();
/* 割り込み禁止期間 */
/* デバイスの割り込みを禁止したり、デバイスの割り込みのクリアを
 * するような書き込み操作 */
sceKernelCpuResumeIntrWithSync( oldstat );
sceKernelIsCpuIntrSuspended
 [割り込み制御]
以前の割り込み可否状態の判定

形式

#include <kernel.h>
int sceKernelIsCpuIntrSuspended(
	int oldstat
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフである

引数

  oldstat     sceKernelCpuSuspendIntrで取得した前の可否状態  

返り値

割り込み許可状態であれば０、割り込み禁止状態であれば０以外の値を返します。

解説

sceKernelCpuSuspendIntr()で得られた以前の状態が割り込み禁止状態であったかどうかを判定します。
本サービスコールは、デバッグ時に確認のために用いることを意図しています。
sceKernelCpuSuspendIntr()で得られた、oldstatが割り込み禁止状態だったとわかったときにsceKernelCpuResumeIntr(oldstat)の呼び出しを省略するといったプログラミングは推奨できません。

sceKernelIsCpuIntrEnable
 [割り込み制御]
現在CPUが割り込み可能かの判定

形式

#include <kernel.h>
int sceKernelIsCpuIntrEnable(
	void
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフである

引数

なし

返り値

割り込み許可状態であれば0以外、割り込み禁止状態であれば0を返します。

解説

現在CPUが割り込み可能か否かを返します。
