SceGsoChunk
 [Structure]
Structure representing a chunk

Structure

#include <gso/gso_format.h>
typedef struct SceGsoChunk {
	unsigned short type ;
	unsigned short args_offs ;
	unsigned int next_offs ;
	unsigned int child_offs ;
	unsigned int data_offs ;
} SceGsoChunk;
Members

  type     Chunk type  
  args_offs     Offset to chunk arguments  
  next_offs     Offset to next chunk  
  child_offs     Offset to child chunk  
  data_offs     Offset to chunk data  

Description

This structure represents the data blocks of the respective layers which comprise the format.

Notes

If a chunk is a half-chunk then the child_offs and data_offs members are not included.
For details regarding the chunk structure, refer to the format documentation.
To access the elements of chunks, use the chunk manipulation macros.
