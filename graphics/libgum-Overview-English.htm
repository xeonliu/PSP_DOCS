libgum Overview

1 Library Overview

1. 1 Features

libgum is a library for supporting 3D operations. It has functions for performing matrix calculations such as rotation, scaling, and perspective projection as well as functions for reflecting the calculation result matrix in a display list that was created using libgu.
libgum performs matrix calculations for the current matrix that it maintains internally. If the current matrix is set to a projection matrix, view matrix, world matrix, or texture matrix, the calculation result matrix can be reflected appropriately in the display list. In addition, libgum has a matrix stack function for maintaining these four types of matrices separately, and calculation result matrices can be collectively reflected in the display list. A function is also provided for reflecting the current matrix to match primitive drawing.

2 Usage Procedure

2. 1 Basic Usage Procedure

(1) Initialize matrix stack

Although no specific initialization is necessary for libgum itself, a matrix stack must be initialized for normal use. Use sceGumSetMatrixStack() to assign a buffer and set the number of stack levels. The number of stack levels can be set separately for projection matrices, view matrices, world matrices, and texture matrices.

(2) Select the mode and perform matrix calculations

libgum matrix calculations are performed for the current matrix that the library maintains internally. When performing calculations, use sceGumMatrixMode() to specify whether the current matrix is a projection matrix, view matrix, world matrix, or texture matrix.
The following functions are provided for performing operations on the current matrix.

  Function     Operation  
  sceGumLoadMatrix     Load arbitrary matrix into current matrix  
  sceGumPopMatrix     Pop current matrix from matrix stack  
  sceGumLoadIdentity     Load unit matrix into current matrix  
  sceGumMultMatrix     Multiply arbitrary matrix by current matrix  
  sceGumRotateX     Multiply rotation matrix by current matrix  
  sceGumRotateY     Multiply rotation matrix by current matrix  
  sceGumRotateZ     Multiply rotation matrix by current matrix  
  sceGumRotateXYZ     Multiply rotation matrix by current matrix  
  sceGumRotateZYX     Multiply rotation matrix by current matrix  
  sceGumScale     Multiply scaling matrix by current matrix  
  sceGumTranslate     Multiply translation matrix by current matrix  
  sceGumLookAt     Multiply viewpoint matrix by current matrix  
  sceGumOrtho     Multiply orthogonal projection matrix by current matrix  
  sceGumPerspective     Multiply perspective projection matrix by current matrix  
  sceGumPushMatrix     Push current matrix on matrix stack  
  sceGumStoreMatrix     Save current matrix in an arbitrary area  


Although the operation result can be immediately reflected in the display list, it can also be saved on the matrix stack.

(3) Reflect result in libgu

The result of a matrix calculation that was performed with libgum can be reflected in a display list created with libgu.
sceGumSetCurMatrix() is provided as the function for reflecting the current matrix, and sceGumUpdateMatrix() is provided as the function for collectively reflecting updated matrices among the matrices in the matrix stack.
sceGumDrawArray(), sceGumDrawArrayN(), sceGumDrawBezier(), and sceGumDrawSpline() are also provided as functions for drawing primitives after reflecting updated matrices.
