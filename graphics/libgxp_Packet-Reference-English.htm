sceGxpFileSetHeapFunc
 [Function]
Set memory management functions

Syntax

void sceGxpFileSetHeapFunc(
	int heap,
	void *(*alloc_func)( size_t ),
	void (*free_func)( void * ) 
) ;
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Not multithread safe

Arguments

  Heap     Heap type       
       SCEGXP_HEAP_1     Heap 1  
       SCEGXP_HEAP_2     Heap 2  
       SCEGXP_HEAP_3     Heap 3  
  alloc_func     Memory allocation function       
  free_func     Memory free function       

Return Value

None

Description

This function sets the memory management functions to be used by the library.

Up to three sets of memory management functions can be set, and the particular set is selected at the time of use.
Normally, the three heaps are used for the following purposes.
  • Ordinary data
  • Data in DDR that is transferred to the GE
  • Data in eDRAM that is transferred to the GE

Memory allocated by memory management functions must be 64-byte aligned.
If this function is not used, memory is allocated and freed using memalign() and free().

Example

sceGxpFileSetHeapFunc( SCEGXP_HEAP_1, malloc, free ) ;
See Also

sceGxpFileCreateFromFile(), sceGxpFileDelete(),
sceGxpFileDuplicateSection(), sceGxpFileDeleteSection()
sceGxpFileSetBindFunc
 [Function]
Set object binding functions

Syntax

void sceGxpFileSetBindFunc(
	SceGxpObject *(*get_func)(
		int format, int type, const char *name 
	),
	void (*set_func)(
		nt format, int type, const char *name, SceGxpObject *object
	)
) ;
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Not multithread safe

Arguments

  get_func     Object acquisition function  
  set_func     Object registration function  

Return Value

None

Description

This function sets the object binding functions to be used by the library.

get_func is a callback that makes an inquiry for an object. The callback should return a pointer to an object that corresponds to the format, type, and name that are passed in the arguments. If the setup function finds an object that references an external object, it calls this callback and links that object with the object packet that is returned by the callback.

set_func is a callback that reports an object. If the setup function finds an object that can be referenced from an external object, it calls this callback and reports that object to the application.

If this function is not used, these callbacks are not called.

Example

sceGxpFileSetBindFunc( get_object, 0 ) ;
See Also

sceGxpFileCreateFromFile(), sceGxpFileSetup(), sceGxpFileBindObjects()

sceGxpFileDuplicateSection
 [Function]
Duplicate section

Syntax

bool sceGxpFileDuplicateSection
	SceGxpFile *file,
	int type,
	int heap
) ;
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Not multithread safe

Arguments

  file     File structure       
  type     Section type       
       SCEGXP_SECTION_OBJECT     Object section  
       SCEGXP_SECTION_PACKET     Packet section  
       SCEGXP_SECTION_VERTEX     Vertex section  
       SCEGXP_SECTION_PIXEL     Pixel section  
  heap     Heap type       
       SCEGXP_HEAP_1     Heap 1  
       SCEGXP_HEAP_2     Heap 2  
       SCEGXP_HEAP_3     Heap 3  

Return Value

Returns true if the section configuration has changed.

Description

This function uses the specified heap to duplicate the specified section.
The current section is deleted if it is a duplicated section.
Memory is allocated using the function that was set with sceGxpFileSetHeapFunc().
If the section configuration has changed, call sceGxpFileRelocate().

Example

SceGxpFile old = *file ;
sceGxpFileDuplicateSection( file, SCEGXP_SECTION_VERTEX, SCEGXP_HEAP_3 ) ;
sceGxpFileDuplicateSection( file, SCEGXP_SECTION_PIXEL, SCEGXP_HEAP_3 ) ;
sceGxpFileRelocate( file, &old ) ;
See Also

sceGxpFileDeleteSection(), sceGxpFileRelocate()
