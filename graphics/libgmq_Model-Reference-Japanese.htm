sceGmqModelSetMemoryManager / sceGmqModelSetMemoryManager2 / sceGmqModelSetMemoryManager3
 [関数]
メモリ管理関数の設定

形式

void sceGmqModelSetMemoryManager(
	void *(*malloc)( size_t ),
	void (*free)( void * ),
	size_t aligned_size,
	bool delay_release
) ;
void sceGmqModelSetMemoryManager2(
	void *(*malloc)( size_t ),
	void (*free)( void * ),
	size_t aligned_size,
	bool delay_release
) ;
void sceGmqModelSetMemoryManager3(
	void *(*malloc)( size_t ),
	void (*free)( void * ),
	size_t aligned_size,
	bool delay_release
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  malloc     メモリ確保関数  
  free     メモリ解放関数  
  aligned_size     アラインメント  
  delay_release     遅延解放フラグ  

返り値

なし

解説

モデルのためのメモリを確保する関数と解放する関数を設定します。

sceGmqModelSetMemoryManager()は通常メモリ管理関数を設定します。このメモリには構造体と配列データを格納します。省略時は通常のmallocとfreeが使用されます。
sceGmqModelSetMemoryManager2()はDMAメモリ管理関数を設定します。このメモリにはパケットと頂点データを格納します。省略時は通常メモリ管理関数が使用されます。
sceGmqModelSetMemoryManager3()はeDRAMメモリ管理関数を設定します。このメモリにはパケットと頂点データを格納します。省略時はeDRAMメモリを確保しません。

aligned_sizeにはメモリ管理関数が保証する最小のアラインメントを指定してください。これにより不要なパディングを抑制することができます。不明ならば1を指定してください。

delay_releaseには遅延解放を行うかどうかを指定してください。遅延解放を有効にすると、sceGmqModelCleanMemory()が呼び出されるまでメモリ解放が遅延されます。この機能はDMAメモリやeDRAMメモリの解放を安全なタイミングで行うために使用します。不要ならばfalseを指定してください。

使用例

sceGmqModelSetMemoryManager( malloc, free, 16, true ) ;

 [関数]
プリセットパケットのロード

形式

bool sceGmqModelLoadPreset(
	SceGmqModel *model,
	int n_frames
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）

引数

  model     モデル構造体  
  n_frames     フレーム数  

返り値

成功したときはtrue、失敗したときはfalseを返します。

解説

プリセットパケットをロードします。

モデルの現在の状態のパケットを作成して保持しておきます。作成されたパケットは描画時に自動的に使用され、パケット生成のCPU負荷を軽減することができます。
ただし、マトリクス以外のパラメータの変更は表示に反映されなくなります。

フレーム数を1として作成された場合は、パケットコピーで描画を行います。
その場合は、モデルの複製同士でプリセットパケットを共有できます。

フレーム数を2として作成された場合は、パケットコールで描画を行います。
その場合は、モデルの複製ごとにプリセットパケットの作成が必要となります。

プリセットパケットが自動的に使用されるのを抑制するには、プリセットパケットに関するオプションフラグを無効にしてください。

なお、プリセットパケットには、外部のプリミティブパケットやテクスチャパケットへのコールコマンドが含まれますが、それらのバッファは直接的には参照カウンタで保護されておりませんので、プリミティブやテクスチャおよびそれらのキャッシュが、プリセットパケットの使用中に解放されないようご注意ください。

また、パケットコールで描画する場合、モデル構造体そのものも確実に複製してください。
SCEGMQ_DUPLICATE_AUTOMATIC | SCEGMQ_DUPLICATE_MODELを指定します。
アニメーションを持たないモデルの複製には特に注意が必要です。

sceGmqModelLoadPreset( model, 1 ) ;
