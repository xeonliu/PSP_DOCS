SceGsoChunk
 [構造体]
チャンクを表す構造体

構造

#include <gso/gso_format.h>
typedef struct SceGsoChunk {
	unsigned short type ;
	unsigned short args_offs ;
	unsigned int next_offs ;
	unsigned int child_offs ;
	unsigned int data_offs ;
} SceGsoChunk;
メンバ

  type     チャンクタイプ  
  args_offs     チャンク引数へのオフセット  
  next_offs     次のチャンクへのオフセット  
  child_offs     子チャンクへのオフセット  
  data_offs     チャンクデータへのオフセット  

解説

フォーマットを構成する各階層のデータブロックを表す構造体です。

備考

チャンクがハーフチャンクならば child_offs, data_offs メンバは含まれません。
チャンク構造の詳細についてはフォーマットドキュメントを参照してください。
チャンクの要素にアクセスするにはチャンク操作マクロを使用してください。

sceGsoSceneCheckFileHeader
 [関数]
ファイルヘッダの確認

形式

bool sceGsoSceneCheckFileHeader( 
	const void *buf,
	int size 
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  buf     ファイルイメージのアドレス  
  size     ファイルイメージのサイズ（不明ならば0）  

返り値

成功したときはtrue、失敗したときはfalseを返します。

解説

シーンファイルのヘッダを確認するための関数です。
bufとsizeで指定された領域に適切なヘッダが存在すればtrue、存在しなければfalseを返します。

使用例

if ( sceGsoSceneCheckFileHeader( buf, size ) == false ) return ;

sceGsoSceneGetChunk
 [関数]
シーンチャンクの取得

形式

	const void *buf,
	int size,
	int index 
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  buf     ファイルイメージのアドレス  
  size     ファイルイメージのサイズ（不明ならば0）  
  index     ファイル内でのシーン番号  

返り値

シーンチャンクへのポインタを返します。

解説

シーンファイルに含まれるシーンチャンクを取得するための関数です。
bufとsizeで指定された領域にあるindex番目のシーンチャンクへのポインタを返します。
該当するシーンチャンクが存在しないときは 0 を返します。

使用例

SceGsoChunk *chunk = sceGsoSceneGetChunk( buf, size, 0 ) ;

	const char *str,
