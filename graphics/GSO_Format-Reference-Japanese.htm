FCurve
 [ブロック]
関数データ

形式

FCurve "name" format n_dims n_keys {
	body
}
解説

モーションにおける一つの関数データを表すサブブロックです。

サブブロック

なし

コマンド

なし

引数

 補間形式   enum   キーの内挿方法（以下の定数が使用可能） 
     CONSTANT：補間なし 
     LINEAR：線形補間 
     HERMITE：HERMITE補間 
     CUBIC：CUBIC補間 
     SPHERICAL：球面線形補間（クォータニオン用） 
     キーの外挿方法（以下の定数が使用可能） 
     HOLD：時刻を固定する 
     CYCLE：時刻を循環させる 
     SHUTTLE：時刻を往復させる 
     REPEAT：繰り返し（値を累積） 
     EXTEND：線形拡張 
 次元数   int   キーフレームの次元数です。フレーム値は含みません。 
 キー数   int   キーフレームの数です。 

データ形式

関数は複数のキーフレームの補間によって表現されます。
データは（キーフレーム×キー数）で構成されます。
キーフレームは、フレームと関数値で構成されます。
関数値はベクトルであり、次元数だけデータを持ちます。
キーフレームはフレームの小さい値から大きい値に（昇順に）並んでいる必要があります。

例：
  フレーム　関数値　関数値　関数値 ・・・    // キーフレーム  
  フレーム　関数値　関数値　関数値 ・・・    // キーフレーム  
  フレーム　関数値　関数値　関数値 ・・・    // キーフレーム  
  ・・・  


ただしHERMITE補間およびCUBIC補間の場合はハンドルを含むため、おのおの（次元数×3）、（次元数×5）だけデータを持ちます。

例：HERMITE 補間
  フレーム　関数値　in_dy　out_dy　関数値　in_dy　out_dy ・・・  
  フレーム　関数値　in_dy　out_dy　関数値　in_dy　out_dy ・・・  
  フレーム　関数値　in_dy　out_dy　関数値　in_dy　out_dy ・・・  
  ・・・  

例：CUBIC 補間
  フレーム　関数値　in_dx　in_dy　out_dx　out_dy　関数値 ・・・  
  フレーム　関数値　in_dx　in_dy　out_dx　out_dy　関数値 ・・・  
  フレーム　関数値　in_dx　in_dy　out_dx　out_dy　関数値 ・・・  
  ・・・  


in_dx, in_dyは左ハンドルを、out_dx, out_dyは右ハンドルを表します。

BlindBlock
 [ブロック]

	body

コマンド

DefineEnum
 [定義コマンド]

引数

DefineCommand
 [定義コマンド]
コマンド定義

形式

DefineCommand type_id " type_name " arg_type ・・・
引数

 type_id   int   コマンドのタイプID 
 type_name   string   コマンドのタイプ名 
 arg_type   enum   引数の型（以下の定数が使用可能） 
     char：符号つき8ビット整数 
     short：符号つき16ビット整数 
     int：符号つき32ビット整数 
     u_char：符号なし8ビット整数 
     u_short：符号なし16ビット整数 
     u_int：符号なし32ビット整数 
     float：32ビット浮動小数点数 
     string：文字列 
     ブロックタイプ名：ブロック参照 
     列挙定数タイプ名：列挙定数 
     void：上記の任意の型 
     ...：可変長引数 

解説

コマンドを定義します。
タイプID、タイプ名は重複しないように注意してください。
タイプIDは0x1000〜0x3FFFの値が使用可能です。
タイプ名には適当なプレフィクスをつけることを推奨します。

引数タイプが void の場合、引数の型は記述形式から推測されます。
その場合、ブロック参照、列挙定数は、"ブロックタイプ名::ブロック名"、
"列挙定数タイプ名::シンボル名" のように、型限定子を明示的に指定してください。

使用例

例1:

 
DefineCommand 0x1020 "MyCommand" float float float

MyCommand 1.0 2.0 3.0

例2:

 
DefineEnum "DEBUG_MODE" "OFF" 0
DefineEnum "DEBUG_MODE "ON" 1
DefineCommand 0x1020 "MyCommand" Node DEBUG_MODE

MyCommand "node-0" OFF

例3:

 
DefineEnum "DEBUG_MODE" "OFF" 0
DefineEnum "DEBUG_MODE "ON" 1
DefineCommand 0x1020 "MyCommand" void ...

MyCommand "Node::node-0" DEBUG_MODE::OFF

FogRange
 [フォグコマンド]
フォグ範囲

形式

FogRange min max
引数

  min     float     フォグの開始距離  
  max     float     フォグの終了距離  

解説

フォグの範囲を指定します。フォグタイプがLINEARのときに使用されます。
省略時の値は表示プログラムにより異なります。

フォグ混合係数fは以下のように計算されます（カメラからの距離をDとします）。

f = ( D - min ) / ( max - min )

（ただし D < min のときは 0.0、D > max のときは 1.0 ）
FogDensity
 [フォグコマンド]
フォグ密度

形式

FogDensity density
引数

  density     float     フォグの密度  

解説

フォグの密度を指定します。フォグタイプがEXP EXP2のときに使用されます。
省略時の値は表示プログラムにより異なります。

フォグ混合係数fは以下のように計算されます（カメラからの距離をDとします）。

f = 1 - exp( - D * density )			（タイプが EXP のとき）
f = 1 - exp( - D * D * density * density )	（タイプが EXP2 のとき）

省略時は 0.0 0.0 0.0 1.0 となります。

LightAtten
 [ライトコマンド]
ライト減衰係数

形式

LightAtten atten0 atten1 atten2
引数

  atten0     float     ０次減衰係数  
  atten1     float     １次減衰係数  
  atten2     float     ２次減衰係数  

解説

ライトの減衰係数を指定します。ライトタイプがPOINT SPOTのときに使用されます。
省略時の値は表示プログラムにより異なります。

ライト強度fは以下のように計算されます（ライトからの距離をDとします）。

f = 1 / ( atten0 + atten1 * D + atten2 * D * D )

LightCone / LightCone2
 [ライトコマンド]
ライト円錐係数

形式

LightCone falloff field
LightCone2 falloff field beam
引数

  falloff     float     減衰係数  
  field     float     明るさ最小となる頂角（単位＝ラジアン）  
  beam     float     明るさ最大となる円錐の頂角（単位＝ラジアン）  

解説

ライトの円錐係数を指定します。ライトタイプがSPOTのときに使用されます。
省略時の値は表示プログラムにより異なります。

ライト強度fはおのおの以下のように計算されます（前方軸との角度をRとします）。

f = pow( cos( R ), falloff )
f = pow( ( cos( R ) - cos( field / 2 ) ) / ( cos( beam / 2 ) - cos( field / 2 ) ), falloff )

（ただし R > field / 2 のときは 0.0 ）

 [ノードコマンド]

形式

Scale2 scale_x scale_y scale_z
Scale3 scale_x scale_y scale_z
引数

  scale     float     スケーリング値  

解説

階層座標計算のための、スケーリングを指定します。
Scale はスケーリング値が行列スタックに直接影響します。
Scale2 はスケーリング値が行列スタックに直接影響せず、スケーリング値を独立に積算した結果が最終的に影響します。

Scale、Scale2、Scale3 はいずれか一つだけ指定できます。
すべて省略した場合はスケーリング計算を行いません。

Animate
 [モーションコマンド]
アニメーション

形式

Animate "block" command mode "fcurve"
引数

  block     ref     アニメーションするブロックへの参照  
  command     enum     アニメーションするコマンドのタイプ  
  mode     enum     アニメーション制御モード  
  fcurve     ref     関数データへの参照  

解説

他のブロックのコマンドを、指定した関数データでアニメーションします。

block, commandはアニメーション対象となるブロックとコマンドを指定します。アニメーション可能なブロックとコマンドの組み合わせは表示プログラムにより異なります。

modeは、アニメーションのサブブロック番号およびサブチャンネル番号を指定します。
サブブロックはINDEXn、サブチャンネルはOFFSETnで表します（n=0〜255)。
これらの値を論理和で指定することができます。通常は0にしてください。

fcurveは同じモーション内の関数データを指定します。
