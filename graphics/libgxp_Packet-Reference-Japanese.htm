sceGxpFileSetHeapFunc
 [関数]
メモリ管理関数の設定

形式

void sceGxpFileSetHeapFunc(
	int heap,
	void *(*alloc_func)( size_t ),
	void (*free_func)( void * ) 
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  heap     ヒープタイプ       
       SCEGXP_HEAP_1     ヒープ１  
       SCEGXP_HEAP_2     ヒープ２  
       SCEGXP_HEAP_3     ヒープ３  
  alloc_func     メモリ確保関数       
  free_func     メモリ解放関数       

返り値

なし

解説

ライブラリで使用するメモリ管理関数を設定します。

メモリ管理関数は３組まで設定することができ、使用時に選択することができます。
通常、各ヒープは以下のような用途で使用されます。
  • 一般的なデータ
  • GEに転送されるDDR上のデータ
  • GEに転送されるeDRAM上のデータ

メモリ管理関数によって確保されるメモリは64バイト単位のアラインメントが必要です。
省略時はmemalign(), free()によりメモリが確保、解放されます。

使用例

sceGxpFileSetHeapFunc( SCEGXP_HEAP_1, malloc, free ) ;
関連項目

sceGxpFileCreateFromFile(), sceGxpFileDelete()
sceGxpFileDuplicateSection(), sceGxpFileDeleteSection()
sceGxpFileSetBindFunc
 [関数]
オブジェクト結合関数の設定

形式

void sceGxpFileSetBindFunc(
	SceGxpObject *(*get_func)(
		int format, int type, const char *name 
	),
	void (*set_func)(
		nt format, int type, const char *name, SceGxpObject *object
	)
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  get_func     オブジェクト取得関数  
  set_func     オブジェクト登録関数  

返り値

なし

解説

ライブラリで使用するオブジェクト結合関数を設定します。

get_funcオブジェクトを問い合わせるコールバックです。引数に渡されたフォーマット、タイプ、名前に該当するオブジェクトのポインタを返してください。セットアップ関数は外部のオブジェクトを参照しているオブジェクトを見つけるとこのコールバックを呼び出し、そのオブジェクトとコールバックが返すオブジェクトのパケットを結合します。

set_funcはオブジェクトを通知するコールバックです。セットアップ関数は外部のオブジェクトから参照可能なオブジェクトを見つけるとこのコールバックを呼び出し、アプリケーションにそのオブジェクトを通知します。

省略時はこれらのコールバックは呼び出されません。

使用例

sceGxpFileSetBindFunc( get_object, 0 ) ;
関連項目

sceGxpFileCreateFromFile(), sceGxpFileSetup(), sceGxpFileBindObjects()


sceGxpFileCreateFromFile
 [関数]
ファイル構造体の生成

形式

SceGxpFile *sceGxpFileCreateFromFile(
	const char *filename 
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  filename     ファイル名  

返り値

ファイル構造体（失敗したときは0）

解説

指定されたファイルをロードし、ファイル構造体を生成します。
メモリはsceGxpFileSetHeapFunc()で設定された関数により確保されます。
ファイル構造体の使用を終了する時はsceGxpFileDelete()を呼び出してください。
この関数は内部でsceGxpFileSetup()を呼び出します。

使用例

SceGxpFile *file = sceGxpFileCreateFromFile ( "model.gmp" ) ;
関連項目

sceGxpFileDelete(), sceGxpFileSetHeapFunc()

sceGxpFileDuplicateSection
 [関数]
セクションの複製

形式

bool sceGxpFileDuplicateSection
	SceGxpFile *file,
	int type,
	int heap
) ;
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  file     ファイル構造体       
  type     セクションタイプ       
       SCEGXP_SECTION_OBJECT     オブジェクトセクション  
       SCEGXP_SECTION_PACKET     パケットセクション  
       SCEGXP_SECTION_VERTEX     頂点セクション  
       SCEGXP_SECTION_PIXEL     ピクセルセクション  
  heap     ヒープタイプ       
       SCEGXP_HEAP_1     ヒープ１  
       SCEGXP_HEAP_2     ヒープ２  
       SCEGXP_HEAP_3     ヒープ３  

返り値

セクション構成が変化したらtrue

解説

指定されたセクションを、指定されたヒープを使用して複製します。
現在のセクションはそれが複製されたものである場合は削除されます。
メモリはsceGxpFileSetHeapFunc()で設定された関数により確保されます。
セクション構成が変化した場合は sceGxpFileRelocate() を実行してください。

使用例

SceGxpFile old = *file ;
sceGxpFileDuplicateSection( file, SCEGXP_SECTION_VERTEX, SCEGXP_HEAP_3 ) ;
sceGxpFileDuplicateSection( file, SCEGXP_SECTION_PIXEL, SCEGXP_HEAP_3 ) ;
sceGxpFileRelocate( file, &old ) ;
関連項目

sceGxpFileDeleteSection(), sceGxpFileRelocate()

  file     ファイル構造体       
  type     セクションタイプ

       SCEGXP_SECTION_PACKET     パケットセクション  
       SCEGXP_SECTION_VERTEX     頂点セクション

) ;
