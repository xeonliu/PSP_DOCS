解説

ピクチャのためのメモリを確保する関数と解放する関数を設定します。

sceGiqPictureSetMemoryManager()は通常メモリ管理関数を設定します。このメモリには構造体と配列データを格納します。省略時は通常のmallocとfreeが使用されます。
sceGiqPictureSetMemoryManager2()はDMAメモリ管理関数を設定します。このメモリにはパケットとピクセルデータを格納します。省略時は通常メモリ管理関数が使用されます。
sceGiqPictureSetMemoryManager3()はeDRAMメモリ管理関数を設定します。このメモリにはパケットとピクセルデータを格納します。省略時はeDRAMメモリを確保しません。

aligned_sizeにはメモリ管理関数が保証する最小のアラインメントを指定してください。これにより不要なパディングを抑制することができます。不明ならば1を指定してください。

delay_releaseには遅延解放を行うかどうかを指定してください。遅延解放を有効にすると、sceGiqPictureCleanMemory()が呼び出されるまでメモリ解放が遅延されます。この機能はDMAメモリやeDRAMメモリの解放を安全なタイミングで行うために使用します。
不要ならばfalseを指定してください。

使用例

sceGiqPictureSetMemoryManager( malloc, free, 16, true ) ;

  SCEGIQ_DUPLICATE_AUTOMATIC     時間情報を含むならば複製  
  （タイプ別フラグ）

ピクチャ構造体を複製するための関数です。

		SCEGIQ_DUPLICATE_AUTOMATIC ) ;
備考

強制的に複製するにはタイプ別フラグを指定してください。

  フラグ     GEコマンド

  SCEGIQ_ENABLE_CROP     TU TV SU SV  
  SCEGIQ_ENABLE_BLEND     BLEND FIXA FIXB  
  SCEGIQ_ENABLE_FUNC     TFUNC  
  SCEGIQ_ENABLE_FILTER     TFILTER  
  SCEGIQ_ENABLE_WRAP     TWRAP

現在のディゾルブ値を返します。返される値の範囲は0.0f〜1.0fです。
解説

ディゾルブ値、つまり2つのフレームを重ね合わせるディゾルブ処理において補間比率を表す値を取得するための関数です。
sequenceで指定したシーケンスについて、現在のディゾルブ値を取得して返します。
使用例

float dissolve = sceGiqSequenceGetDissolve( sequence ) ;
sceGuTexLevelMode( SCEGU_LOD_CONSTANT, dissolve ) ;
関連項目

sceGiqSequenceSetDissolve()
