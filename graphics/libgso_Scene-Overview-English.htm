libgso_scene Overview

1 Library Overview

1. 1 Functional Characteristics

libgso is a library which handles 3D scene data. libgso provides structure definitions and macro definitions for the 3D scene format. Using these, information on lights, cameras, etc. declared in the 3D scene format can be obtained.

1. 2 Related Files

The following files are required in order to use libgso.
