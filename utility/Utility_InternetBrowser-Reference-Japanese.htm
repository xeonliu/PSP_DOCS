SceUtilityHtmlViewerAutoDisconnectMode
 [定数定義]
終了時のネットワーク接続の自動切断指定

定義

#include <utility_htmlviewer.h>
typedef enum {
	SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_ENABLE 	= 0,
	SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_DISABLE	= 1,
	SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_CONFIRM	= 2
} SceUtilityHtmlViewerAutoDisconnectMode;
解説

SceUtilityHtmlViewerParam構造体のauto_disconnectに設定する値です。インターネットブラウザユーティリティ終了時、ネットワーク接続を自動切断するかどうかを表しています。

  設定値     意味  
  SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_ENABLE     自動切断する  
  SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_DISABLE     自動切断しない  
  SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_CONFIRM     切断するか確認する

#include <utility_htmlviewer.h>

ret = sceUtilityHtmlViewerUpdate(1);
