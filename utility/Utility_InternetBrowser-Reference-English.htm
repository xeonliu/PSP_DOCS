sceUtilityHtmlViewerInitStart
 [Function]
Start Internet browser utility initialization

Syntax

#include <utility_htmlviewer.h>
int sceUtilityHtmlViewerInitStart(
	SceUtilityHtmlViewerParam *param
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in interrupt-enabled state)
Not multithread safe

Arguments

  param     Initialization parameters  

Return Value

If the function completes normally, 0 is returned.
If an error occurs, an error code from SCE_UTILITY_COMMON_ERROR_xxx and SCE_UTILITY_HTMLVIEWER_ERROR__xxx is returned.

Description

This function starts the initialization processing of the Internet browser utility.
This function must be called first before the Internet browser utility can be used.
If the processing of this function is successful, the Internet browser utility
transitions to SCE_UTILITY_COMMON_STATUS_INITIALIZE and then to SCE_UTILITY_COMMON_STATUS_RUNNING.

Example

SceUtilityHtmlViewerParam param;

memset(&param, 0, sizeof(SceUtilityHtmlViewerParam));
param.base.size = sizeof(SceUtilityHtmlViewerParam);

param.heap_size = 10 * 1024 * 1024;
param.heap_addr = malloc(param.heap_size);
param.initial_url = "http://www.yahoo.co.jp/";
param.num_tab = 1;
param.ui_level = SCE_UTILITY_HTMLVIEWER_UI_LEVEL_LIMITED;
param.flag = SCE_UTILITY_HTMLVIEWER_DISABLE_START_CTRL;
param.temp_config.cookie_mode = SCE_UTILITY_HTMLVIEWER_COOKIE_ACCEPT;
param.auto_disconnect = SCE_UTILITY_HTMLVIEWER_AUTO_DISCONNECT_DISABLE;

ret = sceUtilityHtmlViewerInitStart(&htmlviewer_param);
Notes

The Internet browser utility may call other utilities internally. Therefore, when calling the Internet browser utility, do not start up other utilities.
