SceUtilityParamBase
 [構造体]
ユーティリティの初期化パラメータに共通の構造体

構造

#include <utility_common.h>
typedef struct {
	unsigned int size;
	int message_lang;
	int ctrl_assign;
	int main_thread_priority;
	int sub_thread_priority;
	int font_thread_priority;
	int sound_thread_priority;
	int result;
	int reserved1;
	int reserved2;
	int reserved3;
	int reserved4;
} SceUtilityParamBase;
メンバ

  size     SceUtility***Param構造体のサイズ。  
  message_lang     表示言語。  
  ctrl_assign     コントローラのボタンアサイン。  
  main_thread_priority     ユーティリティのグラフィックススレッドのプライオリティ。  
  sub_thread_priority     ファイルアクセスやネットワークアクセスを行う  
  スレッドのプライオリティ。  
  font_thread_priority     ユーティリティのフォント読み込みスレッドのプライオリティ。  
  sound_thread_priority     ユーティリティの効果音再生スレッドのプライオリティ。  
  result     ユーティリティの返り値。  
  成功/キャンセル/エラー値が返ります。  
  エラー値は共通エラーコードではなく、各サービス  
  （セーブデータ、文字列入力、ネットワーク設定等）の  
  エラー値が戻ります。  
  reserved1     予約（0を設定すること）  
  reserved2     予約（0を設定すること）  
  reserved3     予約（0を設定すること）  
  reserved4     予約（0を設定すること）  

解説

各種ユーティリティ初期化関数sceUtilitySavedataInitStart()／sceUtilityOskInitStart()／sceUtilityNetconfInitStart()／sceUtilityMsgDialogInitStart()に渡すパラメータ構造体SceUtilitySavedataParam／SceUtilityOskParam／SceUtilityNetconfParam／SceUtilityMsgDialogParamの共通属性を定義する構造体です。

resultには次のいずれかの値、あるいは各ユーティリティが定義した値がストアされます。

  マクロ     値     説明  
  SCE_UTILITY_COMMON_RESULT_OK  
    		   0     ユーティリティの処理は  
  成功した  
  SCE_UTILITY_COMMON_RESULT_CANCELED     1     ユーティリティの処理は  
  ユーザーによって  
  キャンセルされた  
  SCE_UTILITY_COMMON_RESULT_ABORTED     2     ユーティリティの処理は  
  強制的に終了された  


関連項目

sceUtilityMsgDialogInitStart()、
SceUtilitySavedataParam、SceUtilityOskParam、SceUtilityNetconfParam、
SceUtilityMsgDialogParam

#include <utility_auto_connect.h>

解説

自動接続ユーティリティ内部の、イベント処理、アニメーション処理、描画処理、といった毎フレームの内部処理を行います。

この関数を呼び出す前に、ゲームプログラム側の描画パケットの転送の終了を待つようにしてください。

この関数は、sceUtilityAutoConnectGetStatus()で取得されるセーブデータユーティリティの状態がSCE_UTILITY_COMMON_STATUS_RUNNING、もしくはSCE_UTILITY_COMMON_STATUS_WAITINGになった状態で呼び出してください。

ret = sceUtilityAutoConnectUpdate (1);
