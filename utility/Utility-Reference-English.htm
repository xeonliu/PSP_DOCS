SceUtilityParamBase
 [Structure]
Common structure for utility initialization parameters

Structure

#include <utility_common.h>
typedef struct {
	unsigned int size;
	int message_lang;
	int ctrl_assign;
	int main_thread_priority;
	int sub_thread_priority;
	int font_thread_priority;
	int sound_thread_priority;
	int result; 
	int reserved1;
	int reserved2;
	int reserved3;
	int reserved4;
} SceUtilityParamBase;
Members

  size     Size of SceUtility***Param structure   
  message_lang     Display language  
  ctrl_assign     Controller button assignment  
  main_thread_priority     Priority of the utility's graphics thread.  
  sub_thread_priority     Priority of thread for performing file access or network access.  
  font_thread_priority     Priority of the utility's font reading thread   
  sound_thread_priority     Priority of the utility's sound effect playback thread   
  result     Return value of the utility.  
  A value indicating success, cancellation, or error is returned.  
  The error value is not the same as the common error code.  
  An individual error value is returned for each service (such as  
  saved data, character string input, and network configuration).  
  reserved1     Reserved (must be set to 0)  
  reserved2     Reserved (must be set to 0)  
  reserved3     Reserved (must be set to 0)  
  reserved4     Reserved (must be set to 0)  

Description

This structure defines common attributes of the parameter structures SceUtilitySavedataParam/SceUtilityOskParam/SceUtilityNetconfParam/
SceUtilityMsgDialogParam which are passed to the various utility initialization functions sceUtilitySavedataInitStart()/sceUtilityOskInitStart()/sceUtilityNetconfInitStart()/
sceUtilityMsgDialogInitStart().

One of the following values or a value that was defined by the relevant utility is stored in result.

  Macro     Value     Description  
  SCE_UTILITY_COMMON_RESULT_OK  
    		   0     Utility processing was   
  successful  
  SCE_UTILITY_COMMON_RESULT_CANCELED     1     Utility processing was   
  canceled by the user  
  SCE_UTILITY_COMMON_RESULT_ABORTED     2     Utility processing was   
  forcibly terminated

	0x529b, 0x30d5, 0x30a3, 0x30fc, 0x30eb, 0x30c9, 0x0000};

	0x3075, 0x305f, 0x3064, 0x3081, 0x306e, 0x5165,
	0x529b, 0x30d5, 0x30a3, 0x30fc, 0x30eb, 0x30c9, 0x0000};

param.base.size = sizeof (SceUtilityOskParam);

param.input_field_info = inputField;

memset (inputField, 0x00, sizeof (inputField));

inputField [0].input_method_type
	= SCE_UTILITY_OSK_INPUT_METHOD_JAPANESE_BUNSETSUHENKAN;
inputField [0].input_method_attributes = 0;

	= SCE_UTILITY_OSK_WRITING_LANGUAGE_TYPE_DEFAULT;
inputField [0].input_character_type
	= SCE_UTILITY_OSK_INPUT_CHARACTER_TYPE_ALL_TYPE;
inputField [0].message = messageA;
inputField [0].init_text = 0x0000; // When no initial string is assigned
inputField [0].result_text_buffer_size = sizeof (resultBufferA) / sizeof (SceWChar16);
inputField [0].result_text_buffer = resultBufferA;
inputField [0].num_lines = 4;

inputField [1].input_method_type
	= SCE_UTILITY_OSK_INPUT_METHOD_JAPANESE_BUNSETSUHENKAN;
inputField [1].input_method_attributes = 0;
inputField [1].writing_language_type
	= SCE_UTILITY_OSK_WRITING_LANGUAGE_TYPE_DEFAULT;
inputField [1].input_character_type
	= SCE_UTILITY_OSK_INPUT_CHARACTER_TYPE_ALL_TYPE;
inputField [1].message = messageB;
inputField [1].init_text = 0x0000; ; // When no initial string is assigned
inputField [1].result_text_buffer_size = sizeof (resultBufferB) / sizeof (SceWChar16);
inputField [1].result_text_buffer = resultBufferB;
inputField [1].num_lines = 4;

If the first character is a null character then the connection is made without setting a group name.
timeout specifies the timeout, in seconds, for an ad hoc connection (join). The start-up type (type) is only referenced in the case of SCE_UTILITY_NETCONF_TYPE_JOIN_ADHOC. For an IBSS scan, which is performed before the processing for joining, if the target IBSS is not found, the scan will be repeated for timeout seconds. If timeout is not specified or is set to 0, the scan is not performed and an error is returned.
