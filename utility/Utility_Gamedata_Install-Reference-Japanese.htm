メンバ変数abortedStatusについて
ゲームデータインストールユーティリティが強制終了され、SceUtilityParamBaseのメンバ変数resultにSCE_UTILITY_COMMON_RESULT_ABORTEDが返った場合、強制終了直前のゲームデータインストールユーティリティのステータスがabortedStatusに返されます。これにより、強制終了ステータスに実際のステータスが書き潰されてしまうことがありません。
たとえば、強制終了（サスペンド）されると、resultにはSCE_UTILITY_COMMON_RESULT_ABORTEDが返ります。ただこれだけだと、実際にインストールが完了したかどうかわかりません。この時abortedStatusに、SCE_UTILITY_COMMON_RESULT_OKが返っている場合は、インストール後に強制終了されていますので、インストール作業は完了しています。
abortedStatusがSCE_UTILITY_COMMON_RESULT_CANCELEDの場合は、インストール完了前に強制終了したことになります。
resultがSCE_UTILITY_COMMON_RESULT_ABORTED以外の場合には、abortedStatusへステータスは出力されません。

sceUtilityGamedataInstallInitStart
 [関数]
ゲームデータインストールユーティリティの初期化の開始

形式

#include <utility_gamedata_install.h>
int sceUtilityGamedataInstallInitStart(
	SceUtilityGamedataInstallParam *param
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフでない

引数

  param     ゲームデータインストールユーティリティ起動パラメータ  

返り値

正常終了の場合、0を返します。
エラーがあった場合、SCE_UTILITY_COMMON_ERROR_xxxを返します。

解説

ゲームデータインストールユーティリティの初期化処理を開始します。
ゲームデータインストールユーティリティを使用する場合には、まずこの関数を呼び出す必要があります。
この関数が成功した場合、ゲームデータインストールユーティリティはSCE_UTILITY_COMMON_STATUS_INITIALIZE状態になり、その後SCE_UTILITY_COMMON_STATUS_RUNNING状態に遷移します。

使用例

SceUtilityGamedataInstallParam param;

memset(&param, 0x00, sizeof(SceUtilityGamedataInstallParam));
param.size = sizeof(SceUtilityGamedataInstallParam);

≪SceUtilityGamedataInstallParamへ必要な値を設定≫
≪詳しくはサンプルを参照してください≫

ret = sceUtilityGamedataInstallInitStart(&param);
備考

ゲームデータインストールユーティリティは、他のユーティリティと共通のメモリ領域を使用するため、すでに他のユーティリティが動作中である場合は、この関数の呼び出しは失敗します。

ゲームデータインストールユーティリティは、インストールされたデータがMemory Stick™に存在する場合、そのデータを削除してからインストール作業を開始します。
ゲームがMemory Stick™上のインストールされたファイルをオープンしている場合、ゲームデータインストールユーティリティを使用する前にクローズする必要があります。

sceUtilityGamedataInstallUpdate
 [関数]
ゲームデータインストールユーティリティの毎フレーム処理

形式

#include <utility_gamedata_install.h>
int sceUtilityGamedataInstallUpdate(
	int animation_speed
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み禁止/許可状態に依存しない）
マルチスレッドセーフでない

引数

  animation_speed     ゲームデータインストールユーティリティのアニメーション速度  
  秒間60回の場合1  

返り値

正常終了の場合、0を返します。
エラーがあった場合、SCE_UTILITY_COMMON_ERROR_xxxを返します。

解説

ゲームデータインストールユーティリティ内部の、イベント処理、アニメーション処理、描画処理、といった、毎フレーム処理を行います。
引数animation_speedには、アニメーションの速度を設定します。たとえばこの関数を秒間60回呼び出す場合には1、秒間30回呼び出す場合には2を設定します。

この関数を呼び出すと、ユーティリティ内部で描画コンテキストを保存し、ユーティリティの描画を行い、そして描画コンテキストを復帰させます。
この関数を呼び出す前に、ゲームプログラム側の描画パケットの転送の終了を待つようにしてください。

この関数は、sceUtilityGamedataInstallGetStatus()で取得されるゲームデータインストールユーティリティの状態がSCE_UTILITY_COMMON_STATUS_RUNNINGになった状態で呼び出してください。

使用例

ret = sceUtilityGamedataInstallUpdate(1);

SCE_UTILITY_COMMON_RESULT_xxx
 [実行結果]
ユーティリティ実行結果マクロ

定義

#include <utility_common.h>
#define SCE_UTILITY_COMMON_RESULT_OK		(0)
#define SCE_UTILITY_COMMON_RESULT_CANCELED	(1)
#define SCE_UTILITY_COMMON_RESULT_ABORTED	(2)
解説

SceUtilityParamBaseのメンバ変数resultに返るユーティリティサービスの実行結果ですが、ゲームデータインストールユーティリティでは特殊な返り方をする場合があります。

キャンセル終了SCE_UTILITY_COMMON_RESULT_CANCELED
ユーザ操作、もしくは、sceUtilityGamedataInstallAbort()を呼んでインストールがキャンセルされた場合、ゲームデータインストールユーティリティはresultにSCE_UTILITY_COMMON_RESULT_CANCELEDを返します。
その他の場合として、resultにSCE_UTILITY_COMMON_RESULT_ABORTEDが返った場合、abortedStatusにSCE_UTILITY_COMMON_RESULT_CANCELEDが返ることがあります。
詳細はSceUtilityGamedataInstallParamの「メンバ変数abortedStatusについて」を参照してください。

解説

ゲームデータインストールユーティリティの実行結果は、SceUtilityParamBaseのメンバ変数resultに返されます。
sceUtilityGamedataInstallGetStatus()で取得されるゲームデータインストールユーティリティの状態がSCE_UTILITY_COMMON_STATUS_NONE状態となった後で、resultを参照してください。
エラーの場合に返される値は以下のとおりです。
※さらに、実行結果についてはSceUtilityGamedataInstallParamの解説にありますabortedStatusも合わせて参照ください。

不正パラメータのエラー詳細
SCE_UTILITY_GAMEDATA_INSTALL_ERROR_PARAMは、ユーティリティへ渡すパラメータが不正だった場合に返ります。
このときユーティリティは、どのパラメータが不正だったかを番号で報告します。
開発ホスト上のターミナルコンソール（dspconsで起動）に以下のように出力されます。
GamedataInstall ParamError Info(xxx).
xxxはエラー番号となっています。

エラー番号

  番号     不正なパラメータ  
  1     titleIdに不正な文字列が入っている、もしくは文字列が設定されていない  
  2     userIdに不正な文字列が入っている  
  3     titleがNULL TERMINATEされていない、もしくは設定されていない  
  4     savedataTitleがNULL TERMINATEされていない  
  5     detailがNULL TERMINATEされていない  
  6     parentalLevが範囲外  
  7     INSDIRディレクトリ以下にPARAM.SFOが存在する  
  8     ゲームデータインストールユーティリティでサポートしているファイル上限値を  
  超えている  
  9     reserved? 変数が0クリアされていない  
  10     INSDIRディレクトリ以下のファイル名が不正  
  11     UMD®上にINSDIRディレクトリがない  

SceUtilityGamedataInstallFileList特有のパラメータエラー

  番号     不正なパラメータ  
  12     ファイルリストの設定に矛盾がある  
  fileNum != 0 && pFiles == NULL || fileNum == 0 && pFiles != NULL  
  13     UMD®上にpFilesに指定されたファイルがない  
  14     SceUtilityGamedataInstallFileList.isLastOfGroupにSCE_UTILITY_GAMEDATA_INSTALL_GROUP_XXX以外の値が設定されている  
  15     一番最後のグループにSCE_UTILITY_GAMEDATA_INSTALL_GROUP_LASTがない  
  16     pFilesに設定されているファイル名が重複している  
  17     UMD®上のINSDIR以下にあるファイル数よりもfileNumに指定されたファイル数の方が多い  
  18     SCE_UTILITY_GAMEDATA_INSTALL_GROUP_NUM_MAX数を超えた
