u-law

sceUlawEncode
 [関数]
PCM形式データのu-law形式への変換

形式

#include <sceulaw.h>
int sceUlawEncode (
	unsigned char *out,
	short *in,
	int len
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  out     u-lawデータが出力されるバッファへのポインタ  
  in     エンコードしたいPCMデータへのポインタ  
  len     エンコードしたいPCMデータ長(単位:サンプル数)  

返り値

常に0を返します。

解説

16ビットのPCM波形からu-law圧縮形式へ圧縮（エンコード）します。
入出力データでサンプル数は同じで、かつ、u-law形式の1サンプルのデータは1バイトなので、出力バッファにはlenバイトだけの領域を確保する必要があります。

sceUlawDecode
 [関数]
u-law形式データのPCM形式への変換

形式

#include <sceulaw.h>
int sceUlawDecode (
	unsigned char *out,
	short *in,
	int len
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  out     PCM形式データが出力されるバッファへのポインタ  
  in     デコードしたいu-lawデータへのポインタ  
  len     デコードしたいu-lawデータ長(単位:サンプル数)  

返り値

常に0を返します。

解説

u-law圧縮形式のデータをPCM波形に展開（デコード）します。
入出力データでサンプル数は同じで、かつ、出力PCMデータは16ビットなので、出力バッファはlenの2倍以上バイトの領域を確保する必要があります。
