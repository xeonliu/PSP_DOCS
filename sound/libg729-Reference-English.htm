SCE_G729_RATE_6400 / SCE_G729_RATE_8000 / SCE_G729_RATE_11800 / SCE_G729_RATE_SID / SCE_G729_RATE_SID_OCTET / SCE_G729_RATE_0
 [Constant Definitions]
Codec data bit sizes

Definition

#include <libg729.h>
#define  SCE_G729_RATE_6400	(64)
#define  SCE_G729_RATE_8000	(80)
#define  SCE_G729_RATE_11800	(118)
#define  SCE_G729_RATE_SID	(15)
#define  SCE_G729_RATE_SID_OCTET	(16)
#define  SCE_G729_RATE_0	(0)
Description

These values correspond to the size (bit size) of encoded data for each bitrate.
When VAD frame generation is disabled during encoder initialization, the encoder only generates data with a size corresponding to the specified bitrate (either SCE_G729_RATE_6400, SCE_G729_RATE_8000, or SCE_G729_RATE_11800).
When VAD frame generation is enabled during encoder initialization, the encoder outputs data with sizes of SCE_G729_RATE_SID_OCTET and SCE_G729_RATE_0 in addition to data with a size that corresponds to the specified bitrate.
* Although SCE_G729_RATE_SID is defined, the library does not output data of this size.

sceG729EncodeInitResource
 [Function]
Get encoder resources

Syntax

#include <libg729.h>

SceInt32 sceG729EncodeInitResource(
	SceUInt32 num
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Not multithread safe

Arguments

  num     Number of channels to be encoded simultaneously: 1 to 4  

Return Value

When the function completes normally, SCE_G729_ERROR_NONE is returned.
If the return value is negative (< 0), an error occurred.
G729-specific errors are returned under the following conditions.
  Macro     Description  
  SCE_G729_ERROR_FEATUREUNAVAILABLE     The argument value is greater than 4  
  SCE_G729_ERROR_ALREADYINUSE     The resource has already been obtained  
  SCE_G729_ERROR_INSUFFICIENT     Failed to obtain the resource  
  (insufficient memory)  

(For error values, see the List of Error Codes.)
Additional system-related errors may also be returned.

Description

When the number of encoder channels to be used simultaneously is specified in num, this function obtains memory for all required resources. This function must be called first before using the G729 encoder. Values from 1 to 4 can be specified for the number of resources.
To release resources, call sceG729EncodeTermResource().

Example

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
 
    /* Obtain 4 resources*/
    ret = sceG729EncodeInitResource(4);
    if(ret < 0) return -1 /* Error */


    /* Release all resources */
    ret = sceG729EncodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

sceG729EncodeInit
 [Function]
Initialize encoder

Syntax

#include <libg729.h>

SceInt32 sceG729EncodeInit(
	SceUInt32 bitrate,
	SceUInt32 vadflag
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  bitrate     Bitrate for encoding  
  (Use the following values, which are defined in "libg729.h.")  
  6400bps : SCE_G729_Annex_D  
  8000bps : SCE_G729_Annex_A  
  11800bps : SCE_G729_Annex_E  
  vadflag     Whether or not to generate SID frame and DTX frame  
  (Use the following values, which are defined in "libg729.h.")  
  Disable VAD frame generation: SCE_G729_DTX_DISABLE  
  Enable VAD frame generation: SCE_G729_DTX_ENABLE  

Return Value

When the function completes normally, a handle (greater than or equal to zero) is returned.
If the return value is negative (< 0), an error occurred.
G729-specific errors are returned under the following conditions.
  Macro     Description  
  SCE_G729_ERROR_ILLEGALARGUMENT     The specified argument is out of range  
  SCE_G729_ERROR_NOTINITIALIZED     Resources have not been obtained  
  (sceG729EncodeInitResource() has not been called)  
  SCE_G729_ERROR_INSUFFICIENT     Handle could not be obtained  
  (All channels are already in use)  

(For error values, see the List of Error Codes.)
Additional system related errors may also be returned.

Description

This function returns a handle for using a resource so that one new channel can be used from among the resources that were obtained by sceG729EncodeInitResource(). If channels have already been allocated for the number of resources that were obtained by sceG729EncodeInitResource(), an error (SCE_G729_ERROR_INSUFFICIENT) will occur.
This function is thread safe. The same handle will not be returned even if the same function is called at the same time from another thread.

Example

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
    SceInt32 handle1;
    SceInt32 handle2;
 
    /* Obtain 2 resources */
    ret = sceG729EncodeInitResource(2);
    if(ret < 0) return -1 /* Error */

    /* Allocate a channel */
    handle1 = sceG729EncodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);

    /* Allocate a channel */
    handle2 = sceG729EncodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);



    /* Release a channel */
    sceG729EncodeExit(handle1);

    /* Release a channel */
    sceG729EncodeExit(handle2);

    /* Release all resources */
    ret = sceG729EncodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

sceG729EncodeCore
 [Function]
Core encoder processing

Syntax

#include <libg729.h>

int sceG729EncodeCore(
	SceInt32 handle,
	SceShort16 *pcm_in,
	SceUChar8 *es_out
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe
Arguments

  handle     Handle  
  pcm_in     PCM data (input)  
  es_out     Encoded result (output)  

Return Value

When the function completes normally, SCE_G729_ERROR_NONE is returned.
If the return value is negative (< 0), an error occurred.
G729-specific errors are returned under the following conditions.
  Macro     Description  
  SCE_G729_ERROR_ILLEGALARGUMENT     handle is out of range  
  pcm_in or es_out is a null pointer  
  pcm_in is not 2-byte aligned  
  SCE_G729_ERROR_NOTINITIALIZED     Resources have not been obtained  
  Invalid handle  

(For error values, see the List of Error Codes.)
Additional system related errors may also be returned.

Description

This function encodes one frame of PCM data specified by the input argument pcm_in, according to the bitrate specified by sceG729EncodeInit(). The encoded result is written to es_out. The handle that was obtained from sceG729EncodeInit() should be passed in handle. The only PCM data that can be input is 16-bit monaural data with a sampling frequency of 8KHz, and the input size is always 80 samples (= SCE_G729_ENCODE_IN_SAMPLES).
The input data pcm_in is 2-byte aligned.
The output data is 1-byte aligned.
The data size that is output in es_out is as follows.

  Bitrate     Description  
  6400bps     64 bits  
  8000bps     80 bits  
  11800bps     118 bits  


If VAD frame generation was enabled by sceG729EncodeInit(), SID frames and DTX frames are also generated. The SID frame and DTX frame sizes are as follows.

  Type     Description  
  SID frame     16 bits  
  DTX frame     0 bits  


For output size details, refer to the "libg729 Overview" document.

Example

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
    SceInt32 handle;
    SceUChar8 es_out[SCE_G729_RATE_8000/8];
    SceShort16 pcm_in[SCE_G729_ENCODE_IN_SAMPLES];

    /* Obtain a resource */
    ret = sceG729EncodeInitResource(1);
    if(ret < 0) return -1 /* Error */

    /* Allocate a channel */
    handle = sceG729EncodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);

    /*
        Input 80 samples of PCM data in pcm_in
    */

    /* Encode 1 frame */
    sceG729EncodeCore(handle, pcm_in, es_out);

    /* Release a channel */
    sceG729EncodeExit(handle);

    /* Release all resources */
    ret = sceG729EncodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

sceG729DecodeCore
 [Function]
Core decoder processing

Syntax

#include <libg729.h>

int sceG729DecodeCore(
	SceInt32 handle,
	SceUInt32 bitsize,
	SceUInt32 eraseflag,
	SceUChar8 *es_in,
	SceShort16 *pcm_out
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  handle     Handle  
  bitsize     Bit size of input encoded data  
  eraseflag     Frame corruption (loss) flag  
  es_in     Encoded data (input)  
  pcm_out     Decoded result (output)  

Return Value

When the function completes normally, SCE_G729_ERROR_NONE is returned.
If the return value is negative (< 0), an error occurred.
G729-specific errors are returned under the following conditions.
  Macro     Description  
  SCE_G729_ERROR_ILLEGALARGUMENT     handle is out of range  
  es_in or pcm_out is a null pointer  
  pcm_out is not 2-byte aligned  
  bitsize or eraseflag is out of range  
  SCE_G729_ERROR_NOTINITIALIZED     Resources have not been obtained  
  Invalid handle   

(For error values, see the List of Error Codes.)
Additional system related errors may also be returned.

Description

This function decodes one frame of encoded data specified by the input argument es_in, at a bitrate corresponding to the bit size specified by bitsize and writes the result to pcm_out. The handle that was obtained from sceG729DecodeInit() should be passed in handle.

The input data is 1-byte aligned. The encoded data that is input is either one frame of data with a bitrate of 6400bps, 8000bps, or 11800bps or a SID frame or a DTX frame. When 6400bps, 8000bps, or 11800bps frame data is passed, 8, 10, or 118 is passed, respectively in bitsize. When a SID frame is passed, 16 is passed in bitsize. When a DTX frame is passed, 0 is passed in bitsize.

Always set eraseflag to 0. When this function is used for communication such as VoIP and if the received frame gets corrupted or lost during communication and the frame data is uncertain, call this function with eraseflag set to 1.

The only data that can be output is 16-bit monaural data with a sampling frequency of 8KHz, and the input size is always 80 samples (= SCE_G729_DECODE_OUT_SAMPLES).
The output data of pcm_out is 2-byte aligned.

Example

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
    SceInt32 handle;
    SceUChar8 es_in[SCE_G729_RATE_8000/8];
    SceShort16 pcm_out[SCE_G729_DECODE_OUT_SAMPLES];

    /* Obtain a resource */
    ret = sceG729DecodeInitResource(1);
    if(ret < 0) return -1 /* Error */

    /* Allocate a channel */
    handle = sceG729DecodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);

    /*
        Input one frame of encoded data in es_in
	bitsize = bit size of encoded data
    */

    /* Decode 1 frame */
    sceG729DecodeCore(handle, bitsize, 0, es_in, pcm_out);

    /* Release a channel */
    sceG729DecodeExit(handle);

    /* Release all resources */
    ret = sceG729DecodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}
