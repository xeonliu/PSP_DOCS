SCE_G729_Annex_D / SCE_G729_Annex_A / SCE_G729_Annex_E
 [定数定義]
変換ビットレートの指定値

定義

#include <libg729.h>
#define	SCE_G729_Annex_D	(0)
#define	SCE_G729_Annex_A	(1)
#define	SCE_G729_Annex_E	(2)
解説

sceG729EncodeInit()で指定する変換ビットレートの指定値です。

  SCE_G729_Annex_D     6400 bit/sec  
  SCE_G729_Annex_A     8000 bit/sec  
  SCE_G729_Annex_E     11800 bit/sec  

関連項目

sceG729EncodeInit()

SCE_G729_DTX_DISABLE / SCE_G729_DTX_ENABLE
 [定数定義]
VADフレーム生成の有効／無効を設定する値

定義

#include <libg729.h>
#define	SCE_G729_DTX_DISABLE	(0)
#define	SCE_G729_DTX_ENABLE 	(1)
解説

sceG729EncodeInit()で指定するVADフレーム生成の有効／無効を設定するための値です。

  SCE_G729_DTX_DISABLE     VADフレーム生成無効  
  SCE_G729_DTX_ENABLE     VADフレーム生成有効  

関連項目

sceG729EncodeInit()

SCE_G729_RATE_6400 / SCE_G729_RATE_8000 / SCE_G729_RATE_11800 / SCE_G729_RATE_SID / SCE_G729_RATE_SID_OCTET / SCE_G729_RATE_0
 [定数定義]
コーデックデータのビットサイズ

定義

#include <libg729.h>
#define	SCE_G729_RATE_6400		(64)
#define	SCE_G729_RATE_8000		(80)
#define	SCE_G729_RATE_11800   	(118)
#define	SCE_G729_RATE_SID		(15)
#define	SCE_G729_RATE_SID_OCTET	(16)
#define	SCE_G729_RATE_0		(0)

解説

各ビットレートでのエンコードデータの結果サイズ（ビットサイズ）です。
デコーダの初期化時においてVADフレームの生成を無効に設定している場合は、エンコード結果は指定したビットレートに対応したサイズのデータ（SCE_G729_RATE_6400/ SCE_G729_RATE_8000/ SCE_G729_RATE_11800のいずれか）のみ生成されます。
エンコーダの初期化時において、VADフレームの生成を有効に設定している場合は、指定したビットレートに対応したサイズの他にSCE_G729_RATE_SID_OCTETおよびSCE_G729_RATE_0のサイズのデータを出力します。
※SCE_G729_RATE_SIDは定義されていますが、本ライブラリでは出力されません。

sceG729EncodeInitResource
 [関数]
エンコーダリソースの取得

形式

#include <libg729.h>

SceInt32 sceG729EncodeInitResource(
	SceUInt32 num
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  num     同時にエンコードするチャンネル数：1〜4  

返り値

正常終了の場合、SCE_G729_ERROR_NONEが返ります。
返り値が負（<0）の場合はエラーです。
G729固有のエラーは以下の条件で返されます。
  マクロ     説明  
  SCE_G729_ERROR_FEATUREUNAVAILABLE     引数の値が4より大きい  
  SCE_G729_ERROR_ALREADYINUSE     すでにリソースが取得されている  
  SCE_G729_ERROR_INSUFFICIENT     リソースの取得に失敗  
  （メモリが足りない）  

（エラー値はエラーコード一覧を参照ください。）
他にもシステムに関わるエラー値が返ることがあります。

解説

同時に利用するエンコーダのチャンネル数をnumで指定することで、必要なリソースの全メモリを取得します。G729エンコーダを使用するときには、まずこの関数を呼ぶ必要があります。
リソースの数は、1〜4が指定できます。
リソースを開放するときは、sceG729EncodeTermResource()を呼びます。

使用例

#include <libg729.h>

int main(void)
{
    SceInt32 ret;

    /* リソースを4つ取得 */
    ret = sceG729EncodeInitResource(4);
    if(ret < 0) return -1 /* Error */


    /* リソースをすべて開放 */
    ret = sceG729EncodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

sceG729EncodeInit
 [関数]
エンコーダ初期化

形式

#include <libg729.h>

SceInt32 sceG729EncodeInit(
	SceUInt32 bitrate,
	SceUInt32 vadflag
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフである

引数

  bitrate     エンコードするビットレート  
  ("libg729.h"で定義された以下の値を利用してください。)  
  6400bps : SCE_G729_Annex_D  
  8000bps : SCE_G729_Annex_A  
  11800bps : SCE_G729_Annex_E  
  vadflag     SIDフレーム,DTXフレーム生成の可否  
  ("libg729.h"で定義された以下の値を利用してください。)  
  VADフレーム生成を無効 : SCE_G729_DTX_DISABLE  
  VADフレーム生成を有効 : SCE_G729_DTX_ENABLE  

返り値

正常終了の場合、ハンドル値（0以上の値）が返ります。
返り値が負（<0）の場合はエラーです。
G729固有のエラーは以下の条件で返されます。
  マクロ     説明  
  SCE_G729_ERROR_ILLEGALARGUMENT     範囲外の引数を指定した  
  SCE_G729_ERROR_NOTINITIALIZED     リソースが取得されていない  
  (sceG729EncodeInitResource()が  
  呼ばれていない)  
  SCE_G729_ERROR_INSUFFICIENT     ハンドルが取得できない  
  （すでにすべてのチャンネルが利用  
  されている）  

（エラー値はエラーコード一覧を参照ください。）
他にもシステムに関わるエラー値が返ることがあります。

解説

sceG729EncodeInitResource()で取得したリソースの中から新たに1つ分のチャンネルを利用できるようにし、リソースを利用するためのハンドル値を返します。すでにsceG729EncodeInitResource()で取得したリソース数のチャンネルが確保されている場合は、エラー(SCE_G729_ERROR_INSUFFICIENT)になります。
この関数は、スレッドセーフです。他のスレッドより同タイミングで同関数が呼ばれても同じハンドル値を返すことはありません。

使用例

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
    SceInt32 handle1;
    SceInt32 handle2;

    /* リソースを2つ取得 */
    ret = sceG729EncodeInitResource(2);
    if(ret < 0) return -1 /* Error */

    /* チャンネル確保 */
    handle1 = sceG729EncodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);

    /* チャンネル確保 */
    handle2 = sceG729EncodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);



    /* チャンネル開放 */
    sceG729EncodeExit(handle1);

    /* チャンネル開放 */
    sceG729EncodeExit(handle2);

    /* リソースをすべて開放 */
    ret = sceG729EncodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

sceG729EncodeCore
 [関数]
エンコーダ定期処理

形式

#include <libg729.h>

int sceG729EncodeCore(
	SceInt32 handle,
	SceShort16 *pcm_in,
	SceUChar8 *es_out
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフである

引数

  handle     ハンドル値  
  pcm_in     PCMデータを渡します。  
  es_out     エンコード結果を返します。  

返り値

正常終了の場合、1フレームの出力ビットサイズが返ります。
返り値が負（<0）の場合はエラーです。
G729固有のエラーは以下の条件で返されます。
  マクロ     説明  
  SCE_G729_ERROR_ILLEGALARGUMENT     handleが範囲外  
  pcm_inもしくはes_outがNullポインタ  
  pcm_inが2byteアラインメントでない  
  SCE_G729_ERROR_NOTINITIALIZED     リソースが取得されていない  
  無効なハンドル値  

（エラー値はエラーコード一覧を参照ください。）
他にもシステムに関わるエラー値が返ることがあります。

解説

引数pcm_inに入力した1フレーム分のPCMデータをsceG729EncodeInit()で指定したビットレートに従ってエンコードしes_outに書き込みます。引数handleには、sceG729EncodeInit()で取得したハンドル値を渡します。入力するPCMデータは、サンプリング周波数8Khz,16bit,モノラルのみで入力サイズは、80サンプル(=SCE_G729_ENCODE_IN_SAMPLES)固定です。pcm_inの入力データアラインメントは2byteです。
出力データのアラインメントは、1バイトアラインメントです。
es_outに出力されるデータサイズは以下の通りです。

  ビットレート     説明  
  6400bps     64bit  
  8000bps     80bit  
  11800bps     118bit  


sceG729EncodeInit()でVADフラグ生成を有効にしていた場合は、このほかにSIDフレームとDTXフレームを生成します。SIDフレームとDTXフレームのサイズは以下の通りです。

  種類     説明  
  SIDフレーム     16bit  
  DTXフレーム     0bit  


出力サイズの詳細は、「libg729概要」ドキュメントをご参照ください。

使用例

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
    SceInt32 handle;
    SceUChar8 es_out[SCE_G729_RATE_8000/8];
    SceShort16 pcm_in[SCE_G729_ENCODE_IN_SAMPLES];

    /* リソースを取得 */
    ret = sceG729EncodeInitResource(1);
    if(ret < 0) return -1 /* Error */

    /* チャンネル確保 */
    handle = sceG729EncodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);

    /*
        pcm_in にPCMデータを80サンプル入力
    */

    /* 1フレームエンコード */
    sceG729EncodeCore(handle, pcm_in, es_out);

    /* チャンネル開放 */
    sceG729EncodeExit(handle);

    /* リソースをすべて開放 */
    ret = sceG729EncodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

    /* チャンネル確保 */
    handle1 = sceG729DecodeInit();

    /* チャンネル確保 */
    handle2 = sceG729EncodeInit();


    /* チャンネル開放 */
    sceG729DecodeExit(handle1);

    /* チャンネル開放 */
    sceG729DecodeExit(handle2);

    /* リソースをすべて開放 */
    ret = sceG729DecodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}

sceG729DecodeCore
 [関数]
デコーダ定期処理

形式

#include <libg729.h>

int sceG729DecodeCore(
	SceInt32 handle,
	SceUInt32 bitsize,
	SceUInt32 eraseflag,
	SceUChar8 *es_in,
	SceShort16 *pcm_out
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフである

引数

  handle     ハンドル値  
  bitsize     入力エンコードデータのビットサイズ  
  eraseflag     フレーム破損（喪失）フラグ  
  es_in     エンコードデータを渡します。  
  pcm_out     デコード結果を返します。  

返り値

正常終了の場合、SCE_G729_ERROR_NONEが返ります。
返り値が負（<0）の場合はエラーです。
G729固有のエラーは以下の条件で返されます。
  マクロ     説明  
  SCE_G729_ERROR_ILLEGALARGUMENT     handleが範囲外  
  es_inもしくはpcm_outがNullポインタ  
  pcm_outが2byteアラインメントでない  
  bitsizeもしくはeraseflagの値が範囲外  
  SCE_G729_ERROR_NOTINITIALIZED     リソースが取得されていない  
  無効なハンドル値  

（エラー値はエラーコード一覧を参照ください。）
他にもシステムに関わるエラー値が返ることがあります。

解説

引数es_inに入力した1フレーム分のエンコードデータを引数bitsizeで指定したビットサイズに従ったビットレートでデコードしpcm_outに書き込みます。引数handleには、sceG729DecodeInit()で取得したハンドル値を渡します。

入力データのアラインメントは、1バイトアラインメントです。入力するエンコードデータは、6400bps/8000bps/11800bpsのビットレートの1フレーム分のデータおよびSIDフレームおよびDTXフレームのいずれかです。6400bps/800bps/11800bpsのフレームデータを渡すときは、bitsizeにはそれぞれ8/10/118を渡します。SIDフレームを渡すときは16を渡します。DTXフレームを渡すときは0を渡します。

eraseflagには通常0を設定してください。VoIPなどの通信によりこの関数を利用している場合に、通信による受信フレームが破損または喪失し、フレームデータが不確定な場合は、eraseflag=1を設定してこの関数を呼びます。

出力データは、サンプリング周波数8Khz,16bit,モノラルのみで、入力サイズは80サンプル(=SCE_G729_DECODE_OUT_SAMPLES)固定です。pcm_outの出力データアラインメントは2byteです。

使用例

#include <libg729.h>

int main(void)
{
    SceInt32 ret;
    SceInt32 handle;
    SceUChar8 es_in[SCE_G729_RATE_8000/8];
    SceShort16 pcm_out[SCE_G729_DECODE_OUT_SAMPLES];

    /* リソースを取得 */
    ret = sceG729DecodeInitResource(1);
    if(ret < 0) return -1 /* Error */

    /* チャンネル確保 */
    handle = sceG729DecodeInit(SCE_G729_Annex_A, SCE_G729_DTX_DISABLE);

    /*
        es_in にエンコードデータ1フレームぶん入力
	bitsize = エンコードデータのビットサイズ
    */

    /* 1フレームデコード */
    sceG729DecodeCore(handle, bitsize, 0, es_in, pcm_out);

    /* チャンネル開放 */
    sceG729DecodeExit(handle);

    /* リソースをすべて開放 */
    ret = sceG729DecodeTermResource();
    if(ret < 0) return -1 /* Error */

    return 0;
}
