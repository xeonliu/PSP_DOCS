SCE_AUDIOROUTING_NORMAL
 [定数定義]
スピーカー出力切替通常モード

定義

#include <audiorouting.h>
#define	SCE_AUDIOROUTING_NORMAL		0
解説

通常のスピーカー出力切替状態(PSP®本体ヘッドホン端子への機器接続の有無によりスピーカー出力の無効有効を切り替える)を表す定数です。

SCE_AUDIOROUTING_VOLUME_MODE_NORMAL
 [定数定義]
通常音量モード

定義

#include <audiorouting.h>
#define	SCE_AUDIOROUTING_VOLUME_MODE_NORMAL		0
解説

通常の音量出力状態(PSP®本体で設定した音量でサウンド出力する)を表す定数です。

関連項目

sceAudioRoutingSetVolumeMode()

SCE_AUDIOROUTING_VOLUME_MODE_FORCE_MAX
 [定数定義]
常時最大音量モード

定義

#include <audiorouting.h>
#define	SCE_AUDIOROUTING_VOLUME_MODE_FORCE_MAX		1
解説

PSP®本体で設定した音量にかかわらず、常時最大音量でサウンド出力するモードを表す定数です。

sceAudioRoutingSetVolumeMode()

sceAudioRoutingSetMode
 [スピーカー出力切替機能]
スピーカー出力自動切替の動作設定

形式

#include <audiorouting.h>
int sceAudioRoutingSetMode(
	SceInt32 mode
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  mode     切替モードの指定  
  SCE_AUDIOROUTING_MODE0(=SCE_AUDIOROUTING_NORMAL)  
  SCE_AUDIOROUTING_MODE1  
  SCE_AUDIOROUTING_MODE2  
  SCE_AUDIOROUTING_MODE3  
  のいずれかを指定してください。  

返り値

正常動作時に0、引数modeが不正なときにSCE_AUDIOROUTING_ERROR_INVALを返します。

解説

PSP®内蔵ソフトウェアが自動で行う、スピーカー出力切替の挙動を変化させることができます。具体的には、ヘッドホン端子に何かがつながれていないときのみスピーカー出力を行うモード、ヘッドホン端子への機器の接続の有無にかかわらずスピーカー出力を行うモード、ヘッドホン端子への出力は行わずスピーカー出力のみ行うモード、スピーカーへの出力は行わずヘッドホン端子出力のみ行うモードのいずれかに設定することができます。
sceAudioRoutingGetMode
 [スピーカー出力切替機能]
スピーカー出力自動切替のモード取得

形式

#include <audiorouting.h>
int sceAudioRoutingGetMode(
	void
);
呼出条件

割り込みハンドラから呼び出し可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

なし

返り値

現在のスピーカー切替機能の状態を返します。SCE_AUDIOROUTING_MODE0、SCE_AUDIOROUTING_MODE1、SCE_AUDIOROUTING_MODE2またはSCE_AUDIOROUTING_MODE3のいずれかになります。

解説

PSP®の内蔵ソフトウェアが自動で行う、スピーカー出力切替の挙動が現在どの状態になっているかを返します。

sceAudioRoutingSetVolumeMode
 [音量モード設定機能]
音量モード動作設定

形式

#include <audiorouting.h>
int sceAudioRoutingSetVolumeMode(
	SceInt32 mode
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  mode     音量モードの指定  
  SCE_AUDIOROUTING_VOLUME_MODE_NORMAL  
  SCE_AUDIOROUTING_VOLUME_MODE_FORCE_MAX  
  のいずれかを指定してください。  

返り値

正常動作時に0、引数modeが不正なときにSCE_AUDIOROUTING_ERROR_INVALを返します。

解説

PSP®本体の音量モードの設定を変更させることができます。具体的には、PSP®本体の音量ボタンで設定した音量でサウンド出力するモードと、常に最大音量でサウンド出力するモードのどちらかに設定することができます。
