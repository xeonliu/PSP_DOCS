解説

SCE_WAVE_LOOP
 [定数定義]
ループ指定フラグ

定義

#include <libwave.h>
#define	 SCE_WAVE_LOOP		(1 << 0)
解説

libwaveのADPCM(VAG)再生サウンドシステムにおいて、無限ループ再生を行う場合に指定するフラグです。sceWaveSetVoice(), sceWaveSetLoopMode()で指定できます。
SCE_WAVE_LOOPが指定されている場合、VAG内にループ指定があればそのループ指定を元に無限ループ再生を行います。VAG内にループ指定がない場合にはVAG全体をループ範囲とみなし、無限ループ再生を行います。

SCE_WAVE_LOOPはSCE_WAVE_VAGFLAGと同時に指定できません。

関連項目

sceWaveSetVoice(), sceWaveSetLoopMode(), SCE_WAVE_VAGFLAG

SCE_WAVE_VAGFLAG
 [定数定義]
VAGファイル内のループ設定を利用する

定義

#include <libwave.h>
#define	SCE_WAVE_VAGFLAG		(1 << 1)
解説

libwaveのADPCM(VAG)再生サウンドシステムにおいて、VAGファイル内のループフラグを利用してループ再生を行う場合に指定するフラグです。sceWaveSetVoice(), sceWaveSetLoopMode()で指定できます。
SCE_WAVE_VAGFLAGが指定されている場合、VAG内にループ設定があればそのループ設定を元に無限ループ再生を行います。VAG内にループ設定がない場合にはワンショット再生を行います。

SCE_WAVE_VAGFLAGはSCE_WAVE_LOOPと同時に指定できません。

関連項目

sceWaveSetVoice
 [関数]
サウンドデータの設定

形式

int sceWaveSetVoice(
	int iVoice,
	const void* pBuf,
	SceSize size,
	int iMode
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

引数

  iVoice     ボイス番号 (0 - 7)  
  pBuf     サウンドデータが存在するバッファへのポインタ  
  size     サウンドデータのサイズ(バイト数)  
  iMode     モード  
  SCE_WAVE_LOOP:無限回ループ指定  
  SCE_WAVE_VAGFLAG:VAGファイル内のループ設定を利用  
  0:oneshot再生  

返り値

  マクロ     値     説明  
  SCE_OK     0     正常終了  
  SCE_WAVE_ERROR_VOICENUM     0x80440005     ボイス番号が不正  
  SCE_WAVE_ERROR_BUFFPTR     0x80440006     バッファへのポインタが不正  
  SCE_WAVE_ERROR_SIZE     0x80440007     データのサイズが不正  
  SCE_WAVE_ERROR_MODE     0x80440008     モードが不正  
  SCE_WAVE_ERROR_VOICEPLAY     0x80440009     指定のボイスは再生中  


解説

iVoiceで指定した番号のボイスに、pBufとsizeで指定したバッファ内のVAGデータをセットします。セットしたVAGデータは、sceWavePlay()でボイス番号を指定することによって再生できます。
iModeにて各種設定が可能です。VAGデータに含まれるループ設定を利用したい場合にはiModeにSCE_WAVE_VAGFLAGを、VAGデータを強制的にone shot再生する場合にはiModeに0を指定してください。また強制的にループ再生させたい場合にはiModeに SCE_WAVE_LOOPを指定してください。
なお指定したボイスが再生中である場合には設定できません。ボイスが停止状態(SCE_WAVE_STAT_STOP)であることを確認してから設定を行ってください。

sceWaveSetVoice()はpBufで指定されたバッファ内VAGデータのコピーを作成しません。指定されたバッファ内のVAGデータを直接再生します。VAGデータの再生中はバッファ内データを保持しておいてください。
pBufで指定したバッファはpBufで指定したバッファ内VAGデータの再生中に開放しないでください。再生中にバッファを開放した場合の動作は不定となります。バッファを開放する場合には必ず停止状態(SCE_WAVE_STAT_STOP)であることを確認してから行ってください。同様に再生中にバッファ内の内容を変更した場合の動作も不定となります。バッファ内の内容を変更する場合にも必ず停止状態(SCE_WAVE_STAT_STOP)であることを確認してから行ってください。

使用例

/* 0ボイスにpVagBufを設定してone shot再生 */
sceWaveSetVoice(0, pVagBuf, VagSize, 0);

/* 0ボイスにpVagBufを設定して無限回ループ再生 */
sceWaveSetVoice(0, pVagBuf, VagSize, SCE_WAVE_LOOP);

/* 0ボイスにpVagBufを設定してVAG内のループ設定に従って再生 */
sceWaveSetVoice(0, pVagBuf, VagSize, SCE_WAVE_VAGFLAG);

スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフでない

  マクロ     値     説明  
  SCE_OK     0     正常終了  
  SCE_WAVE_ERROR_VOICENUM     0x80440005     ボイス番号が不正

使用例

sceWaveSetVolume(0, SCE_WAVE_VOL_MAX, SCE_WAVE_VOL_MAX );

備考

オーディオドライバの出力処理が終わるまで、出力データバッファの内容を変更しないでください。
