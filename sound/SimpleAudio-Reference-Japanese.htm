sceAudioOutput2Reserve
 [関数]
オーディオ出力機能の初期化と使用予約

形式

int sceAudioOutput2Reserve(
	int iSampleNum
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  iSampleNum     粒度(一度に出力するサンプル長、17 - 4111)  

返り値

正常終了時には0が、エラー時にはそれ以外の値が返ります(詳細はエラーコード一覧を参照してください)。

解説

与えられた粒度(一度の出力関数呼び出しで出力されるサンプル数)で、オーディオ出力機能の初期化を行い、その機能を使用することを予約します。
粒度にはSCE_AUDIO_OUTPUT2_SMPL_MIN以上SCE_AUDIO_OUTPUT2_SMPL_MAX以下の任意の値が設定可能です。

使用例

/* 粒度512 サンプルでオーディオ出力機能を初期化する */
sceAudioOutput2Reserve(512 );

sceAudioOutput2ChangeLength
 [関数]
出力するオーディオサンプルの長さを設定

形式

int sceAudioOutput2ChangeLength (
	int iSampleNum
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  iSampleNum     粒度(一度に出力するサンプル長、17 - 4111)  

返り値

正常終了時には0が、エラー時にはそれ以外の値が返ります(詳細はエラーコード一覧を参照してください)。

解説

粒度(一度の出力関数呼び出しで出力されるサンプル数)を変更します。オーディオ出力中であった場合、この関数を呼び出した次の出力からサンプル長が変化します。
粒度にはSCE_AUDIO_OUTPUT2_SMPL_MIN以上SCE_AUDIO_OUTPUT2_SMPL_MAX以下の任意の値が設定可能です。

使用例

/*出力長を 700 サンプル( 700 * sizeof(short) * 2(ch) バイト)に変更 */
sceAudioOutput2ChangeLength(700);
関連項目

sceAudioOutput2OutputBlocking()

sceAudioOutput2OutputBlocking
 [関数]
オーディオデータの出力(ブロッキング)

形式

int sceAudioOutput2OutputBlocking(
	int iVol,
	const void* pBuf
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  iVol     ボリューム値 (0 - SCE_AUDIO_VOLUME_0dB*32-1)  
  pBuf     出力データバッファへのポインタ  

返り値

0以上の場合はオーディオドライバに登録できたサンプル数を返します。それ以外の場合はエラー値となります(詳細はエラーコード一覧を参照してください)。

解説

オーディオ出力を行います。具体的にはpBufに指定された出力データバッファをオーディオドライバに登録する処理を行います。出力データバッファがオーディオドライバに登録できなかった場合、登録できるまで当APIを呼び出したスレッドをWAIT状態にし、登録できるまで待ちます。オーディオドライバへ登録可能な状態になるとスレッドはREADY→RUN状態となり、オーディオドライバへの出力バッファ登録処理を行います。オーディオドライバへの登録が終了すると呼び出し元の処理に戻ります。登録されるデータの長さは、オーディオ出力を初期化したときに指定した値となります。ただしsceAudioOutput2ChangeLength()にて設定変更を行なっていた場合は、同関数にて最後に指定された値が使用されます。
なお、pBufにNULLを指定すると、実際の音声出力は行わず、現在オーディオドライバに登録されているが、まだ出力されていない音声データがすべて出力されるまで待つということが実現できます。ただし、これは「現在」登録されているデータの出力を待つだけです。別スレッドがオーディオ出力を行っている場合、待ちに入った後に別スレッドによりオーディオ出力が新規に行われる可能性があるため、別スレッドの行っているオーディオ出力がすべて終了するまで待つ、といった使い方はできません(このような目的にはイベントフラグなどのスレッド間同期機能を用いてください)。

備考

オーディオドライバの出力処理が終わるまで、出力データバッファの内容を変更しないでください。
音量として非常に大きな値まで設定が可能ですが、通常は音が割れますので、SCE_AUDIO_VOLUME_0dBに比べて大きすぎる値は出来るだけ設定しないようにしてください。

使用例

/* pcmBufにある16 bit PCM STEREOデータを音量変化無しで出力する */
sceAudioOutput2OutputBlocking(
	SCE_AUDIO_VOLUME_0dB,
	pcmBuf
);

sceAudioOutput2GetRestSample
 [関数]
未再生サンプル数を取得する

形式

int sceAudioOutput2GetRestSample(
	void
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

なし

返り値

返り値が0以上の場合、未再生サウンドデータのサンプル数を返します。それ以外の場合はエラー値となります(詳細はエラーコード一覧を参照してください)。

解説

現在ドライバ内で出力が予定されているが、まだ未再生となっているサウンドデータのサンプル数を返します。

使用例

/*未再生サンプル数を得る */
int restSample;
restSample = sceAudioOutput2GetRestSample();
