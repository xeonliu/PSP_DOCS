SceP3daVector
 [構造体]
ベクトル（直交座標）

構造

typedef struct{
	int x,
	int y,
	int z
} SceP3daVector;
メンバ

  x     ベクトルのX軸方向の成分  
  y     ベクトルのY軸方向の成分  
  z     ベクトルのZ軸方向の成分  

解説

3次元の距離ベクトル、もしくは速度ベクトルを直交座標で表現します。
距離ベクトルの場合単位は、ミリメートル[mm]で28bitの整数表現になります。
したがって-0x0fffffff 〜 0 〜 +0x0fffffff (±268435455)の範囲になります。
速度ベクトルの場合単位は、ミリメートル／秒[mm/sec]になります。
速度ベクトルの値も-0x0fffffff 〜 0 〜 +0x0fffffff (±268435455)の範囲になります。

sceP3daInit
 [関数]
初期化

形式

#include <libp3da.h>

int sceP3daInit(
	int iNumInputs,
	int iAudioBlocksize
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止 / 許可状態に依存しない）
マルチスレッドセーフでない

引数

  iNumInputs     扱うサウンドソースの数 1〜SCE_P3DA_MAXCHANNELS  
  iAudioBlocksize     コア処理1回の処理サンプル数  

返り値

正常終了の場合、SCE_P3DA_ERROR_NONE
返り値が負（<0）の場合エラー
（詳細はエラーコード一覧を参照して下さい）

解説

P3DAの状態を初期化します。
P3DAを使用するときに、まずこの関数を呼ぶ必要があります。

iNumInputsは、入力サウンドソースの最大数です。
iAudioBlocksizeでは、sceP3daCore()が処理するサンプル数を指定します。
このサンプル数は、関数sceP3daCore()の出力サンプル数でもあります。
iAudioBlocksizeは、最小値SCE_P3DA_BLOCK_SAMPLES_MIN (=64)と、最大値SCE_P3DA_BLOCK_SAMPLES_MAX (=2048)の間を、SCE_P3DA_BLOCK_SAMPLES_STEP (=32)ステップで指定可能です。
それ以外の値は、エラーになります。
入出力波形データの渡し方についての詳細については、ドキュメント「libp3da-Overview」をご参照ください。

使用例

#include <libp3da.h>

int main(void)
{
    int result;
    
    result = sceP3daInit(SCE_P3DA_MAXCHANNELS,  1024);


    sceP3daExit();
    return 0;
}

sceP3daCore
 [関数]
P3DA定期処理

形式

#include <libp3da.h>

int sceP3daCore(
	const short *const *ppsInputs,
	short *psOutput
);
呼出条件

割り込みハンドラから呼び出し不可能
スレッドから呼び出し可能（割り込み許可状態で呼び出すこと）
マルチスレッドセーフである

引数

  ppsInputs     波形入力バッファのポインタの配列  
  psOutput     波形出力バッファのポインタ  

返り値

正常終了の場合、SCE_P3DA_ERROR_NONE
返り値が負（<0）の場合エラー
（詳細はエラーコード一覧を参照して下さい）

解説

当関数をコールする事で、一粒度分の波形を入出力します。一粒度のサンプル数は、関数sceP3daInit()で指定したサンプルサイズになります。
入出力データそのものは、64バイトアラインメントが要求されます。
ppsInputs[]には、各サウンドソース用入力波形データの先頭アドレスを格納しておく必要があります。ppsInputs[]は、必ずsceP3daInit()で指定したチャンネル数の配列を確保し、利用しないサウンドソースについては、NULLを書いてください。
この関数は、マルチスレッドセーフですので、この関数の呼び出し中に例えばsceP3daSetPosition()関数などサウンドコントロールの関数を呼び出しても問題ありません。ただし、sceP3daCore()を複数スレッドから同時に呼び出すことはできません(SCE_P3DA_ERROR_ILLEGALSTATEが返ります)。

使用例

#include<libp3da.h>

#define NINPUT 3   /* 3つのサウンドソースを処理 */
#define STEREO 2

int main(void)
{
    int i;
    int result;
    short *apsInputPCM[SCE_P3DA_MAXCHANNELS];
    static short aasInputPCM[SCE_P3DA_MAXCHANNELS][SCE_P3DA_BLOCK_SAMPLES_DEFAULT * STEREO] __attribute__((aligned(64)));
    static short asOutputPCM[SCE_P3DA_BLOCK_SAMPLES_DEFAULT * STEREO] __attribute__((aligned(64)));

    /* 初期化 */
    sceP3daInit(NINPUT, SAMPLES);
    for(i=0; i<SCE_P3DA_MAXCHANNELS; i++)  apsInputPCM[i]=NULL;

    /* apsInputPCM[] の設定 */
    for(i=0; i<NINPUT; i++)  apsInputPCM[i]=aasInputPCM[i];

    for(;;){
        /* aasInputPCM[0,1,2][] に入力波形をセットする */

        /* 1粒度分波形出力処理 */
        result = sceP3daCore(apsInputPCM, asOutputPCM);

        /* asOutputPCM をサウンド出力 */
    }

    return 0;
}
