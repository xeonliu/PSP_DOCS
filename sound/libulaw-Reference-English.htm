u-law Reference

sceUlawEncode
 [Function]
Convert PCM format data to u-law format

Syntax

#include <sceulaw.h>
int sceUlawEncode (
	unsigned char *out,
	short *in,
	int len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on an interrupt-disabled or -enabled state)
Multithread safe

Arguments

  out     Pointer to buffer where u-law data will be output  
  in     Pointer to PCM data to be encoded  
  len     Length of PCM data to be encoded (units: number of samples)  

Return Value

Zero is always returned.

Description

This function compresses (encodes) a 16-bit PCM waveform into u-law compressed format.
Because the number of samples of input and output data is the same, and one sample corresponds to one byte of u-law data, an area of len bytes must be allocated for the output buffer.

sceUlawDecode
 [Function]
Convert u-law format data to PCM format

Syntax

#include <sceulaw.h>
int sceUlawDecode (
	unsigned char *out,
	short *in,
	int len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on an interrupt-disabled or -enabled state)
Multithread safe

Arguments

  out     Pointer to buffer where PCM-format data will be output  
  in     Pointer to u-law data to be decoded  
  len     Length of to u-law data to be decoded (units: number of samples)  

Return Value

Zero is always returned.

Description

This function expands (decodes) u-law compressed-format data into a PCM waveform.
Because the number of samples of input and output data is the same and the output PCM data is 16 bits, an area of at least two times len bytes must be allocated for the output buffer.
