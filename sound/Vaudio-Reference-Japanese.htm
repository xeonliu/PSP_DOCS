SCE_VAUDIO_VOLUME_0dB
 [定数定義]
音量変化なしのときのボリューム値

定義

#include <vaudio.h>
#define	SCE_VAUDIO_VOLUME_0dB		(0x8000)
解説

仮想オーディオドライバにおいて、音量変化なしで出力を行いたいときのボリューム値です。

sceVaudioChReserve
 [関数]
仮想オーディオ出力チャネルの予約

形式

#include <vaudio.h>
int sceVaudioChReserve(
	int len,
	int frequency,
	int format
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  len     チャネルあたりのデータ長(サンプル数単位で指定)  
  frequency     出力データのサンプリング周波数  
  format     データ形式  

返り値

成功時に0、それ以外の場合には下記エラー値となります。

  マクロ     値     説明  
  SCE_ERROR_INVALID_SIZE     0x80000104     lenの値が不正  
  SCE_ERROR_INVALID_FORMAT     0x80000108     formatの値が不正  
  SCE_AUDIO_ERROR_INVALID_FREQUENCY     0x8026000a     frequencyの値が不正  
  SCE_ERROR_BUSY     0x80000021     既に予約されている状態  
  である  


解説

仮想オーディオ出力チャネルの予約と設定を行います。

frequencyに指定できる値は以下のとおりです。
48000,44100,32000,24000,22050,16000,12000,11025,8000
また、lenに指定できる値は以下のとおりです。
256,576,1024,1152,2048

sceVaudioOutputBlocking
 [関数]
オーディオデータの出力(ブロッキング)

形式

#include <vaudio.h>
int sceVaudioOutputBlocking(
	int iVol,
	void* pBuf
)
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  iVol     ボリューム値(0 - 1048575)  
  pBuf     出力データバッファへのポインタ  

返り値

0以上の場合はオーディオドライバに登録できたサンプル数を返します。それ以外の場合には下記エラー値となります。

  マクロ     値     説明  
  SCE_AUDIO_ERROR_INVALID_VOLUME     0x8026000B     ボリューム値が不正  
  SCE_ERROR_PRIV_REQUIRED     0x80260004     出力バッファへのポインタが不正  


解説

論理チャネルへのオーディオ出力を行います。
pBufに指定された出力データバッファはオーディオドライバに登録されます。オーディオドライバは登録されているサウンドデータをDACへ出力します。出力データバッファがオーディオドライバに登録できなかった場合、登録できるまで当APIを呼び出したスレッドをWAIT状態にし、登録できるまで待ちます。オーディオドライバへ登録可能な状態になるとスレッドはREADY→RUN状態となり、オーディオドライバへの出力バッファ登録処理を行います。オーディオドライバへの登録が終了すると呼び出し元の処理に戻ります。

使用例

/* pcmBuf にあるデータを音量変化なしで出力する */
sceVaudioOutputBlocking(
	SCE_VAUDIO_VOLUME_0dB,
	pcmBuf
);
sceVaudioSetEffectType
 [関数]
仮想オーディオ出力時のエフェクトタイプ設定

形式

#include <vaudio.h>
int sceVaudioSetEffectType(
	int type,
	int vol
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  type     エフェクトタイプ  
  vol     アナログボリューム設定値  

返り値

成功時に補正後のアナログボリューム設定値、それ以外の場合には下記エラー値となります。

  マクロ     値     説明  
  SCE_ERROR_INVALID_INDEX     0x80000102     エフェクトタイプが不正  


解説

仮想オーディオ出力を行ったときにかけられるエフェクトの種別を設定します。
エフェクト処理に伴って音圧レベルが変化してしまうため、それを補正するために、アナログボリューム値を調整する必要があります。この関数の戻り値として調整後のボリューム値が返りますので、この値を使ってボリューム設定を行ってください。
