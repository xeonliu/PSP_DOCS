    The encoding that describes Japanese comments within the source file has been changed to UTF-8. Along with this change, a script that filters comments has been added so that only the Japanese or English comments can be left. You can leave either comment as follows:

  • When you wish to leave only the Japanese comments:
    % find /usr/local/psp/devkit/include -name "*.h" -exec /usr/local/psp/devkit/bin/psp-selcomment.rb --japanese -o {} {} \;

  • When you wish to leave only the English comments:
  • % find /usr/local/psp/devkit/include -name "*.h" -exec /usr/local/psp/devkit/bin/psp-selcomment.rb --english -o {} {} \;

1. 15 Release 2.5.0

  • A library to calculate SHA-224 with RFC3874 has been added (libsha224).

  • The Simple Audio Output library has been added.
    The following files are required to use it:

    Library: libwave.a
    Header file: audiooutput.h

    Also, both the Simple Audio Output library and the libwave have the same audio driver operating part so the Simple Audio Output requires the same .a file as libwave.

    This library never performs mixing, etc. by the CPU and directly uses DMA to output to the hardware. Also, an explanation on the audio output system with digital volume used by this library has been added to the "Sound Overview" document.

    Sound data processing by the CPU has been minimized for mixing, etc. so the sound producing load has also been reduced. The sound volume can be changed by the hardware volume, however most of the other useful functions with libwave such as format conversion, etc., are not supported so they must be implemented on the CPU.

1. 21 Release 1.0.0

powersvc.h, power_stub.a
Added kernel mode service modules (powersvc.h, power_stub.a) for performing control related to PSP™ power management.
These are modules for controlling standby or resume modes by pressing or pressing and holding down the POWER switch.

Accompanying this addition, the automatic backlight off function was added.
The default is "brightness dims to LOW after one minute of idle state and then backlight goes off when another minute elapses." The backlight returns to its original brightness on key input.
For details on how to control these functions from a game, refer to the "Power Overview" document.
