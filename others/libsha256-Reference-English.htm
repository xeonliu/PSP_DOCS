SceSha256Context
 [Structure]
SHA-256 digest value calculation context information

Structure

#include <libsha256.h>
typedef struct SceSha256Context {
	unsigned int h[8];
	unsigned int pad;
	unsigned short usRemains;
	unsigned short usComputed;
	unsigned long long ulTotalLen;
	unsigned char buf[SCE_SHA256_BLOCK_SIZE];
} SceSha256Context;
Members

  h     Work area  
  pad     Padding for adjusting alignment  
  usRemains     Size (less than 64 bytes) of remaining data  
  which was temporarily copied into the  
  SceSha256Context structure  
  usComputed     Digest value calculation completed flag  
  ulTotalLen     Total data size  
  buf     Temporary copy of data (less than 64 bytes)  

Description

This structure is used as a work area when segmenting SHA-256 digest value calculations. The sceSha256BlockInit() function, sceSha256BlockUpdate() function, and sceSha256BlockResult() function use it as a work area, so applications must not access the members of this structure directly.

sceSha256Digest
 [Function]
Calculate SHA-256 digest

Syntax

#include <libsha256.h>
int sceSha256Digest (
	const void *plain,
	unsigned int len,
	unsigned char *digest
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  plain     Specifies a pointer to the plaintext data for which to calculate the   
  digest value  
  len     Specifies the data size (in bytes) of the plaintext data for which to   
  calculate the digest value  
  digest     Returns the computed digest value (32 bytes)  

Return Value

If an error occurs, a negative number is returned.
  Value     Result

Description

This function calculates a SHA-256 digest value.
It is used when the plaintext data on which to calculate the digest value has been completely expanded in memory.

sceSha256BlockUpdate
 [Function]
Perform SHA-256 digest value calculation

Syntax

#include <libsha256.h>
int sceSha256BlockUpdate (
	SceSha256Context *pContext,
	const void *plain,
	unsigned int len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pContext     Specifies the address of the digest value calculation work area  
  plain     Specifies a pointer to the plaintext data for which to calculate the   
  digest value  
  len     Specifies the data size (in bytes) of the plaintext data for which to   
  calculate the digest value  

Return Value

If an error occurs, a negative number is returned.
  Value     Result  
  SCE_OK     Successful completion  
  SCE_ERROR_BUSY     Digest value calculation was already   
  completed on pContext  
  SCE_ERROR_INVALID_POINTER     pContext or plain address is invalid  


Description

This function updates the work area in the SceSha256Context structure using the plaintext specified by plain and len. The sceSha256BlockUpdate() function can be called any number of times in between the sceSha256BlockInit() function and the sceSha256BlockResult() function; by calling the sceSha256BlockUpdate() function in a repetitive manner, digest values can be calculated even for large amounts of data which cannot fit in memory all at once.

See Also

SceSha256Context, sceSha256BlockInit(), sceSha256BlockResult()

sceSha256BlockResult
 [Function]
Get calculated SHA-256 digest

Syntax

#include <libsha256.h>
int sceSha256BlockResult (
	SceSha256Context *pContext,
	unsigned char *digest
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pContext     Specifies the address of the digest value calculation work area  
  digest     Returns the computed digest value (32 bytes)  

Return Value

If an error occurs, a negative number is returned.
  Value     Result  
  SCE_OK     Success  
  SCE_ERROR_INVALID_POINTER     pContext or digest address is invalid  


Description

This function obtains the calculated digest value from the SceSha256Context structure. Since SHA-256 is an algorithm which calculates digest values in 64-byte units, remaining bytes which did not fill 64 bytes by using the sceSha256BlockUpdate() function may be temporarily copied into the SceSha256Context structure. If data remains which has been temporarily copied into the SceSha256Context structure, the final digest value will be determined by calling the sceSha256BlockResult() function. Be sure to call the sceSha256BlockResult() function to obtain the digest value.
The digest value in the SceSha256Context structure will remain valid until the next time that the sceSha256BlockInit() function is called.
