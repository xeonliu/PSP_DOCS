SceMd5Context
 [構造体]
MD5ダイジェスト値計算のコンテキスト情報

構造

#include <libmd5.h>
typedef struct SceMd5Context {
	unsigned int h[4];
	unsigned int pad;
	unsigned short usRemains;
	unsigned short usComputed;
	unsigned long long ulTotalLen;
	unsigned char buf[64];
} SceMd5Context;
メンバ

  h     ワークエリア  
  pad     パディング  
  usRemains     SceMd5Context構造体内に一時コピーされた、64バイト未満の余りデータ数  
  usComputed     ダイジェスト値計算済みフラグ  
  ulTotalLen     総データサイズ  
  buf     64バイト未満のデータの一時コピー  

解説

MD5ダイジェスト値の計算を分割して行う場合のワークエリアとして用いられます。sceMd5BlockInit()関数, sceMd5BlockUpdate()関数, sceMd5BlockResult()関数がワークエリアとして使用しますので、アプリケーションがこの構造体のメンバに直接アクセスしてはいけません。

関連項目

sceMd5BlockInit(), sceMd5BlockUpdate(), sceMd5BlockResult()

sceMd5Digest
 [関数]
MD5ダイジェストの計算

形式

#include <libmd5.h>
int sceMd5Digest (
	const void *plain,
	unsigned int len,
	unsigned char *digest
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  plain     ダイジェスト値を計算する平文のデータへのポインタを指定します。  
  len     ダイジェスト値を計算する平文のデータサイズ（バイト数）を指定します。  
  digest     算出されたダイジェスト値を返します。  
  16バイトの領域を用意してください。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

MD5ダイジェスト値を計算します。
この関数はダイジェスト値を計算する平文のデータがすべてメモリ上に展開されている場合に用います。
sceMd5BlockInit
 [関数]
ダイジェスト値計算ワークの初期化

形式

#include <libmd5.h>
int sceMd5BlockInit (
	SceMd5Context *pContext
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     ダイジェスト値計算ワークのアドレスを指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  
  SCE_ERROR_INVALID_POINTER     pContextのアドレスが不正  


解説

Md5ダイジェスト値計算に用いるワークエリアを初期化します。
この関数はsceMd5BlockUpdate()関数に先立って呼び出しを行ってください。

関連項目

SceMd5Context, sceMd5BlockUpdate(), sceMd5BlockResult()
sceMd5BlockUpdate
 [関数]
MD5ダイジェスト値計算処理

形式

#include <libmd5.h>
int sceMd5BlockUpdate (
	SceMd5Context *pContext,
	const void *plain,
	unsigned int len
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     ダイジェスト値計算ワークのアドレスを指定します。  
  plain     ダイジェスト値を計算する平文のデータへのポインタを指定します。  
  len     ダイジェスト値を計算する平文のデータサイズ（バイト数）を指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  
  SCE_ERROR_BUSY     ダイジェスト値が計算済みのpContextだった  
  SCE_ERROR_INVALID_POINTER     pContextまたはplainのアドレスが不正  


解説

plain, lenで指定された平文を使ってSceMd5Context構造体内のワークを更新します。sceMd5BlockUpdate()関数は、sceMd5BlockInit()関数とsceMd5BlockResult()関数の間で何回でも呼び出すことが可能で、複数回に分けてsceMd5BlockUpdate()関数を呼び出すことでメモリに入りきらない巨大なデータに対してもダイジェスト値を計算することが可能です。

関連項目

SceMd5Context, sceMd5BlockInit(), sceMd5BlockResult()

sceMd5BlockResult
 [関数]
計算されたMD5ダイジェストを取得

形式

#include <libmd5.h>
int sceMd5BlockResult (
	SceMd5Context *pContext,
	unsigned char *digest
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     ダイジェスト値計算ワークのアドレスを指定します。  
  digest     算出されたダイジェスト値を返します。  
  16バイトのエリアを用意してください。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     成功  
  SCE_ERROR_INVALID_POINTER     pContextまたはdigestのアドレスが不正  


解説

計算されたダイジェスト値をSceMd5Context構造体から取り出します。MD5は64バイト単位でダイジェスト値を計算するアルゴリズムであるため、sceMd5BlockUpdate()関数では64バイトに満たなかった余剰バイトをSceMd5Context構造体内に一時コピーすることがあります。SceMd5Context構造体内一時コピーされたデータが残っていた場合には、sceMd5BlockResult()関数の呼び出しで最終的なダイジェスト値が決定されます。かならず、sceMd5BlockResult()関数でダイジェスト値を取得してください。
SceMd5Context構造体のダイジェスト値は、次にsceMd5BlockInit()関数を呼び出すまで有効です。
