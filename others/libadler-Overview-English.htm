libadler Overview

1 Library Overview

1. 1 Overview

The ADLER-32 Encoding Library (libadler) is a library that is used to compute the ADLER32 checksum.

1. 2 Related Files

The following files are required to use libadler.

  Category     Filename  
  Header file     libadler.h  
  Static link library file     libadler.a  
  Stub library file     libadler_stub.a  
  weak import stub library file     libadler_stub_weak.a  
  PRX module file     libadler.prx  


2 Usage Procedure

2. 1 Basic Usage Procedure

No specific initialization is required to use libadler.

adler32 checksum computation

The following procedure can be used to compute the byte string adler32 format checksum.

unsigned int uiChksum;

uiChkSsum = sceAdler32(0, inputdat, SOURCE_LEN);
