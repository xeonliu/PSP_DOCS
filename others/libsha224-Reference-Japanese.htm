SceSha224Context
 [構造体]
SHA-224ダイジェスト値計算のコンテキスト情報

構造

#include <libsha224.h>
typedef struct SceSha224Context {
	unsigned int h[8];
	unsigned int pad;
	unsigned short usRemains;
	unsigned short usComputed;
	unsigned long long ulTotalLen;
	unsigned char buf[SCE_SHA224_BLOCK_SIZE];
} SceSha224Context;
メンバ

  h     ワークエリア  
  pad     アラインメント調整用パディング  
  usRemains     SceSha224Context構造体内に一時コピーされた、64バイト未満の余りデータ数  
  usComputed     ダイジェスト値計算済みフラグ  
  ulTotalLen     総データサイズ  
  buf     64バイト未満のデータの一時コピー  

解説

SHA-224ダイジェスト値の計算を分割して行う場合のワークエリアとして用いられます。sceSha224BlockInit()関数, sceSha224BlockUpdate()関数, sceSha224BlockResult()関数がワークエリアとして使用しますので、アプリケーションがこの構造体のメンバに直接アクセスしてはいけません。

関連項目

sceSha224BlockInit(), sceSha224BlockUpdate(), sceSha224BlockResult()

sceSha224Digest
 [関数]
SHA-224ダイジェストの計算

形式

#include <libsha224.h>
int sceSha224Digest (
	const void *plain,
	unsigned int len,
	unsigned char *digest
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  plain     ダイジェスト値を計算する平文のデータへのポインタを指定します。  
  len     ダイジェスト値を計算する平文のデータサイズ(バイト数)を指定します。  
  digest     算出されたダイジェスト値(28バイト)を返します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

SHA-224ダイジェスト値を計算します。
この関数はダイジェスト値を計算する平文のデータがすべてメモリ上に展開されている場合に用います。

sceSha224BlockInit
 [関数]
ダイジェスト値計算ワークの初期化

形式

#include <libsha224.h>
int sceSha224BlockInit (
	SceSha224Context *pContext

呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     ダイジェスト値計算ワークのアドレスを指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  
  SCE_ERROR_INVALID_POINTER     pContextのアドレスが不正  


解説

SHA-224ダイジェスト値計算に用いるワークエリアを初期化します。
この関数はsceSha224BlockUpdate()関数に先立って呼び出しを行ってください。

関連項目

sceSha224BlockUpdate
 [関数]
SHA-224ダイジェスト値計算処理

形式

#include <libsha224.h>
int sceSha224BlockUpdate (
	SceSha224Context *pContext,
	const void *plain,
	unsigned int len
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     ダイジェスト値計算ワークのアドレスを指定します。  
  plain     ダイジェスト値を計算する平文のデータへのポインタを指定します。  
  len     ダイジェスト値を計算する平文のデータサイズ(バイト数)を指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  
  SCE_ERROR_BUSY     ダイジェスト値が計算済みのpContextだった  
  SCE_ERROR_INVALID_POINTER     pContextまたはplainのアドレスが不正  


解説

plain, lenで指定された平文を使ってSceSha224Context構造体内のワークを更新します。sceSha224BlockUpdate()関数は、sceSha224BlockInit()関数とsceSha224BlockResult()関数のあいだで何回でも呼び出すことが可能で、複数回に分けてsceSha224BlockUpdate()関数を呼び出すことでメモリに入りきらない巨大なデータに対してもダイジェスト値を計算することが可能です。

関連項目

SceSha224Context, sceSha224BlockInit(), sceSha224BlockResult()

sceSha224BlockResult
 [関数]
計算されたSHA-224ダイジェストを取得

形式

#include <libsha224.h>
int sceSha224BlockResult (
	SceSha224Context *pContext,
	unsigned char *digest
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     ダイジェスト値計算ワークのアドレスを指定します。  
  digest     算出されたダイジェスト値(28バイト)を返します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     成功  
  SCE_ERROR_INVALID_POINTER     pContextまたはdigestのアドレスが不正  


解説

計算されたダイジェスト値をSceSha224Context構造体から取り出します。SHA-224は64バイト単位でダイジェスト値を計算するアルゴリズムであるため、sceSha224BlockUpdate()関数では64バイトに満たなかった余剰バイトをSceSha224Context構造体内に一時コピーすることがあります。SceSha224Context構造体内に一時コピーされたデータが残っていた場合には、sceSha224BlockResult()関数の呼び出しで最終的なダイジェスト値が決定されます。かならず、sceSha224BlockResult()関数でダイジェスト値を取得してください。
SceSha224Context構造体内のダイジェスト値は、次にsceSha224BlockInit()関数を呼び出すまで有効です。
