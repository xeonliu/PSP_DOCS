libsfmt216091


SCE_SFMT216091_ARRAY_SIZE
 [定数定義]
SFMT216091擬似乱数計算の配列サイズ

構造

#include <libsfmt216091.h>

#define SCE_SFMT216091_ARRAY_SIZE	1689	/* (216091 / 128) + 1 */
解説

SFMT216091による擬似乱数の配列サイズを定義します。
SceSfmt216091Context構造体内にステートとして保持される配列サイズを示すほか、sceSfmt216091FillArray32(), sceSfmt216091FillArray64()関数で、乱数を生成する場合の最小サイズも、この定数によって示されています。

関連項目

SceSfmt216091Context, sceSfmt216091FillArray32(), sceSfmt216091FillArray64()

SceSfmt216091Context
 [構造体]
SFMT216091擬似乱数計算のコンテキスト情報

構造

#include <libsfmt216091.h>
typedef struct SceSfmt216091Context {
	unsigned int idx;
	unsigned int sfmt[SCE_SFMT216091_ARRAY_SIZE][4];
} SceSfmt216091Context;
解説

SFMT216091による擬似乱数を計算するためのワークエリアです。
このワークエリアは、乱数系列1つにつき1個ずつ用意する必要があります。

関連項目

SCE_SFMT216091_ARRAY_SIZE, sceSfmt216091InitGenRand(), sceSfmt216091InitByArray()

sceSfmt216091InitGenRand
 [関数]
SFMT216091擬似乱数ワークの初期化

形式

#include <libsfmt216091.h>
int sceSfmt216091InitGenRand (
	SceSfmt216091Context *pCtx,
	SceUInt32 seed
);
呼出条件

割り込みハンドラから呼び出し可能(割り込み禁止、許可状態に依存しない)
スレッドから呼び出し可能(割り込み禁止、許可状態に依存しない)
マルチスレッドセーフである

引数

  pCtx     乱数系列をコンテキストとして表現するSceSfmt216091Context構造体への  
  ポインタを渡します。  
  seed     乱数系列を指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

SceSfmt216091Context構造体によって表現されるSFMT216091乱数系列を32ビットのシード値によって初期化します。この関数は、sceSfmt216091GenRand32(), sceSfmt216091GenRand64(), sceSfmt216091FillArray32(), sceSfmt216091FillArray64()関数に先立って実行しておく必要があります。

初期化されるのは、pCtxで示されるSceSfmt216091Context構造体だけですので、SceSfmt216091Context構造体を複数個持つことにより複数の乱数系列を同時に扱うことができます。

64ビットの擬似乱数値

解説

SFMT216091による64ビットの擬似乱数を生成します。
この関数に先立ってSceSfmt216091Context構造体は、sceSfmt216091InitGenRand()関数または sceSfmt216091InitByArray()関数によって初期化されている必要があります。

sceSfmt216091GenRand32()関数とsceSfmt216091GenRand64()関数を併用する場合、sceSfmt216091GenRand32()関数を奇数回呼び出した状態でsceSfmt216091GenRand64()を呼び出すと、64ビットの乱数値が得られないことに注意してください。この場合、上位32ビットがゼロに固定された64ビット値が得られます。

sceSfmt216091FillArray32
 [関数]
SFMT216091による32ビット擬似乱数配列を取得

形式

#include <libmt216091.h>
int sceSfmt216091FillArray32 (
	SceSfmt216091Context *pCtx,
	SceUInt32 array[],
	int size
);
呼出条件

割り込みハンドラから呼び出し可能(割り込み禁止、許可状態に依存しない)
スレッドから呼び出し可能(割り込み禁止、許可状態に依存しない)
マルチスレッドセーフである

引数

  pCtx     乱数系列をコンテキストとして表現するSceSfmt216091Context構造体への  
  ポインタを渡します。  
  array     乱数値を受け取るバッファ  
  size     arrayの要素数(SCE_SFMT216091_ARRAY_SIZE*4より大きく、4の倍数)  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

SFMT216091による32ビットの擬似乱数の任意個数、生成します。引数arrayの要素数を指定する引数sizeは、4の倍数値で、(SCE_SFMT216091_ARRAY_SIZE * 4)よりも大きな値でなくてはなりません。
この関数に先立ってSceSfmt216091Context構造体は、sceSfmt216091InitGenRand()関数またはsceSfmt216091InitByArray()関数によって初期化されている必要があります。

sceSfmt216091GenRand32()関数と併用する場合、sceSfmt216091GenRand32()関数が(SCE_SFMT216091_ARRAY_SIZE * 4)回呼び出された状態の場合のみ、sceSfmt216091FillArray32()関数を呼び出すことができます。
sceSfmt216091GenRand64()関数の場合、sceSfmt216091GenRand64()関数が(SCE_SFMT216091_ARRAY_SIZE * 2)回呼び出された状態の場合のみ、sceSfmt216091FillArray32()関数を呼び出すことができます。
