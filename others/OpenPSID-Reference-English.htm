sceOpenPSIDGetOpenPSID
 [Function]
Get OpenPSID

Syntax

#include <openpsid.h>

int sceOpenPSIDGetOpenPSID (
	SceOpenPSID *pOpenPSID
);
Calling Conditions

Cannot be called from an interrupt handler
Can be called from a thread (must be called in an interrupt-enabled state)
Multithread safe

Arguments

  pOpenPSID     OpenPSID  

Return Value

If the function completes normally, SCE_OK is returned.
If an error occurs, a negative value is returned.

Description

This function gets the OpenPSID, which is the ID for an individual PSP™ device.
