scePunyEncodeUCS4
 [関数]
Punycodeエンコード

形式

#include <libpuny.h>
int scePunyEncodeUCS4 (
	char *pszDst,
	unsigned int uiSize,
	const SceWChar32 *puiSrc,
	const unsigned char *case_flag
);
呼出条件

割り込みハンドラから呼び出し可能(割り込み許可/禁止状態に依存しない)
スレッドから呼び出し可能(割り込み許可/禁止状態に依存しない)
マルチスレッドセーフである

引数

  pszDst     Punycode文字列出力バッファ  
  uiSize     出力バッファサイズ(バイト)  
  puiSrc     入力UCS4文字列  
  case_flag     入力ケースフラグ配列  

返り値

成功したとき、出力バイト数を返します。
負の値のときエラー

解説

puiSrcとcase_flagで指定された入力UCS4文字列をPunycodeエンコードして、pszDstで指定されたメモリ領域へ出力し、文字列長を返します。出力用のメモリ領域はアプリケーションが用意する必要があります。case_flagにはNULLを指定する事ができます。

備考

出力したデータはNULLターミネートされています。NULL分の1バイトは返り値の出力バイト数には含まれません。

割り込みハンドラから呼び出し可能(割り込み許可/禁止状態に依存しない)
