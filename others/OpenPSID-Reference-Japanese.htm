OpenPSIDサービス

SceOpenPSID
 [構造体]
OpenPSID構造体

構造

#include <openpsid.h>

#define SCE_OPENPSID_SIZE 16
typedef struct SceOpenPSID {
	SceUChar8 data[SCE_OPENPSID_SIZE];
} SceOpenPSID;
メンバ

  data     OpenPSIDデータ  

解説

OpenPSIDを表す構造体です。

関連項目

sceOpenPSIDGetOpenPSID()

sceOpenPSIDGetOpenPSID
 [関数]
OpenPSIDの取得

形式

#include <openpsid.h>

int sceOpenPSIDGetOpenPSID (
	SceOpenPSID *pOpenPSID
);
呼出条件

割り込みハンドラから呼び出し不可
スレッドから呼び出し可能（割り込み許可状態で呼ぶこと）
マルチスレッドセーフである

引数

  pOpenPSID     OpenPSID  

返り値

正常終了の場合は、SCE_OKを返します。
エラーの場合は、負の値を返します。

解説

PSP®機器個別のIDであるOpenPSIDを取得します。
