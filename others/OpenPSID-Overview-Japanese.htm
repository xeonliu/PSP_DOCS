PSP® Programmer Tool Runtime Library Release 6.6.0

OpenPSIDサービス

1 ライブラリ概要

1. 1 概要

OpenPSIDサービスは、PSP®機器個別のIDを提供するためのライブラリです。PSP®機器個別のIDはOpenPSIDと呼ばれます。

1. 2 関連ファイル

OpenPSIDサービスを使用するために必要なファイルは次のとおりです。

  カテゴリ     ファイル名  
  ヘッダファイル     openpsid.h  
  スタブライブラリファイル     openpsid_stub.a  


2 使用手順

2. 1 ライブラリの初期化

OpenPSIDサービスは暗黙のうちに初期化されています。明示的な初期化を行う必要はありません。

2. 2 OpenPSIDの取得

OpenPSIDは以下のようなコードで取得することができます。

SceOpenPSID id;

if ( SCE_OK == sceOpenPSIDGetOpenPSID(&id) ) { 	// OpenPSIDを取得
    printf("%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x\n",
	id.data[0], id.data[1], id.data[2], id.data[3],
	id.data[4], id.data[5], id.data[6], id.data[7],
	id.data[8], id.data[9], id.data[10], id.data[11],
	id.data[12], id.data[13], id.data[14], id.data[15]);
} else {
	printf("ERROR\n"); 	// エラーハンドリングの記述
}

3 注意事項

3. 1 ミッションクリティカルな目的でのOpenPSIDの利用について

このOpenPSIDは、その一貫性、完全性については弊社では保証することはできません。厳密なコピープロテクションやユーザ管理など、ミッションクリティカルな目的を想定する場合は、充分にご注意ください。
