libmt19937


SceMt19937Context
 [構造体]
MT19937擬似乱数計算のコンテキスト情報

構造

#include <libmt19937.h>
typedef struct SceMt19937Context {
	unsigned int count;
	unsigned int state[624];
} SceMt19937Context;
解説

MT19937による擬似乱数を計算するためのワークエリアです。
このワークエリアは乱数系列1つにつき1個ずつ用意する必要があります。

関連項目

sceMt19937Init(), sceMt19937UInt()

sceMt19937Init
 [関数]
MT19937擬似乱数ワークの初期化

形式

#include <libmt19937.h>
int sceMt19937Init (
	SceMt19937Context *pContext,
	unsigned int seed
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである
引数

  pContext     乱数系列をコンテキストとして表現するSceMt19937Context構造体への  
  ポインタを渡します。  
  seed     乱数系列を指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

SceMt19937Context構造体によって表現されるMT19937乱数系列を初期化します。この関数はsceMt19937UInt()関数に先立って実行しておく必要があります。

初期化されるのはpContextで示されるSceMt19937Context構造体だけですので、SceMt19937Context構造体を複数個持つことにより複数の乱数系列を同時に扱うことができます。

関連項目

SceMt19937Context, sceMt19937UInt()

sceMt19937UInt
 [関数]
MT19937による擬似乱数値取得

形式

#include <libmt19937.h>
unsigned int sceMt19937UInt (
	SceMt19937Context *pContext
);
呼出条件

割り込みハンドラから呼び出し可能
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  pContext     乱数系列をコンテキストとして表現するSceMt19937Context構造体への  
  ポインタを渡します。  

返り値

32ビットの擬似乱数値

解説

MT19937による擬似乱数を取得します。
この関数に先立ってSceMt19937Context構造体はsceMt19937Init()関数によって初期化されている必要があります。

関連項目

SceMt19937Context, sceMt19937Init()
