sceCccJISCS
 [文字コード型]
16bitで表されるJIS X 0213の文字集合

形式

#include <libccc.h>
typedef unsigned short sceCccJISCS;
解説

16bitで表されるJIS X 0213の文字集合。
ビット15が面。ビット8~14に区+0x20の値、ビット0~7に点+0x20の値。

sceCccSetTable
 [関数]
JIS, UCS, 文字コード変換テーブル設定

形式

#include <libccc.h>
void sceCccSetTable (
	sceCccUCS2 const* jis2ucs,
	sceCccJISCS const* ucs2jis
);
引数

  jis2ucs     JISからUCS-2への対応表（JIS並び）のポインタ  
  ucs2jis     UCS-2からJISへの対応表（UCS-2並び）のポインタ

解説

sceCccSwprintfUTF8
 [関数]
ワイド文字列としてフォーマット

形式

#include <libccc.h>
int sceCccSwprintfUTF8 (
	SceWChar16 * wcs,
	SceSize maxlen,
	const SceWChar16 *format,
	...
);
引数

  wcs     フォーマットされた文字列の出力バッファ  
  maxlen     出力バッファの文字数  
  format     書式化文字列  

返り値

フォーマットされた文字数を返します。

解説

ワイド文字としてSceWChar16型を使用し、文字列のフォーマット処理を行います。引数wcsには少なくともmaxlen文字のフォーマット済み文字列を格納できるバッファを指定しなくてはなりません。

引数fomatではフォーマット文字列として％から始まるフォーマット指定文字列を行う事ができます。フォーマット指定文字列は

%[flags][width][.precision][modifier]type

の形式をとります。浮動小数点数を扱うtypeのe, f, g, aはサポートしていません。

flagには以下のフラグ文字の指定が可能です。省略可能です。
  -     左詰めで出力します  
  +     符号付きの場合、'+'または'-'を出力します。  
  ' '（空白）     符号付きで正の数の場合は' '（空白）を出力します。  
  #     o, x, Xの各タイプと併用した時、0, 0x, 0Xを出力します。  
  0     widthと併用した場合、指定された最小幅になるまで0が出力されます。  

witdhは出力する文字幅を指定するフィールドで、10進数で指定します。省略可能です。

.presisionは出力の精度を指定するフィールドで'.'（ピリオド）に続いて10進数で指定します。省略可能です。効果はtypeによって異なります。

modifierはtypeで指定される形式に対して修正を与えます。省略可能です。効果はtypeによって異なります。
  hh     整数に変換する引数がsigned charまたはunsigned charであるとみなす  
  h     整数に変換する引数がshort intまたはunsigned short intであるとみなす  
  l     整数に変換する引数がlong intまたはunsigned long intであるとみなす  
  ll     整数に変換する引数がlong long intまたはunsigned long long intであるとみなす  
  j     整数に変換する引数がintmax_tまたはuintmax_tであるとみなす  
  z     整数に変換する引数がsize_tまたはssize_tであるとみなす  
  t     整数に変換する引数がptrdiff_tであるとみなす  

typeには以下の出力形式の指定が可能です。typeの指定は省略できません。
  d     int引数を符号付き10進数に変換します。  
  i     int引数を符号付き10進数に変換します。  
  o     int引数を符号なし8進数に変換します。  
  u     int引数を符号なし10進数に変換します。  
  x     int引数を符号なし16進数に変換します。  
  X     int引数を符号なし16進数に変換します。  
  lc     int引数をunsigned charとしてワイド文字に変換して出力します。  
  c     SceWChar16引数をマルチバイト文字として出力します。  
  ls     const char*ポインタ引数をUTF-8形式のマルチバイト文字列として  
  ワイド文字列に変換して出力します。  
  s     const SceWChar16*ポインタ引数をマルチバイト文字列として出力します。  
  p     void *ポインタ引数をアドレスとして16進数に変換します。  
  n     これまでに出力された文字数をint *引数が指す変数に保存します。  
  %     %自身を出力します。  

sceCccSwprintfSJIS()との違いは、フォーマット指定文字%lsで参照されるconst char *マルチバイト文字列がUTF-8形式であると判断される点です。

 [関数]
文字列符号方式変換

形式

#include <libccc.h>
int sceCccUTF16toSJIS (
	sceCccSJISTF* dst,
	size_t count,
	sceCccUTF16 const* str
);
引数

  dst     出力文字列  
  count     出力バッファサイズ  
  str     入力文字列  

返り値

変換後の文字数

解説

UTF-16文字列をSJIS符号化文字列へ変換します。

sceCccIsValidUCS4

);
引数

注）現在は厳密なチェックを行っていません。
