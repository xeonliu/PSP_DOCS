SceSha384Context
 [Structure]
Context information for SHA-384 digest value computation

Structure

#include <libsha384.h>
typedef struct SceSha384Context {
	unsigned long long h[8];
	unsigned int pad;
	unsigned short usRemains;
	unsigned short usComputed;
	unsigned long long ulTotalLen;
	unsigned char buf[SCE_SHA384_BLOCK_SIZE];
} SceSha384Context;
Members

  h     Work area  
  pad     Padding for adjusting alignment  
  usRemains     Less than 128 bytes of remaining data, which was temporarily copied within the SceSha384Context structure  
  usComputed     Digest value computed flag  
  ulTotalLen     Total data size (bytes)  
  buf     Temporary copy of less than 128 bytes of data  

Description

This structure is used as a work area when computation of the SHA-384 digest value is segmented. Since the sceSha384BlockInit(), sceSha384BlockUpdate(), and sceSha384BlockResult() functions use this structure as a work area, an application must not directly access the members of this structure.

Return Value

If an error occurs, a negative value is returned.
  Value     Result

Description

This function calculates the SHA-384 digest value.
This function is used when all data of the cleartext for which the digest value is to be calculated has been expanded in memory.

sceSha384BlockUpdate
 [Function]
SHA-384 digest value calculation processing

Syntax

#include <libsha384.h>
int sceSha384BlockUpdate (
SceSha384Context *pContext,
	const void *plain,
	unsigned int len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pContext     Address of digest value calculation work area.  
  plain     Pointer to plaintext data for which digest value is to be computed.  
  len     Data size (bytes) of plaintext data for which digest value is to be computed.  

Return Value

If an error occurs, a negative value is returned.
  Value     Result  
  SCE_OK     Successful completion  
  SCE_ERROR_BUSY     Digest value was already computed for pContext  
  SCE_ERROR_INVALID_POINTER     pContext or plain address is invalid  


Description

This function uses the plaintext specified by plain and len to update the work area within the SceSha384Context structure. The sceSha384BlockUpdate() function, which can be called any number of times between the sceSha384BlockInit() function and sceSha384BlockResult() function, enables the digest value to be calculated even for a large amount of data that cannot fit in memory.

sceSha384BlockResult
 [Function]
Get calculated SHA-384 digest

Syntax

#include <libsha384.h>
int sceSha384BlockResult (
	SceSha384Context *pContext,
	unsigned char *digest
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pContext     Address of digest value calculation work area.  
  digest     Returns the calculated digest value (48 bytes).  

Return Value

If an error occurs, a negative value is returned.
  Value     Result  
  SCE_OK     Normal completion  
  SCE_ERROR_INVALID_POINTER     pContext ordigest address is invalid  


Description

This function retrieves the calculated digest value from the SceSha384Context structure. The SHA-384 algorithm calculates a digest value in units of 128 bytes, so a remaining amount less than 128 bytes may have been temporarily copied within the SceSha384Context structure by the sceSha384BlockUpdate() function. If this remaining data exists, the final digest value is determined by calling the sceSha384BlockResult() function. Always use the sceSha384BlockResult() function to obtain the digest value.
The digest value of the SceSha384Context structure is valid until the next call to the sceSha384BlockInit() function.
