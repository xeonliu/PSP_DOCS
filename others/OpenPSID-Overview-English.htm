OpenPSID Service Overview

1 Library Overview

1. 1 Overview

The OpenPSID service is a library that can be used to provide a unique ID for an individual PSP™. Such an ID is known as an OpenPSID.

1. 2 Related Files

The following files are required to use the OpenPSID service.

  Category     Filename  
  Header file     openpsid.h  
  Stub library file     openpsid_stub.a  


2 Usage Procedure

2. 1 Library Initialization

The OpenPSID service is initialized implicitly. Explicit initialization need not be performed.

2. 2 Getting an OpenPSID

An OpenPSID can be obtained by using the following code.

SceOpenPSID id;

if ( SCE_OK == sceOpenPSIDGetOpenPSID(&id) ) { 	// Get OpenPSID
    printf("%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x%02x\n",
	id.data[0], id.data[1], id.data[2], id.data[3],
	id.data[4], id.data[5], id.data[6], id.data[7],
	id.data[8], id.data[9], id.data[10], id.data[11],
	id.data[12], id.data[13], id.data[14], id.data[15]);
} else {
	printf("ERROR\n"); 	// Error handling description
}

3 Precautions

3. 1 Using OpenPSIDs for Mission Critical Applications

SCE cannot guarantee the consistency or completeness of OpenPSIDs. Take necessary precautions when considering their use in mission critical applications such as strict copy protection or user management.
