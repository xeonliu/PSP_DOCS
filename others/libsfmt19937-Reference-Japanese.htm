libsfmt19937


SCE_SFMT19937_ARRAY_SIZE
 [定数定義]
SFMT19937擬似乱数計算の配列サイズ

構造

#include <libsfmt19937.h>

#define SCE_SFMT19937_ARRAY_SIZE	156	/* (19937 / 128) + 1 */
解説

SFMT19937による擬似乱数の配列サイズを定義します。
SceSfmt19937Context構造体内にステートとして保持される配列サイズを示すほか、sceSfmt19937FillArray32(), sceSfmt19937FillArray64()関数で、乱数を生成する場合の最小サイズも、この定数によって示されています。

関連項目

SceSfmt19937Context, sceSfmt19937FillArray32(), sceSfmt19937FillArray64()

SceSfmt19937Context
 [構造体]
SFMT19937擬似乱数計算のコンテキスト情報

構造

#include <libsfmt19937.h>
typedef struct SceSfmt19937Context {
	unsigned int idx;
	unsigned int sfmt[SCE_SFMT19937_ARRAY_SIZE][4];
} SceSfmt19937Context;
解説

SFMT19937による擬似乱数を計算するためのワークエリアです。
このワークエリアは、乱数系列1つにつき1個ずつ用意する必要があります。

関連項目

SCE_SFMT19937_ARRAY_SIZE, sceSfmt19937InitGenRand(), sceSfmt19937InitByArray()

sceSfmt19937InitGenRand
 [関数]
SFMT19937擬似乱数ワークの初期化

形式

#include <libsfmt19937.h>
int sceSfmt19937InitGenRand (
	SceSfmt19937Context *pCtx,
	SceUInt32 seed
);
呼出条件

割り込みハンドラから呼び出し可能(割り込み禁止、許可状態に依存しない)
スレッドから呼び出し可能(割り込み禁止、許可状態に依存しない)
マルチスレッドセーフである

引数

  pCtx     乱数系列をコンテキストとして表現するSceSfmt19937Context構造体への  
  ポインタを渡します。  
  seed     乱数系列を指定します。  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

SceSfmt19937Context構造体によって表現されるSFMT19937乱数系列を32ビットのシード値によって初期化します。この関数は、sceSfmt19937GenRand32(), sceSfmt19937GenRand64(), sceSfmt19937FillArray32(), sceSfmt19937FillArray64()関数に先立って実行しておく必要があります。

初期化されるのは、pCtxで示されるSceSfmt19937Context構造体だけですので、SceSfmt19937Context構造体を複数個持つことにより複数の乱数系列を同時に扱うことができます。

sceSfmt19937GenRand64
 [関数]
SFMT19937による64ビット擬似乱数値取得

形式

#include <libmt19937.h>
SceUInt64 sceSfmt19937GenRand64 (
	SceSfmt19937Context *pCtx
);
呼出条件

割り込みハンドラから呼び出し可能(割り込み禁止、許可状態に依存しない)
スレッドから呼び出し可能(割り込み禁止、許可状態に依存しない)
マルチスレッドセーフである

引数

  pCtx     乱数系列をコンテキストとして表現するSceSfmt19937Context構造体への  
  ポインタを渡します。  

返り値

64ビットの擬似乱数値

解説

SFMT19937による64ビットの擬似乱数を生成します。
この関数に先立ってSceSfmt19937Context構造体は、sceSfmt19937InitGenRand()関数またはsceSfmt19937InitByArray()関数によって初期化されている必要があります。

sceSfmt19937GenRand32()関数とsceSfmt19937GenRand64()関数を併用する場合、sceSfmt19937GenRand32()関数を奇数回呼び出した状態でsceSfmt19937GenRand64()を呼び出すと、64ビットの乱数値が得られないことに注意してください。この場合、上位32ビットがゼロに固定された64ビット値が得られます。

sceSfmt19937FillArray32
 [関数]
SFMT19937による32ビット擬似乱数配列を取得

形式

#include <libmt19937.h>
int sceSfmt19937FillArray32 (
	SceSfmt19937Context *pCtx,
	SceUInt32 array[],
	int size
);
呼出条件

割り込みハンドラから呼び出し可能(割り込み禁止、許可状態に依存しない)
スレッドから呼び出し可能(割り込み禁止、許可状態に依存しない)
マルチスレッドセーフである

引数

  pCtx     乱数系列をコンテキストとして表現するSceSfmt19937Context構造体への  
  ポインタを渡します。  
  array     乱数値を受け取るバッファ  
  size     arrayの要素数(SCE_SFMT19937_ARRAY_SIZE*4より大きく、4の倍数)  

返り値

エラーの場合、負の値が返ります。
  値     結果  
  SCE_OK     正常終了  


解説

SFMT19937による32ビットの擬似乱数の任意個数、生成します。引数arrayの要素数を指定する引数sizeは、4の倍数値で、(SCE_SFMT19937_ARRAY_SIZE * 4)よりも大きな値でなくてはなりません。
この関数に先立ってSceSfmt19937Context構造体は、sceSfmt19937InitGenRand()関数またはsceSfmt19937InitByArray()関数によって初期化されている必要があります。

sceSfmt19937GenRand32()関数と併用する場合、sceSfmt19937GenRand32()関数が(SCE_SFMT19937_ARRAY_SIZE * 4)回呼び出された状態の場合のみ、sceSfmt19937FillArray32()関数を呼び出すことができます。
sceSfmt19937GenRand64()関数の場合、sceSfmt19937GenRand64()関数が(SCE_SFMT19937_ARRAY_SIZE * 2)回呼び出された状態の場合のみ、sceSfmt19937FillArray32()関数を呼び出すことができます。
