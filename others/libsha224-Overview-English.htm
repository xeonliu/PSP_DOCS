libsha224 Overview


1 Library Overview

1. 1 Overview

libsha224 is a library for generating a SHA-224-format digest defined by RFC3874. It can be used to detect data corruption and prevent data tampering by applying Keyed-Hashing for Message Authentication (HMAC).

2 Usage Procedure

2. 1 Basic Usage Procedure

(1) SHA-224 digest value computation (comprehensive)


No specific initialization is required to use libsha224.

u_char digest[SCE_SHA224_DIGEST_SIZE];

sceSha224Digest(plaintext, length, digest);

You can compute the digest value simply by calling the sceSha224Digest() function, as shown above.

(2) SHA-224 digest computation (divided)

You can compute the digest value for a large amount of data by breaking up the calculation of the hash value as shown below.

SceSha224Context sha;
u_char digest[SCE_SHA224_DIGEST_SIZE];

sceSha224BlockInit(&sha);
sceSha224BlockUpdate(&sha, plain1, len1);
sceSha224BlockUpdate(&sha, plain2, len2);
sceSha224BlockUpdate(&sha, plain3, len3);
	:		Repeat an arbitrary number of times
sceSha224BlockResult(&sha, digest);

After using the sceSha224BlockInit() function to initialize the SceSha224Context structure and calling the sceSha224BlockUpdate() function an arbitrary number of times, the digest value can be obtained by calling the sceSha224BlockResult() function.
