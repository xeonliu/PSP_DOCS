sceAdler32
 [Function]
Compute ADLER-32 checksum encoding

Syntax

#include <libadler.h>
unsigned int sceAdler32(
	unsigned int uiAdler,
	const unsigned char *pSrc,
	unsigned int uiSize
);
Calling Conditions

Can be called from a thread
Multithread safe

Arguments

  uiAdler     Initial value  
  pSrc     Input data  
  uiSize     Input data byte size  

Return Value

ADLER-32 checksum

Description

This function computes a checksum using the ADLER-32 algorithm.
