libsha256 Overview


1 Library Overview

1. 1 Overview

libsha256 is a library used to generate digest values in the SHA-256 format defined by FIPS 180-2. It can be used, for example, in order to detect data corruption, or to prevent data tampering by applying HMAC (Keyed Hashing for Message Authentication).

2 Usage Procedure

2. 1 Basic Usage Procedure

(1) SHA-256 Digest Value Calculation (comprehensive)


No particular initialization is required in order to use libsha256.

u_char digest[SCE_SHA256_DIGEST_SIZE];

sceSha256Digest(plaintext, length, digest);

In the above manner, the sceSha256Digest() function can be used at any time to perform digest value calculation.

(2) SHA-256 Digest Value Calculation (divided)

When calculating digest values for large amounts of data, hash calculations can be broken into parts.

SceSha256Context sha;
u_char digest[SCE_SHA256_DIGEST_SIZE];

sceSha256BlockInit(&sha);
sceSha256BlockUpdate(&sha, plain1, len1);
sceSha256BlockUpdate(&sha, plain2, len2);
sceSha256BlockUpdate(&sha, plain3, len3);
	:		Repeat any number of times
sceSha256BlockResult(&sha, digest);

Initialize the SceSha256Context structure using the sceSha256BlockInit() function, call the sceSha256BlockUpdate() function any number of times, and then finally obtain the digest value using the sceSha256BlockResult() function.
