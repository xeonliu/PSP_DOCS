libpuny Overview

1 Library Overview

1. 1 Overview

libpuny is a library for encoding/decoding Punycode format, which is defined in RFC3492.

1. 2 Related Files

The following files are required to use libpuny.

  Category     Filename  
  Header file     libpuny.h  
  Static link library file     libpuny.a  
  Stub library file     libpuny_stub.a  
  weak import stub library file     libpuny_stub_weak.a  
  PRX module file     libpuny.prx  


2 Using the Library

2. 1 Basic Usage Procedure

No specific initialization is required to use libpuny.

Encoding to Punycode format
The following procedure can be used to encode a UCS4 string to Punycode format. Note that the application must allocate memory for output since libpuny does not allocate memory internally.

SceWChar32 ucs4[SOURCE_LEN];
char	punycode[DSTBUF];

res = scePunyEncodeUCS4(punycode, DSTBUF, ucs4, NULL);	//Encode
Encoding to Punycode format is performed with the scePunyEncodeUCS4() function as shown above.

Decoding from Punycode format
The following procedure can be used to decode a previously encoded Punycode byte string. Note that the application must allocate memory for output since libpuny does not allocate memory internally.

SceWChar32 ucs4[DSTBUF];
char	punycode[SOURCE_LEN];

res = scePunyDecodeUCS4(ucs4, NULL, DSTBUF, punycode);	//Decode
Decoding from Punycode format is performed with the scePunyDecodeUCS4() function as shown above.
