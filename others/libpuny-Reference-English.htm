scePunyEncodeUCS4
 [Function]
Encode Punycode

Syntax

#include <libpuny.h>
int scePunyEncodeUCS4 (
	char *pszDst,
	unsigned int uiSize,
	const SceWChar32 *puiSrc,
	const unsigned char *case_flag
);
Calling Conditions

Can be called from an interrupt handler (does not depend on interrupt-disabled or -enabled state)
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pszDst     Punycode string output buffer  
  uiSize     Output buffer size (bytes)  
  puiSrc     Input UCS4 string  
  case_flag     Input case flag array  

Return Value

When the function is successful, the number of output bytes is returned.
If an error occurs, a negative value is returned.

Description

This function performs Punycode encoding of the input UCS4 string specified by puiSrc and case_flag. Then, it outputs the encoded result to the memory area specified by pszDst and returns the length of the output string. The application must provide a memory area for the output string. case_flag can be specified as NULL.

Notes

The output data string is NULL terminated. The one byte for the NULL terminator is not included in the number of output bytes that is returned.

scePunyDecodeUCS4
 [Function]
Decode Punycode

Syntax

#include <libpuny.h>
int scePunyDecodeUCS4 (
	SceWChar32 *puiDst,
	unsigned char *case_flag,
	unsigned int uiSize,
	const char *pszSrc
);
Calling Conditions

Can be called from an interrupt handler (does not depend on interrupt-disabled or -enabled state)
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  puiDst     Output UCS4 string  
  case_flag     Output case flag array  
  uiSize     Output buffer size (characters)  
  pszSrc     Input Punycode string  

Return Value

When the function is successful, the number of output characters is returned.
If an error occurs, a negative value is returned.

Description

This function decodes the input Punycode string specified by pszSrc. Then, it outputs the result to the memory area specified by puiDst and case_flag and returns the number of characters in the output string. The application must provide a memory area for the output string. Data is output in UCS4 format to puiDst. case_flag can be specified as NULL.

Notes

The output data string is NULL terminated. The one byte for the NULL terminator is not included in the number of output bytes that is returned.

scePunyDecodeUCS2
 [Function]
Decode Punycode

Syntax

#include <libpuny.h>
int scePunyDecodeUCS2
	SceWChar16 *pusDst,
	unsigned char *case_flag,
	unsigned int uiSize,
	const char *pszSrc
);
Calling Conditions

Can be called from an interrupt handler (does not depend on interrupt-disabled or -enabled state)
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pusDst     Output UCS2 string  
  case_flag     Output case flag array  
  uiSize     Output buffer size (characters)  
  pszSrc     Input Punycode string  

Return Value

When the function is successful, the number of output characters is returned.
If an error occurs, a negative value is returned.

Description

This function decodes the input Punycode string specified by pszSrc. Then, it outputs the result to the memory area specified by puiDst and case_flag and returns the number of characters in the output string. The application must provide a memory area for the output string. Data is output in UCS2 format to puiDst. An error will occur if the code point cannot be represented in UCS2 format. case_flag can be specified as NULL.

Notes

The output data string is NULL terminated. The one byte for the NULL terminator is not included in the number of output bytes that is returned.
