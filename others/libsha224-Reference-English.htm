SceSha224Context
 [Structure]
Context information for SHA-224 digest value computation

Structure

#include <libsha224.h>
typedef struct SceSha224Context {
	unsigned int h[8];
	unsigned int pad;
	unsigned short usRemains;
	unsigned short usComputed;
	unsigned long long ulTotalLen;
	unsigned char buf[SCE_SHA224_BLOCK_SIZE];
} SceSha224Context;
Members

  h     Work area  
  pad     Padding for adjusting alignment  
  usRemains     Less than 64 bytes of remaining data, which was copied temporarily within the SceSha224Context structure  
  usComputed     Digest value computed flag  
  ulTotalLen     Total data size  
  buf     Temporary copy of data of which there is less than 64 bytes  

Description

This structure is used as a work area when computation of the SHA-224 digest value is divided. Since the sceSha224BlockInit() function, sceSha224BlockUpdate() function, and sceSha224BlockResult() function use this structure as a work area, applications must not directly access the members of this structure.

sceSha224BlockUpdate
 [Function]
SHA-224 digest value computation processing

Syntax

#include <libsha224.h>
int sceSha224BlockUpdate (
	SceSha224Context *pContext,
	const void *plain,
	unsigned int len
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pContext     Address of digest value computation work area.  
  plain     Pointer to plaintext data for which digest value is to be computed.  
  len     Data size (bytes) of plaintext data for which digest value is to be computed.  

Return Value

If an error occurs, a negative value is returned.
  Value     Result  
  SCE_OK     Normal completion  
  SCE_ERROR_BUSY     Digest value was already computed for pContext  
  SCE_ERROR_INVALID_POINTER     pContext or plain address is invalid  


Description

This function uses the plaintext specified by plain and len to update the work area within the SceSha224Context structure. The sceSha224BlockUpdate() function, which can be called any number of times between the sceSha224BlockInit() function and sceSha224BlockResult() function, enables the digest value to be computed even for a large amount of data that cannot fit in memory.

sceSha224BlockResult
 [Function]
Get computed SHA-224 digest

Syntax

#include <libsha224.h>
int sceSha224BlockResult (
	SceSha224Context *pContext,
	unsigned char *digest
);
Calling Conditions

Can be called from an interrupt handler
Can be called from a thread (does not depend on interrupt-disabled or -enabled state)
Multithread safe

Arguments

  pContext     Address of digest value computation work area.  
  digest     Returns the computed digest value (28 bytes).  

Return Value

If an error occurs, a negative value is returned.
  Value     Result  
  SCE_OK     Normal termination  
  SCE_ERROR_INVALID_POINTER     pContext ordigest address is invalid  


Description

This function extracts the computed digest value from the SceSha224Context structure. Since SHA-224 is an algorithm that computes a digest value in units of 64 bytes, less than 64 remaining bytes may have been temporarily copied within the SceSha224Context structure by the sceSha224BlockUpdate() function. If data remains which has been temporarily copied into the SceSha224Context structure, the final digest value will be determined by calling the sceSha224BlockResult() function. Always use the sceSha224BlockResult() function to obtain the digest value.
The digest value of the SceSha224Context structure is valid until the next time the sceSha224BlockInit() function is called.
