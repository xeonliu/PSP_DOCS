GPI/GPO操作ユーティリティ


sceKernelGetGPI
 [GPI操作]
GPIスイッチの参照

形式

#include <kernel.h>
unsigned char sceKernelGetGPI (
	void
);
呼出条件

割り込みハンドラから呼び出し可能（割り込み禁止、許可状態に依存しない）
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

なし

返り値

GPIスイッチの設定を返します。

解説

PSP®開発TOOLのフロントパネルにあるGPIスイッチの設定を参照します。スイッチ7〜0が返り値のbit 7〜0に対応し、スイッチをONにしたビットが1になります。

GPIスイッチはゲームデベロッパが自由に使用できるデバッグ用のハードウェアスイッチです。PSP®本体ではsceKernelGetGPI()関数は常に0を返します。

sceKernelSetGPO
 [GPO操作]
GPO発光ダイオードの設定

形式

#include <kernel.h>
void sceKernelSetGPO (
	unsigned char gpo
);
呼出条件

割り込みハンドラから呼び出し可能（割り込み禁止、許可状態に依存しない）
スレッドから呼び出し可能（割り込み禁止、許可状態に依存しない）
マルチスレッドセーフである

引数

  gpo     GPO設定値  

返り値

なし

解説

PSP®開発TOOLのフロントパネルにあるGPO LEDの設定を行います。LED 7〜0が引数gpoのbit 7〜0に対応し、ビットを1にしたLEDが発光します。

GPO LEDはゲームデベロッパが自由に使用できるデバッグ用の発光ダイオードです。PSP®本体では使用してもなにもおきません。
